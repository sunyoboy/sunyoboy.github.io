## 项目概览

这是一个**个人知识管理与 A 股交易复盘系统**，基于 VitePress 构建为静态站点，内容用 Markdown 编写，Git + 双 remote（GitHub + Gitee）管理版本。

核心理念：**知行合一** — 每日复盘驱动交易系统迭代，问题清单驱动认知升级，OKR 驱动个人成长。

## 🧠 精力分配总纲

> **原则：高能量做判断，中能量做执行，低能量做维护。** 把最清醒的时间留给最重要的决策，把能自动化的事交给脚本。

### Eisenhower 优先级矩阵

```
                紧急                      不紧急
        ┌──────────────────┬──────────────────┐
  重要   │  🔴 立即做（高能量）  │  🟡 安排做（保护时段） │
        │  止损触发/清仓信号    │  量化回测开发        │
        │  孕期异常/检查异常    │  AI Skill/脚本编写    │
        │  关键节点票务到期     │  写作精品输出         │
        │  医生紧急沟通        │  Python 技能深化      │
        ├──────────────────┼──────────────────┤
 不重要  │  🔵 快速做（批量处理）  │  ⚪ 不 做            │
        │  跑脚本抓数据        │  工具美化/换主题       │
        │  Git add/commit     │  追新框架/新工具       │
        │  更新 catalog/sidebar│  无目的刷社交媒体      │
        │  财务数据录入        │  和陌生人争论          │
        └──────────────────┴──────────────────┘
```

> **核心纪律**：🟡「重要不紧急」象限是第二增长曲线的位置。**它的敌人不是 🔴 紧急事项，而是 ⚪ 无意义消耗。** 周末上午的 3 小时深度块就是为这个象限设的保护时段——没有它，🟡 永远是「等有时间再做」，而那一天永远不会来。

| 精力需求 | 任务类型 | 最佳时段 |

| 💤 **不做** | 盯盘、刷社交媒体、追热点、工具美化 | 任何时间都不要 |

## 每日操作清单

### 🌅 盘前 · 9:00-9:25 · 🔋🔋🔋 高能量

> 一天中最重要的 25 分钟。只做需要判断力的事。

- [ ] 🔋🔋🔋 查看前一日美股收盘（道指/纳指/标普/A50期货）
- [ ] 🔋🔋🔋 查看当日重大事件（FOMC/数据公布/除权除息）→ 参考 `review/2026/2026.md` 经济日历
- [ ] 🔋🔋🔋 回顾持仓盈亏和昨日操作计划 → 参考最近一期 `review/2026/06/` 日复盘
- [ ] 🔋🔋🔋 确认各标的止损/止盈/低吸区间 → 运行 `python3 scripts/ma5-deviation.py`
- [ ] ⚠️ **任何操作建议必须同时给出三个数字：入场价、止损价、仓位。缺一不可。** 不报这三个数字的建议不算操作建议。
- [ ] 🔋🔋 如果有触发紧急信号 → 立即在开盘后执行（不等盘后）

### 🌇 收盘后 · 15:00-16:00 · 🔋🔋 中能量

> 执行为主。跑脚本、填数据、写记录。不做盘中决策。

- [ ] 🔋 **运行 `python3 scripts/fetch-market-data.py`** — 自动抓取 7 大指数 + 6 个重点标的行情，生成当日复盘模板
- [ ] 🔋 **运行 `python3 scripts/ma5-deviation.py`** — 输出三维指标全景诊断，检查是否有清仓/减仓信号
- [ ] 🔋🔋 填写复盘模板（按顺序，不跳项）：
  - 盘面特征（黄白线分化、成交量、涨跌比）
  - 今日驱动（第一/第二/第三驱动）
  - 热门板块（前5 + 逻辑）
  - 资金流向（流入/流出）
  - 技术面（支撑/压力/形态）
  - 后市关键（2-3 个核心观察信号）
- [ ] 🔋 填写「做了什么」— 当日交易记录
- [ ] 🔋 填写「持仓盈亏」— 三账户汇总
- [ ] 🔋🔋 填写「学到了什么」— 至少 1 条反思（需要一点思考，不要流水账）
- [ ] 🔋🔋 填写「明日计划」— 具体执行清单（含价格/时间条件）

### 🌙 晚间 · 20:30-21:30 · 🔋 低能量

> 情感连接 + 轻量打卡。不动脑。不写代码。不看账户。

- [ ] 🔋 **孕期关怀** — 按「有效关心清单」发消息，确认当日用药和身体感受
- [ ] 🔋 **一句话反思** — 当天最大的情绪波动或认知收获（就一句，不展开）
- [ ] 🔋 **运动打卡** — 记录当日运动，目标 ≥2 次/周
- [ ] 🔋 **🚀 飞轮扫一眼** — 今天有没有在量化/AI/写作/Python 上推进？有→记一笔。没有→别自责，明天来

### 📋 盘后检查项 · 🔋 低能量 · 收盘后顺手做

- [ ] 🔋🔋 **MA5 偏离度**是否有触发减仓/清仓信号的标的 → 对照 `docs/ma5-deviation-discipline.md` 决策矩阵
- [ ] 🔋🔋 **止损线**是否被触发 → 天山铝业 11.8/11.5、卫星ETF 1.35、紫金矿业 29.50
- [ ] 🔋 **低吸区间**是否需要调整 → 参考当日收盘价与 BUY_ZONES 的差距
- [ ] 🔋 **资金归集**进度 → 超小仓位清仓（有色ETF/500ETF）
- [ ] 🔋 **月度市场数据**是否需要更新（月底最后一周）
- [ ] 🔋🔋 **离合器三项**（下班族交易必备，收盘后挂完再关电脑）：
  - ① 条件单检查：今天挂的条件单触发了吗？明天的单要改吗？
  - ② 天山减仓监控：今日价格 ___（< 11.80？→ 挂卖出条件单）
  - ③ 事件日历：今天/明天有 PCE/FOMC/财报吗？→ 有则延迟新开仓
- [ ] 🔋 **候选池扫一眼**（30秒·不碰≠不看）：
  → 38候选池 + 8全球补充，每天扫一眼涨跌
  → 「不买」≠「不看」，条件变了就重新评估
  → 教训：科创50从1925→2080 (+8%)，因为「不碰」变「不看」全错过
