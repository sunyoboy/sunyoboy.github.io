# ⚖️ 韩非·李斯 · 法家双峰

> 荀子的两个学生——韩非是法家理论的集大成者，李斯是法家实践的操盘手。
>
> **借其制度设计，补其无礼之失。** 没有礼的法 = 秦朝十五年。

---

## 一、韩非：法 · 术 · 势 三角架构

韩非口吃但文笔锋利。核心信念：**不相信人的自觉，只相信制度的设计。**

### 法：明确公开的规则

「法者，宪令著于官府，刑罚必于民心。」——法是明文规定的，惩罚必须人人信服。

| 核心 | 韩非原文 | 交易映射 |
|------|------|------|
| 法要公开 | 著于官府 | SOP、止损线、仓位上限——写下来，不是记脑子里 |
| 法要执行 | 刑罚必 | 止损触发 → 不动摇。不找理由 |
| 法要统一 | 不因人而异 | 三账户用同一套纪律。东北和广发不能两种逻辑 |

### 术：管理方法 · 循名责实

「术者，因任而授官，循名而责实。」——根据能力任命职位，根据名义考核实效。

> **核心**：让每个人说了什么，就考核他做了什么。名和实对不上的，罚。

| 交易映射 | 说明 |
|------|------|
| 「明日计划」vs 第二天「做了什么」 | 7/21 案例：上午计划"东北试仓沪深300"→ 实际卖沪深300。**名实不符 = 纪律漏洞** |
| 复盘里「学到了什么」vs 后续执行 | 学到了"中午不操作"→ 下次中午还操作 → 名实不符 |

### 势：权力和地位的势能

「势者，胜众之资也。」——势是让人服从的资本。

> **交易映射**：条件单 = 势。不是靠你手动执行（你没势——你会上班、会开会、会忘记），而是条件单替你执行（它有势——盯盘 24 小时、触发就下单、不带情绪）。

### 三者关系

```
法（规则）+ 术（考核）+ 势（执行力）
     ↓
光有法没有术 → 规则是废纸（没人检查执行情况）
光有术没有势 → 查出问题也改不了（没有条件单兜底）
三者缺一不可
```

### 其他可借鉴思想

#### 不期修古，不法常可

「是以圣人不期修古，不法常可。」——不迷信旧经验，条件变了方法要变。

> 交易映射：上次 -20% 抄底赚了 ≠ 这次 -20% 也能抄。**每次底部条件不同，用 capitulation 信号具体分析，不套模板。**

#### 赏罚分明

「刑过不避大臣，赏善不遗匹夫。」——惩罚不管你是谁，奖励不遗漏小人物。

> 交易映射：止损面前每只票平等——不管你对它有多少感情、亏了多少、有什么故事。

## 二、李斯：实干家的工具箱

李斯从「粮仓老鼠」悟出人生哲学——厕所老鼠又瘦又怕人，粮仓老鼠又肥又不怕人。**人和老鼠一样，在什么环境里比你是什么重要得多。**

### 环境决定论

「人之贤不肖，譬如鼠矣，在所自处耳。」

> **核心启示：别考验意志力。建一个让自己没法犯错的系统，而不是在诱惑面前硬扛。** 条件单就是这个逻辑——不让手动操作有机会犯错。

### 泰山不让土壤

「泰山不让土壤，故能成其大；河海不择细流，故能就其深。」

> **交易映射**：CLAUDE.md「最小推进原则」——1 行代码 > 0 行。1 段话 > 空白。25 分钟番茄钟 > 等周末 4 小时。第二增长曲线不需要天天有突破，只需要每天不中断。

### 统一标准化

最大功绩：书同文、车同轨、统一度量衡。每个地方标准不同 → 全部统一 → 帝国才有了运转基础。

> **交易映射**：三账户统一逻辑、统一条件单格式、统一复盘模板。需要「总指挥官视角」——不是东北一个逻辑、广发另一个逻辑。

## 三、不能借鉴的

### 韩非 · 不能借

| 缺陷 | 说明 |
|------|------|
| **只靠法不靠礼** | 否定人的内在修养——纯外部约束 → 高压 → 反弹。秦朝 15 年灭亡 |
| **君主本位** | 术是教君王控制臣下的——用到人际关系上是灾难 |

### 李斯 · 不能借

| 缺陷 | 说明 |
|------|------|
| **焚书坑儒** | 消灭异见维持统一——极端威权逻辑 |
| **妒杀韩非** | 怕同学韩非被秦王重用 → 下毒害死。**有才无德——反面典型** |
| **沙丘之变** | 勾结赵高篡改始皇遗诏 → 废扶苏立胡亥 → 秦朝三年灭亡的导火索。**为权力不择手段最终被权力反噬** |

> 🚨 **李斯 = 「有术无道」的终极反面教材。** 临死前对儿子说"吾欲与若复牵黄犬，俱出上蔡东门逐狡兔，岂可得乎"——死了才想起最简单的快乐。

## 四、荀子 → 韩非 → 李斯 → 你的体系

```
荀子：人性恶 → 化性起伪 → 需要制度
    ├── 韩非：制度 = 法+术+势（理论体系）
    └── 李斯：制度 = 标准化+统一执行（实践操作）

借什么：
  韩非法+术+势 → 纪律 + 复盘考核 + 条件单
  李斯环境决定论 → 不考验意志力·建系统
  李斯标准化 → 三账户统一逻辑
  荀子礼 → 信条+内省（补充法家缺失的内部约束）

不借什么：
  韩非纯法无礼 → 只用纪律不用信条 = 高压必崩
  李斯有术无道 → 有执行力没底线 = 死在自己造的绞肉机里
```
