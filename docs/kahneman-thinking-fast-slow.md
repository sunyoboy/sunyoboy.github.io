## 两套思维系统

```
系统1（快）              系统2（慢）
───────                  ───────
自动的                    需要主动调用的
不费力的                  消耗能量的
情绪驱动的                逻辑驱动的
"我感觉该买了"             "成交额情绪 1.06，正常，可评估"
能同时处理很多事           一次只能做一件事
几乎不耗能                做完累
```

大多数时候系统1在替你活着——你走路不会想"先迈左脚还是右脚"。**问题是系统1也替你做了很多它不该做的决定。比如在分时图跳动的时候点击买入。**

### 各自擅长什么

系统1和系统2不是"好系统 vs 坏系统"——是**分工不同**。用错场景才是问题。

| | 系统1（快）| 系统2（慢）|
|------|------|------|
| **擅长的** | 熟练技能的实时执行、模式识别、危险闪避 | 复杂计算、多选项比较、长期后果评估 |
| **不擅长的** | 概率判断、反直觉推理、延迟满足 | 实时反应、并行处理、自动执行 |
| **能耗** | ≈ 零 | 高，做累了会罢工 |
| **正确用法** | 让熟练的事自动发生 | 让重要的事被仔细分析 |
| **常见错误** | 被用来做投资决策 | 被用来纠结午餐吃什么 |

```
该用系统1的事 → 走路、认脸、母语理解、老司机开车、库里投篮
该用系统2的事 → 买哪只股票、要不要辞职、手术方案、学新技能
```

> 🎯 核心原则：**能自动化的自动化（交给系统1），该分析的分析（留给系统2）。把系统2的精力省下来，花在真正值得的事上。**

### 运动：系统1还是系统2？

> 运动不是天生用系统1——是**从系统2练进系统1**。

```
初学投篮（系统2）              职业赛场（系统1）
─────────────────             ────────────────
每一帧都有意识控制              0.3 秒出手，身体自动完成
"屈膝、抬肘、压腕"              大脑不需要经过语言区
→ 动作僵硬、慢、容易断          → 流畅、快、不可中断
```

**这就是"刻意练习"的本质**：通过系统2的反复纠正，把正确动作写入系统1的肌肉记忆。一旦写入成功，执行阶段主要靠系统1——但系统2始终在后台监控：

| 运动场景 | 哪个系统主导 | 说明 |
|------|:--:|------|
| 投篮动作执行 | **系统1** | 肌肉记忆，自动化 |
| 阅读防守阵型 | 系统1 + 系统2 | 经验直觉 + 有意识搜索 |
| 关键时刻决策（最后5秒）| **系统2** | 传还是投？需要计算 |
| 训练中纠正动作 | **系统2** | 教练说"肘偏了"→ 有意识调整 |
| 受伤后复健 | **系统2** | 旧模式失效，重建神经通路 |

> 这和交易完全同构：**框架是系统2慢慢建的，条件单是系统1自动执行的。** 你不需要盘中思考"该不该卖"——就像库里不需要投篮时思考"手腕该压多少度"。

### 怎么失灵

```
系统1看到一个东西 → 立刻给答案
系统2 → "系统1给答案了，那我也别费劲了"

例: 屏幕上科创50 +3.87%
系统1: "涨了！机会！"（0.3秒）
系统2: （沉默。默认接受系统1的结论。）
→ 你差点追进去。每次盘中看一眼都是同样的流程。
```

> 这不是你一个人的问题——是人类大脑的出厂设置。每点开一次APP，系统2就输一次。

## 三个核心发现

### 1. 系统1抢答，系统2懒得检查

卡尼曼因为这个发现拿了诺贝尔奖——市场整体的非理性不是偶尔出现，是系统1的结构性必然。几百万个系统1同时在分时图面前做决策 = 市场情绪周期。

### 2. 你对自己的判断过度自信

"我觉得科创还能涨"——这个"觉得"从哪里来？大概率来自刚看到它涨了3%。这叫**可用性启发式**：大脑自动把"最近发生的事"标记为"更重要的事"。

```
今天科创 +3.87% → "它还会涨"
昨天有色 -8.36% → "它还会跌"

同一套大脑。同一个误导。每天都在发生。
```

你的框架是你对抗这种误导的唯一武器——因为框架不管你最近看到什么，只算数据。

### 3. 损失比收益痛两倍（损失厌恶）

卡尼曼最著名的实验结论：**丢掉 100 块钱的痛苦，约等于捡到 200 块钱的快乐。**

这解释了你这三天的全部情绪：

```
6/22 "踏空了！"    → 没赚到的钱 = 损失 → 痛感 ≈ 双倍亏损
6/23 "亏了"        → 真损失 → 双倍打击
6/24 "又踏空了"    → 从头循环

同一个框架，同一个结论（"不加"）。
但系统1每次都在对你尖叫。不是因为判断变了——
是因为损失厌恶机制被反复触发。
```

## 与你每天在发生的

| 你今天发生的 | 卡尼曼怎么解释 |
|------|------|
| 上班时总想打开APP看一眼 | **间歇性强化**——不规则的、不可预测的奖励是多巴胺最强的触发器。红绿跳动的分时图和老虎机同一原理 |
| 看着科创+3.87%差点追进去 | **系统1抢答**——0.3秒给出"机会！"，系统2默认接受了。你的框架是系统2的产物，但在盘中它来晚了 |
| 6/22 踏空比 6/23 亏钱还难受 | **损失厌恶**——没赚到 = 损失，痛感是实际亏损的两倍 |
| 认为天山还会跌、科创还会涨 | **可用性启发式**——最近发生的事 = 最真实的事 |
| 觉得"今天做不了别的" | **自我损耗**——一上午频繁切APP，系统2的能量已经消耗殆尽，下午没有精力做任何深度工作 |

## 这本书不是看完就会变理性

卡尼曼在书结尾自己写了这段话：

> "我研究思维偏误 45 年。知道它们全在。但我仍然会在被问到'这个基金经理怎么样'的时候，用'他看起来很有信心'来替代'我需要看他的三年回报数据'。"

系统1永远在你脑子里。你能做的不是消灭它——是设计系统2的强制检查点，让它在关键时刻拦住系统1。

## 对你操作系统的映射

| 你的工具 | 卡尼曼的哪条结论 |
|------|------|
| **CLAUDE.md** "9:30-15:00 只看极端波动" | 减少系统1被刺激的次数 |
| **条件单** | 盘中不需要你打开APP——系统1被物理隔离了 |
| **七层决策框架** | 系统2的强制检查清单——数据代替感觉 |
| **收盘后跑脚本→决策** | 等系统1的盘中热度消退后，让系统2接管 |
| **"5分钟后再看"规则** | 系统1的冲动高峰只有几分钟——等它自己凉掉 |
| **手机删除交易APP** | 最彻底的方案——**没有刺激，就没有系统1劫持** |

### 运动训练 ⇄ 交易系统 同构映射

```
运动训练                    交易系统
────────                   ────────
刻意练习正确动作             建立七层过滤框架（系统2 慢建）
反复重复形成肌肉记忆          收盘跑脚本→填复盘（系统2 固化）
比赛时自动执行               盘中条件单自动触发（系统1 快执行）
教练暂停纠正偏差             周复盘修正框架参数（系统2 纠偏）
受伤后重建动作模式            框架失效后重新设计（系统2 重建）
```

> 库里不需要投篮时想"手腕压多少度"——你也不需要盘中想"该不该卖"。两者都是系统2训练完成后，交给系统1自动执行。**自控力的最高手段是不需要自控。**

> 🎯 卡尼曼证明了你永远不可能靠意志力赢系统1。你只能靠设计一个系统2能赢的环境。不是"我逼自己不看盘"——是"我的手机上没有可以看盘的东西"。不是"我逼自己理性判断"——是"收盘后脚本把数据喂给我，我做判断，盘中什么都不需要做"。
