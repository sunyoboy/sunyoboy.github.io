## 全景映射

```
生物系统                        学习系统                      你的工具
───────                        ───────                      ────
进食（咀嚼→吞咽）  ─────────→  信息获取（拆解→筛选）          RSS/书源/搜索
消化（胃酸+酶分解） ─────────→  信息加工（批判+框架拆解）      费曼笔记/七层框架
吸收（小肠→血液）  ─────────→  知识内化（连接→存入长期记忆）   Anki间隔重复/写给别人看
同化（氨基酸→肌肉） ─────────→  构建体系（原子概念→思维模型）   Zettelkasten/技能树
代谢（ATP供能）    ─────────→  知识输出（决策/写作/编码）      复盘/周报/量化脚本
排泄（过滤→排出）  ─────────→  遗忘清理（信号/噪音分离）      阅后即焚/定期整理
储存（糖原/脂肪）  ─────────→  记忆管理（工作/晶体知识）      PKM/笔记系统
饥饿感（激素信号） ─────────→  好奇心（知道该学什么）          问题清单/OKR
肠道菌群           ─────────→  社群学习（协作/讨论/教学）      Code Review/团队质疑
```

> 🎯 核心原则：**缺哪个环节，补哪个环节。** 大部分人不是信息不够——是消化不行（读了就忘），或者吸收不行（记了用不上），或者代谢不行（学了不做）。

## 一、进食 → 信息获取

### 咀嚼 = 拆解

胃没有牙齿。不嚼碎的东西，胃消化不了。**信息也一样。**

```
❌ 狼吞虎咽                       ✅ 细嚼慢咽
───────                          ───────
一篇长文从头刷到尾                第一遍扫标题和小标题
收藏 = 吃了（其实没吃）            用自己的话写一句话摘要
一小时刷 50 篇                    一小时读 2 篇 + 写笔记
```

**操作规则：**

| 吃什么 | 怎么嚼 | 嚼多久 |
|------|------|:--:|
| 行业研报/深度文章 | 读完写 3 句话：①核心论点 ②我信不信 ③哪部分和我有关 | 15-30min |
| 碎片信息（推/公众号）| 摘 1 个关键词存入待消化清单 | 30s |
| 代码/技术文档 | 跑通一个最小示例才算读完 | 1-2h |
| 书 | 读完一章 → 写 200 字总结 → 再读下一章 | 每天 30min |

### 吞咽 = 决定处理

吞咽是不可逆的——食物进入胃就不能回头了。**不是每条信息都值得吞咽。**

```
吞咽前的四道闸门：
1. 这条信息和我当前的 OKR/问题清单 有关吗？  → 无关 → 吐出
2. 这条信息的来源可信吗？                      → 不可信 → 吐出
3. 我能用一句话说清"它说了什么"吗？           → 不能 → 再嚼
4. 一周后我还会关心这个吗？                    → 不会 → 吐出
```

## 二、消化 → 信息加工

### 胃酸 = 批判性思维

胃酸（pH 1.5-3.5）能溶解几乎所有有机物。**批判性思维就是你的胃酸——它把复杂信息分解成可吸收的基础单元。**

```
消化四问（每拿到一条信息必问）：
┌─────────────────────────────────────────┐
│ ① 它的结论建立在什么假设上？那个假设成立吗？    │
│ ② 有没有反例？它为什么没提？                  │
│ ③ 如果它是错的，最可能错在哪？                 │
│ ④ 我能用自己经历中的一个例子验证或反驳它吗？    │
└─────────────────────────────────────────┘
```

### 酶 = 思维模型

胃酸把蛋白质分解成多肽，酶再把多肽分解成氨基酸。**思维模型就是你的酶——把复杂问题分解成可处理的原子概念。**

| 生物酶 | 思维模型 |
|------|------|
| 蛋白酶（分解蛋白质）| 第一性原理（分解到不能再分解的基本事实）|
| 脂肪酶（分解脂肪）| 矛盾论（抓主要矛盾和矛盾的主要方面）|
| 淀粉酶（分解碳水）| 系统思维（输入→过程→输出→反馈）|
| 纤维素酶（人类没有）| 正是"别人能理解但你不能"的盲区 → 需要肠道菌群 |

> ⚠️ 纤维素酶人类没有——牛有。**有些东西你自己消化不了，必须借助他人（导师/社群/团队质疑）。** 承认这一点比假装全能高效得多。

## 三、吸收 → 知识内化

### 小肠绒毛 = 多点连接

小肠表面积约 250m²（一个网球场），靠的是绒毛结构。**记忆的本质也是表面积——和已有知识连接的节点越多，越不可能忘记。**

```
一条新知识进入大脑：

连接点 < 3      → 孤立记忆 → 24h 内遗忘率 > 70%
连接点 3-10     → 理解级别 → 能用但不熟练
连接点 10-50    → 掌握级别 → 能教给别人
连接点 > 50     → 肌肉记忆 → 自动调用，无需意识
```

**操作规则：学到一个新东西后，48h 内至少做 3 个连接：**

1. 和已有知识的连接——"这让我想起 ___"
2. 和个人经历的连接——"我遇到过 ___"
3. 和行动的连接——"下周我可以用在 ___"

### 吸收效率曲线

```
吸收率
100% ┤          ● 教给别人
     │        ●   （学习金字塔：教授他人 = 90%）
 80% ┤      ● 动手做
     │    ●     （实践 = 75%）
 60% ┤  ● 讨论
     │ ●       （讨论 = 50%）
 40% ┤● 只看只听
     │         （被动学习 = 5-10%）
     └──────────────────────→
```

> 费曼学习法本质上就是强迫自己把食物嚼碎到别人能吃的程度——**教别人的同时自己在吸收第二遍。**

## 四、同化 → 构建体系

### 氨基酸 → 蛋白质 → 肌肉 = 原子概念 → 思维模型 → 技能

身体用 20 种氨基酸构建了全部蛋白质，再构建全部组织。**你的知识体系也一样——用有限的原子概念，构建无限的思维模型和技能。**

```
原子概念（~50-100个）        思维模型（~20-30个）        技能（~5-10个）
─────────────────          ─────────────────          ─────────────
复利                        价值投资                   选股
概率                        Bayesian更新               决策
反馈循环                     PDCA                       复盘
供需                         市场微观结构               做T
边际效应                     成本收益分析               仓位管理
杠杆                         系统设计                   量化脚本
```

### 同化不良的信号

| 症状 | 诊断 | 处方 |
|------|------|------|
| "我看过这个但说不清" | 停留在进食阶段，没消化 | 写 200 字费曼笔记 |
| "道理都懂但做不到" | 消化了但没吸收——知识在血液里，没进细胞 | 找 1 个场景立刻用 |
| "只会一种方法解所有题" | 同化不够——原子概念太少，搭不出新模型 | 跨界阅读，引入新氨基酸 |
| "复盘时总能总结，盘中照旧犯" | 系统1没收到更新——肌肉没长出来 | 条件单/环境设计代替意志力 |

## 五、代谢 → 知识输出

### 有氧代谢 vs 无氧代谢 = 深度输出 vs 应急反应

```
有氧代谢（深度输出）            无氧代谢（应急反应）
───────────────              ───────────────
需要氧气 + 时间                 不需要氧气，立刻
1分子葡萄糖 → 36分子ATP         1分子葡萄糖 → 2分子ATP
效率高，持续久                  效率低，有副产物（乳酸）
适合：周报/量化回测/写作         适合：盘中条件单触发/止损执行
```

**你的复盘 = 有氧代谢，你的条件单 = 无氧代谢。** 两者不是竞争关系——是互补关系。条件单负责盘中应急，复盘负责盘后深度。不要把无氧当日常（盘中分析），也不要用有氧做应急（止损还慢慢想）。

### 代谢废物 = 认知负荷

```
乳酸堆积 → 肌肉酸痛 → 需要休息恢复
认知负荷堆积 → 决策疲劳 → 需要睡眠和间隔

→ 这和 CLAUDE.md 的精力曲线完全对应：
  🔋🔋🔋 高能量 = 有氧代谢窗口（深度写作/量化编码）
  🔋 低能量 = 只能无氧（跑脚本/Git提交）
  💤 不做事 = 恢复期（代谢废物清理）
```

## 六、排泄 → 遗忘与清理

### 遗忘不是 bug，是 feature

大肠回收水分，排出不能消化的纤维和代谢废物。**大脑也需要排泄——遗忘不重要的东西，才能给重要的东西腾出空间。**

| 需要排泄的 | 怎么排 |
|------|------|
| 上周公众号的 150 篇文章 | 不读第二遍的，一周后自动删除 |
| 亏钱的执念（"要是当时___就好了"）| 复盘写下来 → 提炼一条规则 → 放下 |
| 过时的策略（6月的入场价到7月可能失效）| 每月第一周重跑框架，旧计划归档 |
| 未完成的 TODO 堆积 | 每周日清理——30% 直接删除，50% 合并，20% 保留 |

### 便秘式学习

```
症状：收藏 500 篇未读、TODO 列表 80 条未完成、桌面 300 个文件未整理
病因：只进不出。大肠塞住了，新的营养进不来。
处方：每个周末删除 20% 的"待处理"——不是做完，是删掉。
```

## 七、储存 → 记忆管理

### 糖原（快速能量）vs 脂肪（长期储存）

```
肝糖原                        体脂肪
──────                        ──────
立即可用                       需要时间动员
容量有限（~400kcal）           容量几乎无限
用不完→转脂肪                  用不完→继续堆积
对应：工作记忆/近期笔记          对应：晶体知识/Zettelkasten
```

**你不需要把所有东西都放在"眼前"（工作记忆）。好的笔记系统就是你的脂肪组织——平时不占地方，需要时分解调用。**

| 存哪里 | 存什么 | 多久调用一次 |
|------|------|:--:|
| 大脑（糖原）| 止损价、入场价、当天计划 | 每天 |
| 笔记/CLAUDE.md（脂肪）| 框架规则、费曼笔记、复盘记录 | 每周 |
| 知识库/脚本（骨）| 长期不变的原理和自动化工具 | 每月 |
| 外部（排泄）| 删掉的东西 | 永不再用 |

## 八、饥饿感 → 好奇心

### 两种饥饿

```
身体饥饿                      认知饥饿
───────                      ───────
ghrelin 激素上升              问题清单堆积
"我想吃东西"                   "我想知道为什么"
吃饱后停止                    理解后停止
营养不良→渴望特定食物           能力缺口→渴望特定知识
```

**你的问题清单就是 ghrelin。没写问题清单 = 不知道饿 = 不知道学什么。**

### 假饿 vs 真饿

| 假饿（该忽略）| 真饿（该喂）|
|------|------|
| "别人都在学这个" → 社群压力，不饿 | "为什么我的框架没预判到这次暴跌？" → 真需求 |
| "这个很火" → 新鲜感，不饿 | "量化回测卡在这一步，需要补 pandas" → 真缺口 |
| 刷屏的焦虑感 → 信息噪音 | "上周复盘三次提到同一个问题" → 真信号 |

## 九、肠道菌群 → 社群学习

### 你自己消化不了的，菌群帮你分解

人类没有纤维素酶——但肠道菌群有。它们帮你分解纤维素，产生短链脂肪酸供你吸收。**你自己理解不了的东西，社群帮你分解。**

```
肠道菌群                      学习社群
───────                      ──────
多样性越丰富，消化能力越强       背景越多元，理解越全面
菌群失衡 → 消化障碍             同质化 → 集体盲区
益生菌 → 补充缺失的分解酶        导师/专家 → 提供你缺少的视角
发酵 → 产生你无法合成的营养       讨论 → 碰撞出你一个人想不到的洞见
```

**对你操作系统的映射：**

| 肠道菌群机制 | 你的学习工具 |
|------|------|
| 纤维素分解菌（人类没有）| 团队质疑七层过滤——别人看到你看不到的盲区 |
| 益生菌补充 | 读反对意见/做空报告——刻意引入对立视角 |
| 菌群失衡恢复 | 每季度引入一个新领域的人聊天 |
| 短链脂肪酸（菌群代谢物）| Code Review / 周报被同事批注——间接产物往往是最高营养 |

## 十一、免疫系统 → 信息鉴别

> 消化系统管「吃什么」，免疫系统管「什么有毒」。两者合作 = 完整的认知安全体系。

### 双层防御架构

```
皮肤/黏膜（物理屏障）          第一道防线
      ↓
先天免疫（快速，非特异性）       看到就反应
      ↓
获得性免疫（慢，精准制造抗体）    分析后精准打击
      ↓
免疫记忆（下次秒杀）            不吃第二遍亏
```

| 免疫机制 | 认知映射 | 你的工具 |
|------|------|------|
| **皮肤/黏膜** | 信息源过滤——不进体内 | 不装行情APP、不关注财经八卦号 |
| **先天免疫** | 直觉判断——"这人说话不靠谱" | 一眼扫过，不加思考就直接关掉 |
| **获得性免疫** | 框架验证——逐层分析 | 七层过滤框架 |
| **记忆B/T细胞** | 模式识别——"上次这种走法最后亏了" | 复盘记录，亏过的坑不再踩 |
| **自身免疫病** | 过度自责——攻击自己正常的决策 | 🔴 把正常的概率亏损当成自己的错 |
| **过敏反应** | 对无害信息过度反应 | 🔴 为无关新闻焦虑到睡不着 |

### 免疫失调的诊断

| 症状 | 免疫对应 | 处方 |
|------|------|------|
| 什么信息都信 | 先天免疫缺失 | 每条信息先问"最可能错在哪" |
| 什么信息都不信 | 自身免疫病——连正确的也攻击 | 引入外部验证（跑数据/问队友） |
| 对某些信息过度反应 | 过敏——看到"崩盘"两个字就恐慌 | 条件单隔离：把身体从过敏原前移开 |
| 同一个坑踩两次 | 免疫记忆缺失 | 写到复盘里，下次框架自动拦 |

### 疫苗原理 → 压力接种

```
疫苗 = 注入灭活病原体 → 免疫系统产生抗体 → 真遇到时已免疫

交易对应：
  小仓位试错 → 框架发现漏洞 → 修正 → 真暴跌时已免疫
  
  6/23 有色 -8.36% 就是你打的疫苗——
  下次再遇到板块暴跌，你知道框架会怎么说。
```

## 十二、肌肉生长 → 刻意练习

> 安德斯·埃里克森的刻意练习理论，和肌肉生长的生物机制完全同构。

### 增长三阶段

```
阶段一：超负荷刺激（>当前能力）
  → 肌纤维微撕裂
  → 信号：酸痛感

阶段二：修复重建（48-72h）
  → 卫星细胞激活
  → 蛋白质合成 > 分解
  → 信号：不再酸痛

阶段三：超量恢复（适应后更强）
  → 下一次能承受更大负荷
  → 信号：同样负重不再费力
```

### 学习精确映射

| 肌肉原理 | 学习法则 | 错误做法 |
|------|------|------|
| 超负荷（>1RM 的 80%）| 永远在舒适区边缘练习 | 反复做已经会的题 |
| 微撕裂 = 生长前提 | 犯错 = 学习前提 | 怕犯错所以不尝试 |
| 修复需要 48-72h | 间隔重复 > 集中突击 | 一天学 8 小时同一个东西 |
| 平台期 = 换动作 | 边际递减 = 换学习方式 | 只会看书，不动手 |
| 蛋白质 + 睡眠 = 合成代谢 | 费曼输出 + 睡眠 = 知识固化 | 学完不写、熬夜不睡 |
| 渐进超负荷（每次加 2.5kg） | 每次只加一点点难度 | 一下跳到完全不会的领域 |

### 你当前的训练计划

```
肌肉训练              你的学习对应
───────              ──────────
深蹲（大肌群）         量化回测（核心技能）
卧推（推力）           写作输出（表达力）
硬拉（全身）           复盘（整合力）
每周 ≥ 2 次            每周 ≥ 2 次深度块
每次 ≤ 90min           每次 ≤ 3h（周末上午）
```

**平台期自检：**

```
连续三周没有"搞不定的感觉" → 平台期 → 加重（加难度）
连续三周每次都很痛苦       → 超负荷过度 → 减重（降难度）
酸痛超过 72h 不消         → 受伤了 → 停止休息
```

## 十三、进化论 → 迭代优化

> PDCA 的本质 = 人工加速的自然选择。

### 三层机制

```
变异（随机尝试新策略）
  ↓
选择（市场用盈亏投票）
  ↓
保留（盈利的留下，亏损的淘汰）
  ↓
重复（下一轮从更高的起点开始）
```

| 进化机制 | 交易/学习映射 | 你的工具 |
|------|------|------|
| 随机变异 | 尝试新策略、新入场条件 | 量化回测的参数搜索 |
| 自然选择 | 市场用盈亏投票 | 实盘盈亏数据 |
| 适者生存 | 正期望的策略保留 | 复盘确认有效规则 |
| 灭绝 | 止损——主动结束无效仓位 | 不止损 = 不让自然选择工作 |
| 红皇后效应 | 市场在进化，不迭代 = 退步 | 每月重跑框架，更新参数 |
| 生态位分化 | 找到你的策略定位 | 不打别人的牌（不追科创就不追）|
| 马太效应 | 正反馈——盈利→自信→更好决策 | 破坏循环：亏损→止损→不报复 |
| 间断平衡 | 长时间稳定 + 短期剧烈变化 | 6月大部分时间不动↔6/26一天入两笔 |

### 进化速度公式

```
进化速度 ∝ 变异率 × 选择压力 × 迭代频率

个人进化：
  变异率 = 尝试新东西的频率
  选择压力 = 复盘的质量（诚不诚实？）
  迭代频率 = PDCA 的周期（天/周/月）
```

**加速进化的三个杠杆：**

| 杠杆 | 怎么拉 |
|------|------|
| 提高选择压力 | 把交易结果用数字记录——盈亏率、胜率、最大回撤。数字比感觉残酷 |
| 缩短迭代周期 | 日复盘 > 周报 > 月回顾。越快知道结果，越快修正 |
| 增加有益变异 | 每周读一本不相关的书、每季度学一个新领域——跨界变异往往是最优解 |

## 十四、菌根网络 → 知识共生

> 森林地下存在巨型的真菌网络——不同树种通过它交换碳、氮、磷和信息。一棵树无法独活。

### 网络结构

```
                 老树（导师）
                /  |  \
          真菌───真菌───真菌（连接层 = 笔记系统/社群）
              /    |    \
         幼苗A  幼苗B  幼苗C（学习者）
              \    |    /
              垂死树（复盘——最后馈赠）
```

| 菌根网络机制 | 学习网络对应 |
|------|------|
| 碳从老树→幼苗 | 导师给新人输送经验 |
| 垂死树把所有养分分给邻居 | 复盘失败 → 团队全部受益 |
| 不同树种通过同一个菌根网络共生 | 跨学科的知识在同一套笔记系统中连接 |
| 菌根真菌完成物质交换 | Zettelkasten / 双链笔记 = 你的真菌层 |
| 砍断菌根网络 → 整片森林变弱 | 闭门造车 → 认知持续退化 |
| 最大的树 = hub 节点 | 你知识库中最密集连接的文档 = hub |

### 你的菌根网络搭建指南

```
真菌层（连接器）：
  → VitePress 知识库（双链笔记）
  → 复盘文档之间的交叉引用
  → 问题和答案之间的双向链接

树节点（内容）：
  → 老树 = CLAUDE.md / 方法论 / 交易纪律（沉淀最久的文档）
  → 幼苗 = 每日复盘 / 问题记录（持续生长中）
  → Hub = 决策方法论（被引用最多的文档）

营养流（输入/输出）：
  → 碳流入 = 阅读/搜索/研究
  → 碳流出 = 写作/编码/交易决策
  → 馈赠 = 复盘记录（未来你回头看时，最值钱的就是这些）
```

## 十五、卖油翁 → 手熟

> 欧阳修《归田录》· 刻意练习的最早版本

```
取一葫芦置于地，以钱覆其口，徐以杓酌油沥之，自钱孔入，而钱不湿。
因曰："我亦无他，惟手熟尔。"
```

油穿过铜钱孔流进葫芦，铜钱一滴不沾。"没别的——手熟。"

### 刻意练习的三要素

| 卖油翁 | 刻意练习 |
|------|------|
| 每天倒油 | 重复 → 肌肉记忆 = 系统1自动化 |
| 铜钱孔越来越小 | 渐进超负荷 → 难度递增 |
| 钱不湿 = 即时反馈 | 错了铜钱就湿 → 每倒一次都是检验 |

### 对交易的映射

```
卖油翁倒油千万次 → 钱不湿
你填复盘千万次   → 盘中不慌

条件单自动执行 = 你的"钱不湿"
不是因为今天状态好——是因为手熟。练出来的。
```

> 🎯 对应肌肉生长模型。卖油翁是执行层——解决"知道了但做不到"。复盘→纠正→重复→自动化，四个字：惟手熟尔。

## 十六、庖丁解牛 → 结构思维

> 《庄子·养生主》· 第一性原理的东方原型

```
始臣之解牛之时，所见无非牛者。三年之后，未尝见全牛也。
方今之时，臣以神遇而不以目视。

"依乎天理，批大郤，导大窾，因其固然。"
```

三年后看牛不是牛——看到骨架缝隙。闭着眼都知道刀往哪走。别人的刀一个月换一把，他的用了十九年还像新磨的。

### 三个层次

| 阶段 | 认知 | 交易对应 |
|------|------|------|
| 所见无非牛者 | 看表面——"这是牛" | 看K线——"涨了买跌了卖" |
| 未尝见全牛 | 看结构——"骨架和缝隙" | 看偏离度/金叉/量能——"为什么涨" |
| 以神遇不以目视 | 直觉——闭眼也能走刀 | 条件单自动执行——不需要看盘 |

### 庖丁 vs 卖油翁

```
卖油翁 = 手熟（执行层）         ← 同一动作千万次
庖丁解牛 = 心通（理解层）       ← 先花三年看结构，再下刀

卖油翁：油不洒                  复盘形成肌肉记忆
庖丁解牛：刀不磨损              七层框架 = 牛的骨架结构，刀顺着缝走
```

> 🎯 卖油翁解决执行问题，庖丁解牛解决理解问题。你需要的不是其中一只——是两只都学。复盘练手熟，框架找骨缝。

## 十七、第一性原理 → 根基层

> 把问题拆到不能再拆的基本事实，从那里重新推理。不借用别人的结论。

### 两种思考方式

```
类比思维                          第一性原理
───────                          ───────
"这个和上次一样"                   "这个由哪几个基本要素构成？"
抄答案                            自己推导
快但有盲区                         慢但不出错
看到牛是牛                         看到骨架缝隙
```

### 交易中的例子

```
类比思维：
  "科创50今天 +3.84%，感觉还能涨"

第一性原理：
  科创50 PE 239x 100分位
  + 成交额 3.5万亿
  + 电子利润 +665%，订单排到2028
  + 一周四个科技大会 = 情绪冲刺尾声
  → 基本面最强但估值最贵 → 等中报验证后再说
  → 七层框架：第二层通过，第一层🟡，第四层🔴
```

### 拆解的自检标准

一个问题拆到位了吗？自检：**还能用"为什么"再问一层吗？** 问到不能再问——就是第一性。

```
"我要买半导体ETF"
  → 为什么？"因为涨了"              ← 还能拆
  → 因为三星SK扩产+长鑫存储          ← 还能拆
  → 因为改变了晶圆制造设备的供需       ← 还能拆
  → 全球产能缺口 23.5%，订单排到2028   ← 这是第一性。供需缺口
```

### 三层认知体系

```
卖油翁     → 手熟 = 系统1自动化 = 执行层     （不复盘就不会进步）
庖丁解牛   → 看结构 = 框架思维 = 理解层      （不看结构刀就磨损）
第一性原理 → 拆到底 = 不可再分 = 根基层      （不拆到底就有盲区）
────────────────────────────────────────────
三只合在一起 = 完整的认知→理解→执行体系
```

**七层框架就是你的第一性原理工具。** 每一层都在问不可再拆的基本事实。不是"我觉得"——是框架在回答。

## 全景总图

### 五个仿生模型

```
        免疫系统 ──→ 信息鉴别·安全
           ↑
消化系统 ──→ 分解吸收·理解
           ↓
      肌肉生长 ──→ 刻意练习·技能
           ↓
        进化论 ──→ 迭代优化·策略
           ↓
      菌根网络 ──→ 知识共生·连接

五个缺一不可：
  没有免疫 → 吃进毒药，框架被噪音攻破
  没有消化 → 读再多不吸收，读过等于没读
  没有肌肉 → 看懂了做不到，道理和行动之间有鸿沟
  没有进化 → 老策略应对新市场，逐渐失效
  没有菌根 → 孤岛式学习，盲区永远看不到
```

### 三层古典智慧

```
第一性原理（根基层）
  拆到底，不留盲区。七层框架的每一层都是不可再拆的基本问题。
      ↓
庖丁解牛（理解层）
  看结构，刀不磨损。顺着市场骨架走，不和趋势对抗。
      ↓
卖油翁（执行层）
  手熟，钱不湿。复盘练到肌肉记忆，条件单自动执行。
```

### 八者统一

| 模型 | 层级 | 核心问题 | 你的工具 |
|------|:--:|------|------|
| 第一性原理 | 根基层 | 为什么？拆到底 | 七层框架每一层的底层逻辑 |

| 免疫系统 | 安全层 | 什么有毒？ | 信息源过滤+框架验证 |
