## 数据 API 参考

> 用于自动化抓取行情数据，标注了不同环境下的可用性。

### 指数数据

| API | 端点 | Python 脚本 | WebFetch | 说明 |
|------|------|:---:|:---:|------|
| **腾讯财经** | `qt.gtimg.cn/q=<code>` | ✅ | ✅ | 主力接口，7大指数+个股均可用，返回 GBK 编码 |
| 新浪财经 | `hq.sinajs.cn/list=...` | ✅ | ❌ 域名受限 | 备选，UTF-8 编码但需 referer |
| 东方财富 | `quote.eastmoney.com/...` | ✅ | ❌ 域名受限 | 网页端，非结构化 |
| 天天基金 | `fundf10.eastmoney.com/...` | ✅ | ❌ 域名受限 | ETF 净值历史 |

### 个股/ETF 数据（补充指数外的标的）

腾讯财经 API 同样支持个股和 ETF，代码规则：

| 类型 | 代码格式 | 示例 |
|------|------|------|
| 沪市 ETF | `sh` + 6位代码 | `sh510500`（中证500ETF） |
| 沪市股票 | `sh` + 6位代码 | `sh601899`（紫金矿业） |
| 深市 ETF | `sz` + 6位代码 | `sz159915`（创业板ETF） |
| 深市股票 | `sz` + 6位代码 | `sz002230`（科大讯飞） |

### 腾讯财经 API 返回格式

```
v_sh510500="1~中证500ETF~510500~9.008~...~2.20~..."
               代码   名称    价格      ...    涨跌幅
```
- 字段索引：`[1]` 名称, `[3]` 现价, `[4]` 昨收, `[32]` 涨跌幅

###  Claude Code 环境可用性说明

| 工具 | 国内财经站点 | 说明 |
|------|:---:|------|
| `WebSearch` | ✅ 间接可用 | 搜索结果会包含 eastmoney/10jqka/stockstar 等站点的数据摘要 |
| `WebFetch` | ❌ 多数受限 | `eastmoney.com`、`sina.com.cn`、`sinajs.cn` 均被企业安全策略拦截 |
| `Bash` → `python3` | ✅ | 通过本地脚本调用腾讯财经 API，最可靠 |

> 🔑 **结论**：自动化抓取用 `scripts/fetch-market-data.py`（腾讯财经），手工补充数据用 `WebSearch` 搜索具体标的。
