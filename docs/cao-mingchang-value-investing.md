# 曹名长 · 深度价值投资框架

> 2026-07-08 · 璞桥资产创始人 · 19年公募年化~15%
>
> 来源：聪明投资者深度访谈

---

## 一、核心理念

### 价值投资 = 增长 + 估值

```
两个极端:
  最成长端 → 看到未来能长成参天大树 → 现在多贵都可以买
  最低估值端 → 完全不增长也行 → 买得足够低+分红=好收益

他在低估值这一端: 「跨低一点的栏杆，越低越好」
→ 要现在就能算得清的确定性
→ 不赌「它能长成参天大树」
```

### 收益公式

```
年化回报 ≈ 股息率 + 内生增长

没有增长·10倍PE·全部分红 = 10%年化
分红5% + 增长5% = 10%年化
→ 两条路径都能到终点
```

### 两个陷阱

| 陷阱 | 特征 | 案例 |
|------|------|------|
| 价值陷阱 | PE便宜但业绩持续下滑 | 地产·需求断崖式下跌 |
| 成长陷阱 | 估值买贵·增长放缓就杀估值 | 高位追芯片 |

## 二、组合三类资产

| 类型 | 特征 | 仓位角色 | 我们的映射 |
|------|------|------|------|
| ① 高分红压舱石 | 增长慢·现金流稳·分红高 | 防守·长期持有 | 银行ETF·红利·30年国债 |
| ② 分红+稳健增长 | 内生增速+分红率=长期回报 | 核心仓位 | 食品饮料ETF |
| ③ 低估值+第二曲线 | 当前估值低·新业务有弹性 | 弹性部分 | 中证500·证券ETF·创新药 |

> 三类不互斥。熊市末期增配②③拿弹性，牛市提高①做防守。

## 三、安全边际不是固定标尺

```
2008 → 泥沙俱下·好公司坏公司一起跌·大胆买好的
2016 → 结构分化·蓝筹跌出便宜价·茅台300块十几倍PE
2021 → 核心资产阴跌·中小价值反而抗跌
现在 → 传统行业出清·估值仍被压制·布局窗口

"安全边际不是一把固定的尺子，你得先看这一轮下跌是什么性质"
```

---

## 四、分散不是不自信 · 是为了不下牌桌

```
集中 = 研究更深·不出事时收益更高
       但小概率事件一旦发生 → 永久出局

分散 = 承认没有绝对安全的资产
       "尾巴比你想得厚得多，出事的概率比大家以为的大"

→ 宁可少赚，保证永远在牌桌上
```

---

## 五、风险只有两种

```
① 估值买贵了 → 高位追进去
② 价值在负增长 → 生意本身在萎缩

股价短期波动 → 不是风险
```

## 六、关键金句

```
1. "买好的，不如买得好" — 价格比质地更重要
2. "跨低一点的栏杆" — 做自己能判断的、确定性强的事
3. "长期年化10%以上就可以了"
4. "赚钱是把一件事做好之后，自然而然的结果"
5. "基金经理和持有人是相互成全的" — 信任不够·收益转不到自己身上
6. "逆风是淬炼场，复利不只在业绩，也在投资体系"
```
