# 投资心智

> 投资是一场修行，认知决定收益。

## 尹烨：改变自己

> 世界上最傻的事，就是宁愿忍受几十年不快乐，也不愿花一年时间改变自己。

你知道这个世界上最傻的事是什么吗？他可以忍受几十年不快乐的人生，却不愿意奋发图强，花一年时间让自己改变一下。他不懂得什么叫投资自己，他浪费的是自己的生命。他明明知道其实有本事去做一些事情。所以你相信，其实人人都可以改变，前提是你什么时候愿意开始改变。

—— 尹烨（华大基因CEO）

**投资启示：**

- **投资自己是回报率最高的投资**：花一年学透一个行业、精进一项技能，复利远超任何股票
- **改变的门槛只在心理**：大部分人不做改变不是因为不能，而是因为不敢开始
- **"一年"是关键时间单位**：太短不足以质变，太长让人畏惧——一年恰好足够完成一次认知升级
- **知行合一**：知道和做到之间隔着执行力，知道该复盘、该做T计划，但能坚持执行才是分水岭

## 相关概念

- 刻意练习：一万小时定律的精华不在时间，在"刻意"
- 复利思维：每天进步1%，一年后是37.8倍
- 能力圈：巴菲特说"重要的是知道自己的能力边界"，而扩展边界需要投资自己
