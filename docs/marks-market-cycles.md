# 🔄 霍华德·马克斯 · 周期与第二层思维

> 橡树资本创始人。巴菲特说他的备忘录每篇必读。
>
> **市场周期不在"贵"和"便宜"之间摆动，而在"狂热"和"绝望"之间摆动。** 你的工作：不要在狂热时买，不要在绝望时卖。

---

## 一、第二层思维

| 第一层思维（普通人） | 第二层思维（马克斯） |
|------|------|
| "这家公司好，买" | "所有人都觉得好，已经太贵了，卖" |
| "经济衰退，卖股票" | "衰退已经被定价了，现在买" |
| "科创跌了 20%，太危险了" | "科创跌了 20%，capitulation 信号出现，该进场了" |

> **别人都在想同一件事的时候，这件事的价格已经反映到市场里了。你要想的是"别人没想到什么"。**

---

## 二、钟摆理论

```
悲观 ◄──────────────────●──► 乐观
     恐惧·绝望·恐慌    理智·均衡  贪婪·狂热·FOMO

钟摆很少停在中间——
  大部分时间在两端之间摆动
  从一端摆到另一端的速度，比你想象得快
```

| 钟摆位置 | 市场表现 | 该做的事 |
|------|------|------|
| 极左（绝望） | 好公司便宜也没人买 | 格伦姆 75% 股票 |
| 中间（理智） | 正常 | 格伦姆 50/50 |
| 极右（狂热） | 差公司贵了也有人抢 | 格伦姆 25% 股票 |

---

## 三、价格与价值的关系

```
价值 = 一条水平线（缓慢上升）
价格 = 一条曲线（围绕价值剧烈波动）

价格的波动幅度远大于价值的变动幅度。
  股市一年涨跌 ±20-30%，
  企业盈利一年变动 ±5-10%。
  → 多出来的波动全是情绪。
```

> 7/14-7/21 的五连跌 -20%——企业一天没变，是钟摆摆到了极左。7/21 科创 +10.73%——企业还是没变，是钟摆从极左往回摆了。**你的 capitulation 判断框架，本质就是在找钟摆的拐点。**

---

## 四、和格伦姆的分工

| | 格伦姆 | 马克斯 |
|------|------|------|
| 核心工具 | 安全边际（价值） | 钟摆理论（情绪） |
| 买点 | 价格 < 价值 × 0.5 | 钟摆在极左 |
| 卖点 | 价格接近价值 | 钟摆在极右 |
| 基础 | 定量 | 定性 |

> **格伦姆给你价格锚，马克斯给你情绪锚。** 便宜还不够——还要确认不是"因为没人要才便宜"。capitulation 三信号就是马克斯思维：银行补跌 = 钟摆极左的最后确认。

## 五、核心备忘录观点

| 备忘录 | 核心观点 | 交易映射 |
|------|------|------|
| 《他们又来了》 | 泡沫的共同特征：每个人都相信"这次不一样" | 科创+10.73% 不追——就是"这次不一样"的反面 |
| 《敢于伟大》 | 超额回报来自做别人不做的事 | 五连跌时做回补计划 = 别人在逃你在算 |
| 《在不确定中前行》 | 不确定是常态·关键是管理风险而不是预测 | capitulation 框架不是预测，是验证 |

## 六、速查卡

| 当你… | 马克斯说 |
|------|------|
| 觉得"这次不一样" | 每次都不一样——每次结果都一样（钟摆会回头） |
| 想追涨时 | 问自己：钟摆在左边还是右边？ |
| 恐慌时 | 问自己：这是绝望的极左点吗？有信号吗？ |
| 看到全市场都在买同一板块 | 走到钟摆另一头去 |
