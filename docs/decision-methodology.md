## 总览：决策链路

```
任何一个"要不要买/卖"的问题，按以下顺序逐层过滤：

第一层：现在是什么市场状态？            → 高潮/正常/恐慌
第二层：具体标的技术面是什么信号？        → 金叉/死叉/偏离度/振幅
第三层：我的仓位是否允许这个操作？        → 单品种上限/板块合计/现金水位
第四层：加仓条件满足了几个？             → 五项条件，至少三项
第五层：全球有没有更便宜的替代？         → A股/港股/美股/日股比价
第六层：当前月份的季节效应是什么？        → 6月不重仓，等7月

六层全部通过 → 执行
任意一层不通过 → 不执行
```

## 第一层：市场状态判断

### 核心指标：成交额情绪指数

```
成交额情绪 = 今日成交额 ÷ 近20日均成交额
```

| 情绪区间 | 数值 | 含义 | 允许的操作 |
|------|:---:|------|------|
| 🟢 正常 | < 1.3 | 市场情绪平稳 | 可按计划操作 |
| 🟡 偏热 | 1.3 ~ 1.5 | 情绪升温，追涨风险加大 | 只减不增 |
| 🔴 高潮 | > 1.5 | 情绪亢奋，次日回吐概率极高 | **禁止开新仓** |

### 验证案例

```
2026-06-22: 成交额情绪 1.27（接近偏热）
  → 当日非银+5.86%、有色+3.93%
  → 按规则：🟡 偏热，不追
  → 次日：有色 -8.36%
  → 验证：✅ 偏热信号有效
```

### 辅助指标：三日振幅

```
三日振幅 = (3日最高价 - 3日最低价) ÷ 3日前收盘价
```

| 振幅区间 | 含义 | 允许的操作 |
|------|------|------|
| < 8% | 正常波动 | 可操作 |
| 8% ~ 15% | 🟡 偏高，谨慎 | 只观察，不动手 |
| > 15% | 🔴 极端波动 | **禁止开仓** |

### 辅助指标：追涨惩罚率

```
追涨惩罚率 = 前一交易日涨停股今日平均涨幅
若 < 0% → 昨天追涨停的人今天全亏 → 市场不奖励追涨
```

## 第二层：标的技术诊断

### 核心工具：MA5 偏离度（三维指标）

> 运行 `python3 scripts/ma5-deviation.py` 输出

```
维度一：偏离方向   = (现价 - MA5) ÷ MA5 — 价格相对短期均线的位置
维度二：持续时间   = 连续偏向天数 + 偏离趋势（扩大/收窄）
维度三：价格位置   = MA20/MA60 + 20日高低点 + 量能
```

### 信号解读矩阵

| 偏离度 | 方向 | 倾向 | 含义 |
|:---:|------|:---:|------|
| ±2% 以内 | 🟢 贴合 | 均衡 | 价格围绕均线正常波动，无方向性信号 |
| -2%~-5% | 🟡 负偏 | 偏冷 | 短期超卖，不宜追空但也不急于抄底 |
| -5%~-10% | 🔵 负偏 | 冰点 | 严重超卖，可能反弹但趋势未稳 |
| -10%+ | 🔴 极值 | 恐慌 | 可能触发止损/减仓 |
| +2%~+5% | 🟡 正偏 | 偏热 | 短期过热，等待回踩 |
| +5%+ | 🔴 正偏 | 高潮 | 追涨风险极大 |

### 均线组合信号

| 信号 | 条件 | 含义 |
|------|------|------|
| **金叉▲** | MA5 上穿 MA20 | 短期走强，可观察做多 |
| **死叉▼** | MA5 下穿 MA20 | 短期走弱，不开新仓 |
| **多头排列** | MA5 > MA20 > MA60 | 中期上升趋势 |
| **空头排列** | MA5 < MA20 < MA60 | 中期下降趋势 |

### 量能辅助

```
量能 = 5日均量 ÷ 20日均量
> 1.5 → 放量，关注方向
< 0.6 → 缩量，底部信号
0.8-1.2 → 正常
```

### 6/24 实际诊断结论

| 标的 | 偏离度 | 信号 | 均线 | 判定 |
|------|:---:|------|:---:|:---:|
| 中证500ETF | +1.2% | 🟢 贴合 | 金叉▲多头 | 唯一候选 |
| 黄金ETF | -2.4% | 🟡 偏冷 | 死叉▼ | 不动 |
| 科大讯飞 | +0.03% | 🟢 贴合 | 死叉▼ | 不动 |
| 天山铝业 | -2.8% | 🟡 偏冷 | 死叉▼ | 减仓线附近 |
| 紫金矿业 | **-5.3%** | 🔵 冰点 | 死叉▼ | 远离 |
| 有色ETF | -3.0% | 🟡 偏冷 | 金叉▲ | 不动 |

## 第三层：仓位合规检查

> 运行 `python3 scripts/discipline-check.py --portfolio` 输出

### 四条红线

| 指标 | 计算公式 | 绿灯 | 黄灯 | 红灯 |
|------|------|:---:|:---:|:---:|
| **单品种仓位** | 单一持仓 ÷ 总资产 | ≤ 10% | 10-20% | > 20% |
| **关联品种合计** | 同板块持仓 ÷ 总资产 | ≤ 20% | 20-30% | > 30% |
| **现金水位** | 现金 ÷ 总资产 | ≥ 50% | 40-50% | < 40% |
| **杠杆** | 融资余额 ÷ 净资产 | 0 | 0 | 0（唯一正确值） |

### 波动率附加约束

```
近20日波动率 > 3% → 该品种标记为"高风险"
→ 不加仓
→ 如果已有持仓 + 仓位 > 5% → 考虑减半
```

### 6/24 实际状态

```
单品种仓位: 全部 🟢（最高科大6.0%）
板块合计:   全部 🟢（最高AI科技6.0%）
现金水位:   84.8% 🟡（偏保守但不违规）
杠杆:       0 🟢
波动率:     6/7品种 >3% 🔴→ 不加仓
```

## 第四层：加仓条件清单

### 五条件必须至少满足三项

| # | 条件 | 指标 | 阈值 | 权重 |
|:---:|------|------|:---:|:---:|
| 1 | **成交额正常** | 成交额情绪 | < 1.3 | ⭐⭐⭐ |
| 2 | **振幅收敛** | 三日振幅 | < 8% | ⭐⭐⭐ |
| 3 | **波动率回落** | 20日波动率 | < 3% | ⭐⭐ |
| 4 | **缩量确认** | 连续缩量天数 | ≥ 3天 | ⭐⭐ |
| 5 | **反弹稳固** | 领涨指数次日不破关键位 | 次日确认 | ⭐⭐ |

> ⚠️ 条件 1（成交额情绪）是**必要条件**——它不满足，其他全满足也不加。

### 6/24 评审

| # | 条件 | 6/24 值 | 判定 |
|:---:|------|:---:|:---:|
| 1 | 成交额情绪 < 1.3 | **1.10** | ✅ 1/5 |
| 2 | 三日振幅 < 8% | 9-10% | ❌ |
| 3 | 波动率 < 3% | 3-9% | ❌ |
| 4 | 缩量 ≥ 3天 | 第2天 | ❌ |
| 5 | 科创次日不破1950 | 待明天 | ❌ |

> **结论：1/5 → 不加仓。** 这就是"纪律优先不加仓"的数学依据。

## 第五层：全球资产比较

### 估值分位数比较

| 市场 | 代表指数 | 估值分位 | 判定 | 操作 |
|------|------|:---:|:---:|:---:|
| 中国A股 | 中证500 | 中等 | 正常 | 🟡 可操作（等信号） |
| 中国港股 | 恒生 | PE 11x, 低估 | 超卖 | 🟡 等企稳 |
| 美国 | 标普500 | **99%分位** | 极度高估 | ❌ 不碰 |
| 美国 | 纳斯达克 | **90%分位** | 高估 | ❌ 不碰 |
| 日本 | 日经225 | **历史新高** | 追涨 | ❌ 不碰 |
| 韩国 | KOSPI | **历史新高** | 追涨 | ❌ 不碰 |

### 比较逻辑

```
第一步：排除高潮位市场（估值 > 80%分位）    → 排除美股、日股、韩股
第二步：排除趋势破位的市场（死叉+头部形态）    → 港股待观察
第三步：在剩下的市场中找信号最干净的           → A股中证500
```

## 第六层：月度季节效应

### 各月历史规律

| 月份 | 平均涨跌 | 特征 | 策略 |
|------|:---:|------|------|
| 1-3月 | 春季躁动 | 偏强 | 可积极参与 |
| 4月 | 震荡下行 | 年报博弈 | 谨慎 |
| 5月 | +0.6% | 五月不穷 | 正常操作 |
| **6月** | **-0.9%** | **分化 17.9%，暴涨暴跌** | **轻仓等待** |
| 7月 | +0.9% | 七翻身较可靠 | **布局窗口** |
| 8-9月 | 分化 | 中报行情 | 个股为主 |
| 10-12月 | 年底行情 | 机构结算 | 逢低布局 |

### 6月特殊规则

```
1. 仓位上限自动下调 5%（从 ≤ 40% → ≤ 35%）
2. 单日盈利超过 2% 的板块，次日禁追
3. 最后一周：只做观察和减仓，不开新仓
4. 例外：季末最后 2 天如果缩量 + 恐慌 + 估值合理 → 可布局7月
```

```
                   成交额情绪正常(<1.3)    成交额情绪偏热(1.3-1.5)    成交额情绪高潮(>1.5)

MA5金叉 + 多头

                    中证500在8.75-8.79      等情绪回落再做              等下一个缩量日

MA5金叉
+ 振幅偏高(8-15%)
+ 波动率高(>3%)     🟡 只观察不动手           ❌ 不动                   ❌ 不动
                    6/24现状                 今天就是这样              不必解释

MA5死叉

                    趋势向下，不逆势          不接飞刀                 等反转信号
```
