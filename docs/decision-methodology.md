## 框架总览

```
任何一个"要不要买/卖"的问题，按以下顺序逐层过滤：

第零层：仓位合规     → 先看自己能不能买
第一层：市场状态     → 再看市场让不让你买
第二层：技术诊断     → 标的值不值得买
第三层：相关性校验   → 会不会和已有持仓撞车
第四层：事件日历     → 接下来几天有没有雷
第五层：全球比价     → 有没有更便宜的选择
第六层：月度效应     → 日历约束
第七层：执行指令     → 买多少、什么价位、哪里止损

所有七层通过 → 生成明确操作指令
任意一层不通过 → 停止，不操作
```

### 为什么是七层

```
原六层框架的缺口:
  ✗ 只能判断「不能买」，不能输出「怎么买」
  ✗ 仓位检查在第三层太靠后——应该第一道就过滤
  ✗ 买入触发机制缺失（入场价/止损/仓位计算）
  ✗ 没有相关性检查（新买入会不会让板块集中度失控）
  ✗ 没有事件日历避雷（FOMC/财报/长假）
  ✗ 第四层「五条件」实际上是与前三层的重复

修正:
  → 仓位合规提到第零层（最快过滤）
  → 市场状态和技术诊断层整合了振幅/波动率/缩量等指标
  → 新增第三层相关性、第四层事件日历
  → 新增第七层执行指令（最重要的一块补丁）
```

## 第零层：仓位合规

> **先看自己能不能买。** 不合规则后面全白算。

### 四条红线

| 指标 | 计算 | 绿灯 | 黄灯 | 红灯 |
|------|------|:---:|:---:|:---:|
| **单品种仓位** | 单一持仓 ÷ 总资产 | ≤ 10% | 10-20% | > 20% |
| **关联品种合计** | 同板块持仓 ÷ 总资产 | ≤ 20% | 20-30% | > 30% |
| **现金最低水位** | 现金 ÷ 总资产 | ≥ 50% | 40-50% | < 40% |
| **杠杆倍数** | 融资余额 ÷ 净资产 | **0** | 0 | 0 |

> 🔴 任何一项红灯 → **停止**。不做后续分析。这是第零层的价值——省掉后面六层的计算。

### 波动率附加约束

```
近20日波动率 > 3% → 该品种标记"高风险"
  → 不加仓
  → 若已持有 + 仓位 > 5% → 考虑减半
  → 这不是建议，是纪律
```

### 6/24 实际状态

```
单品种: 全部 🟢（最高科大讯飞 6.0%）
板块:   全部 🟢（最高AI科技 6.0%）
现金:   84.8% 🟡（偏保守但不违规）
杠杆:   0 🟢
波动率: 6/7品种 >3% 🔴 → 不加仓
```

## 第一层：市场状态

> **再看市场让不让你买。** 高潮日追涨 = 两天亏 10%。

### 核心指标：成交额情绪

```
成交额情绪 = 今日成交额 ÷ 近20日均成交额
```

| 区间 | 数值 | 含义 | 允许的操作 |
|------|:---:|------|------|
| 🟢 正常 | < 1.3 | 情绪平稳 | 可按计划操作 |
| 🟡 偏热 | 1.3 ~ 1.5 | 追涨风险加大 | **只减不增** |
| 🔴 高潮 | > 1.5 | 次日回吐概率极高 | **禁止开新仓** |

### 辅助指标：三日振幅

```
三日振幅 = (3日最高价 - 3日最低价) ÷ 3日前收盘价
```

| 区间 | 含义 | 允许的操作 |
|------|------|------|
| < 8% | 正常波动 | 可操作 |
| 8% ~ 15% | 🟡 偏高 | 只观察，不动手 |
| > 15% | 🔴 极端 | **禁止开仓** |

### 辅助指标：缩量确认

```
连续缩量天数 ≥ 3 → 抛压衰竭信号
3.74→3.44→3.28万亿，降速放慢 → 但还不够三天
```

### 辅助指标：追涨惩罚率

```
前一交易日涨停股今日平均涨幅 < 0%
→ 昨天追涨停的人今天全亏 → 市场不奖励追涨
```

### 验证案例

```
2026-06-22: 情绪 1.27 🟡 偏热
  → 当日非银+5.86%、有色+3.93%
  → 按规则：不追
  → 次日有色 -8.36%
  → ✅ 预警有效
```

## 第二层：标的技术诊断

> **这个标的值不值得买。** 不看基本面——只看技术信号。

### 核心工具：MA5 偏离度（三维指标）

> 运行 `python3 scripts/ma5-deviation.py` 输出

```
维度一：偏离方向   = (现价 - MA5) ÷ MA5
维度二：持续时间   = 连续偏向天数 + 偏离趋势（扩大/收窄）
维度三：价格位置   = MA20/MA60 + 20日高低点 + 量能
```

### 信号解读矩阵

| 偏离度 | 方向 | 倾向 | 含义 |
|:---:|------|:---:|------|
| ±2% 以内 | 🟢 贴合 | 均衡 | 价格围绕均线正常波动 |
| -2%~-5% | 🟡 负偏 | 偏冷 | 短期超卖，不急抄底 |
| -5%~-10% | 🔵 负偏 | 冰点 | 严重超卖，趋势未稳 |
| -10%+ | 🔴 极值 | 恐慌 | 可能触发止损/减仓 |
| +2%~+5% | 🟡 正偏 | 偏热 | 短期过热，等回踩 |
| +5%+ | 🔴 正偏 | 高潮 | 追涨风险极大 |

### 均线组合

| 信号 | 条件 | 含义 |
|------|------|------|
| **金叉▲** | MA5 上穿 MA20 | 短期走强，可观察 |
| **死叉▼** | MA5 下穿 MA20 | 短期走弱，不开新仓 |
| 多头排列 | MA5 > MA20 > MA60 | 中期上升 |
| 空头排列 | MA5 < MA20 < MA60 | 中期下降 |

### 量能辅助

```
量能 = 5日均量 ÷ 20日均量
> 1.5 → 放量，关注方向
< 0.6 → 缩量，底部信号
0.8-1.2 → 正常
```

### 低吸区间

```
当前价必须在低吸区间 ±1% 以内
例：中证500ETF 低吸区 8.75-8.79 → 当前 8.99，差 +2.5%
→ 不买。等。
```

|------|:---:|------|:---:|:---:|
| 中证500ETF | +1.2% | 🟢 贴合 | 金叉▲多头 | 唯一候选，等回踩 |

## 第三层：相关性校验

> **新买入会不会和已有持仓撞车？** 避免表面分散、实质上集中。

### 校验方法

```
1. 新标的属于哪个板块？
2. 已有持仓中同板块合计仓位 = ？
3. 加上新买入后，同板块仓位会不会 > 20%？
   会 → 减量 50% 或换板块

4. 新标的与任何已有持仓的相关系数 > 0.7？
   → 它们本质上在赌同一件事情 → 只保留一个
```

### 6/24 示例

```
假设想买中证500ETF：
  现有宽基持仓：中证500ETF + 卫星ETF = 5.0% → 🟢 不超20%
  与有色/黄金/AI科技板块无重叠 → 🟢

假设想买科创50ETF：
  现有AI科技持仓：科大讯飞 6.0%
  加上科创50ETF 2% → 8.0% → 🟢 未超标
  但科大讯飞和科创50的相关系数可能 > 0.7
  → 如果你已经有科大讯飞，科创50ETF 提供的分散效果有限
  → 不如中证500ETF（宽基中小盘，与科大讯飞重叠度更低）
```

| 7月 | +0.9% | 七翻身较可靠 | **布局窗口** |
| 8-9月 | 分化 | 中报行情 | 个股为主 |
| 10-12月 | 年底行情 | 机构结算 | 逢低布局 |

### 6月特殊规则

```
1. 仓位上限自动下调 5%（从 ≤ 40% → ≤ 35%）

4. 例外：季末最后 2 天若缩量+恐慌+估值合理 → 可布局 7月
```

## 第七层：执行指令

> **最重要的一层。** 前六层只告诉你能不能买——这层告诉你怎么买。

### 六个必须指定的数字

任何一个买入决策，必须同时写出以下六项，缺一不可：

| # | 必须指定 | 例（中证500ETF） |
|:---:|------|------|
| ① | **入场价** | 8.88元（限价单，不是市价单） |
| ② | **止损价** | 8.65元（入场-2.6%，MA20下方） |
| ③ | **仓位大小** | 1,200份 ≈ 10,700元（按公式计算） |
| ④ | **最大持仓周期** | 2周（做T）或 3个月（中期持有） |
| ⑤ | **止盈触发** | +5%（9.32元）或 MA5 死叉信号 |
| ⑥ | **条件单设置** | 入场后立即挂止损条件单 |

### 仓位计算公式

```
可买金额 = 总资产 × 单笔风险上限 ÷ 止损百分比

单笔风险上限 = 最多愿意为这次交易亏损总资产的多少
               保守: 1%  中性: 2%  激进: 3%

止损百分比 = (入场价 - 止损价) ÷ 入场价

例：总资产 283,000，风险上限 1%，入场 8.88，止损 8.65
   止损百分比 = 2.6%
   可买金额 = 283,000 × 1% ÷ 2.6% = 10,885元
   可买数量 = 10,885 ÷ 8.88 ≈ 1,200 份

含义：即使止损触发，也只亏约 2,830 元（总资产 1%）
      而不是之前随口说的 500 份（亏了也无感，但赚了也无感）
```

> 🔑 **不是"感觉应该买一点" → "买500份" → 是被公式算出来的。** 纪律的最高形式不是"管住手"，是"手被公式管住了"。

### 下单时检查

```
□ 是限价单吗？（不是市价追）
□ 止损条件单挂了吗？（成交后立即挂，不是"等会儿"）
□ 这笔亏损在 1% 以内吗？（不是这笔交易能不能亏得起，是这次亏了之后还有心情明天继续）
□ 同板块仓位会不会超标？（第零层再确认一次）
```
