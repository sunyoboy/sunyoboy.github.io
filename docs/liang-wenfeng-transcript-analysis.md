## 一、克制：比新闻版深得多

### 「拿得多的人会被拿得少的人打败」

```
OpenAI 想拿全球 AI GDP 的 5%
    → 被愿意只拿 1% 的人打败
        → 被愿意只拿 0.1% 的人打败
            → 愿景是拿得多，你就先输了
```

> **交易映射**：每次想"这笔要多赚点"——追涨、扛着不止损、临时开仓——就是 OpenAI。格伦姆的合理利润 = 梁文锋的六倍利润，十个月回本够了。

### 「你不克制，就不起」

克制不是为了道德——是**不做杂事才能做成核心。** 团队不加班，因为聚焦所以事少。

> **交易映射**：16 品种 → 9 个。18 条纪律 → 3 条铁律。减法不是放弃，是把精力还给核心。

### 「前面的都是芝麻，后面还有西瓜」

去年 C 端日活暴涨没抢。今年 ToB 收入涨了也没重点做。AGI 才是西瓜。

> **交易映射**：科创 +10.73% 是芝麻。有色 +22% 是芝麻。港股通互联网 ETF 也是芝麻。**四飞轮才是西瓜。**

## 二、AGI 路线图

```
语言模型 → CoT(已解决) → Agent(今年重点)
    → 持续学习(下一步瓶颈·全世界都还没找到方法)
    → 智能奇点(AI 加速 AI 研究·非线性渐变)
    → 具身智能(终极)
```

### 不做的事

不做 3D 生成、视频生成、世界模型——不是商业上不好，是**跟智能主线无关。**

### 奇点不是突变

「它是一个渐进过程。但可以用 AI 来加速 AI 研究——所以是非线性的。」

## 三、组织管理

### 「正式不要超过一半」

```
50% 时间：从上到下 · 正事 · 发版 · 协作
50% 时间：从下到上 · 自由探索 · 想做什么做什么
```

不加班的两个原因：① 研究需要松弛 ② 聚焦所以事少。

> **交易映射**：CLAUDE.md 能量曲线——高能量正事，低能量维护，22:00 后松弛。

### 「自己用得好才是真的好」

模型第一目标不是服务外部——是让 DeepSeek 自己开发下一版更快。

> **交易映射**：复盘不是写给别人看——是让明天的你决策更好。

## 四、算力与中美差距

### 华为 950：四张顶一张 GB300，落后两年

```
华为 950 × 4 ≈ 1 × GB300（性能平替）
时间差：GB300 两年前的货，华为 950 今年 Q3/Q4
总差距：四倍 + 两年
```

### 「落后一到两年，只用二十分之一算力」

目标是改写为：用几分之一算力，差距缩到 6 个月、3 个月。

> **交易映射**：资金量是机构的二十分之一——但纪律（条件单 + 偏离度 + 止损）可以缩差距。

### CUDA 护城河在瓦解

V3 训练用英伟达卡，但已不用英伟达生态——自研 TileLang 编译器。效率损失只有 1-2%。

> **交易映射**：不依赖外部工具，自己掌控交易生态。

## 五、商业化

### 「十个月收回设备成本 = 合理利润」

API 定价逻辑：服务器十个月回本，约六倍利润。团队内曾经定价太高 → 降到四分之一 → 群内欢呼。

> **交易映射**：有色 +22% 偏离高危 → 减仓。十个月回本够了。

### 「最坏情况卖 API 也能撑一个上市公司」

保底：几个亿美金 B 端收入 + C 端用户，离净利润不远。

> **交易映射**：你的保底——工资 + 国债票息。有保底才敢做减法。
