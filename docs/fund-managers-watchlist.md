## 二、⚠️ 先说真相：大多数「明星基金经理」近年表现很差

```
张坤 易方达蓝筹精选 → 从高点回撤超40%，规模从900亿缩至300亿
葛兰 中欧医疗健康   → 从高点回撤超50%，「医药女神」变成「韭菜收割机」
刘彦春 景顺新兴成长  → NAV 1.45，跑输沪深300已超3年
谢治宇 兴全合润     → 近3年收益为负，规模从400亿缩至150亿
```

**这不是他们能力不行——是中国主动基金的结构性问题：**

| 问题 | 说明 |
|------|------|
| 规模暴增 | 明星经理一旦出名→资金涌入→只能买大盘股→丧失了灵活性 |
| 考核周期短 | 季度排名压力→被迫追涨杀跌→风格漂移 |
| 赛道拥挤 | 消费经理被迫买新能源，医药经理被迫买AI |
| 管理费刚性 | 不管涨跌都收1.5%管理费，熊市里这点尤其致命 |

> **对你而言：ETF已经是最好的工具。** 你自己选行业、自己控仓位、自己定止损——这套系统比任何主动基金经理都靠谱。

## 三、主动基金经理 · 候选清单（8位）

> 筛选标准：① 从业≥7年穿越牛熊 ② 风格不漂移 ③ 规模不过度膨胀 ④ 有独特α来源

### 🥇 第一梯队：经过周期验证

#### 1. 朱少醒（富国）— 中国最长寿基金经理

| 维度 | 数据 |
|------|------|
| 代表基金 | 富国天惠成长A（161005） |
| 最新净值 | 3.2809（2026-07-03） |
| 从业年限 | **18年** |
| 投资风格 | GARP（合理价格成长） |
| 年化收益 | ~15% |

> **唯一从2005年管同一只基金到现在的人。** 穿越三轮牛熊。不赌赛道、不追热点。缺点是规模太大（~300亿），近年超额收益在收窄。

**对你的价值：** 🟡 不适合主动买入，但适合作为**业绩基准**——如果你的交易系统年化跑不赢富国天惠，说明系统有问题。

---

#### 2. 曹名长（中欧）— 深度价值，最被低估的经理

| 维度 | 数据 |
|------|------|
| 代表基金 | 中欧价值发现A（166005） |
| 最新净值 | 3.1828（2026-07-03） |
| 从业年限 | **17年** |
| 投资风格 | 深度价值（低PB+低PE） |

> 永远买便宜的、没人要的股票。在赛道股暴涨的2020-2021年被骂「过时了」，结果2022-2023赛道崩了他反而正收益。风格极度稳定、不漂移。

**对你的价值：** ✅ 风格与你互补——你做ETF轮动+趋势，他做深度价值逆向。可以作为**逆向信号**：他的深度价值股被市场抛弃时=价值风格可能触底。

---

### 🥈 第二梯队：风格鲜明但近年挨打

#### 3. 张坤（易方达）— 「酒神」正在渡劫

| 维度 | 数据 |
|------|------|
| 代表基金 | 易方达蓝筹精选（005827） |
| 最新净值 | 1.4990（2026-07-03） |
| 从业年限 | 11年 |
| 投资风格 | 高质量蓝筹（白酒+互联网） |

> 茅台+腾讯的铁杆拥护者。2021年封神，2022-2024跌落神坛。风格不漂移是好品质，但问题是他的「好公司」标准在宏观环境剧变下是否仍然成立？

**对你的价值：** ❌ 不适合买入。他的持仓你通过ETF基本都能覆盖（食品饮料ETF+恒生科技ETF），而且ETF不会收1.5%管理费。但可以作为**持仓参考**验证你的行业判断。

---

#### 4. 葛兰（中欧）— 「医药女神」的警示故事

| 维度 | 数据 |
|------|------|
| 代表基金 | 中欧医疗健康A（003095） |
| 最新净值 | 1.9854（2026-07-03） |
| 从业年限 | 9年 |
| 投资风格 | 医药赛道（高度集中） |

> 成也集中败也集中。医药涨时她是最锋利的矛，医药跌时她是最钝的盾。规模从800亿缩到300亿，但持仓几乎没变——说明她的确是「风格不漂移」，只是赛道本身崩了。

**对你的价值：** ❌ 不适合。做医药ETF（512010）或港股创新药ETF（513120）更灵活——跌了可以止损，葛兰不会替你止损。如果葛兰的基金开始持续跑赢医药ETF=医药主动α回归的**信号**。

---

#### 5. 丘栋荣（中庚）— 小盘价值，独树一帜

| 维度 | 数据 |
|------|------|
| 代表基金 | 中庚价值领航（006551） |
| 最新净值 | 3.6161（2026-07-03） |
| 从业年限 | 9年 |
| 投资风格 | 小盘价值（低估值中小盘） |

> 在「核心资产」盛行的2020年，他买没人要的小盘股。2021-2023小盘崛起时封神。风格极度坚定，擅长在冷门股里挖金子。

**对你的价值：** 🟡 风格上和你持有的中证500/1000有关联。可以作为**小盘风格的参照系**。

> 「买股票就是买公司的一部分」——真正按这个理念执行的人。比曹名长更年轻、规模更小、更灵活。持仓以低估值传统行业为主。

#### 8. 綦鹏（招商）— 逆向投资

| 维度 | 数据 |

> 买别人恐慌时卖出的好公司。在2022年10月、2024年2月底部大幅加仓。逆向能力是主动基金经理最稀缺的品质。

## 四、对你最有用的结论

### 你不需要买主动基金，但可以利用他们

| 用途 | 推荐 | 怎么用 |
|------|------|------|
| **业绩基准** | 朱少醒 161005 | 你的交易系统年化跑不赢富国天惠=系统有问题 |
| **持仓参考** | 张坤 005827 | 他的重仓股=高质量蓝筹，验证食品饮料ETF+恒生科技ETF逻辑 |
| **逆向信号** | 曹名长 166005 | 他的深度价值股被市场抛弃时=价值风格可能触底 |
| **小盘风向标** | 丘栋荣 006551 | 他的持仓方向≈中证500/1000的风向标 |
| **医药α信号** | 葛兰 003095 | 如果她的基金持续跑赢医药ETF=医药主动α回归（大概率不会） |
| **周期择时** | 董辰 | 他在买什么周期股=周期板块可能有底部机会 |

### 与你的ETF系统的关系

```
你的交易系统（核心）          主动基金经理（辅助验证）
─────────────────────      ─────────────────────
自主选行业 ETF              参考张坤/葛兰的持仓方向
自主控仓位                  参考朱少醒的业绩基准
自主定止损                  参考曹名长的逆向信号
自主做T                     参考邱栋荣的小盘风格

          └──── 交集：风格判断验证 ────┘
```

> 📌 **一句话：你已经有了一套完整的ETF交易系统——这比任何一个主动基金经理都更可靠。** 基金经理不知道你的止损线、不知道你的仓位、不知道你的资金计划。你知道。上面8位经理最大的价值不是「买他们的基金」，而是「用他们的持仓和业绩来验证你自己的判断」。
