## 一、三个法宝是什么

1939 年 10 月，毛泽东在《〈共产党人〉发刊词》中总结中国革命 18 年经验，提出：

> **统一战线，武装斗争，党的建设，是中国共产党在中国革命中战胜敌人的三个法宝，三个主要的法宝。**

| 法宝 | 核心问题 | 一句话 |
|------|------|------|
| 统一战线 | 谁是朋友，谁是敌人？ | 团结一切可以团结的力量 |
| 武装斗争 | 用什么方式战斗？ | 枪杆子里出政权 |
| 党的建设 | 谁来领导？ | 打铁必须自身硬 |

## 二、逐一解读

### 法宝一：统一战线

**核心：分清敌友，团结大多数，孤立极少数。**

```
敌人（极少数）  ←  孤立、打击
中间派（大多数） ←  争取、团结
朋友（可靠力量） ←  依靠、巩固
```

- 不是消灭所有反对者，而是**把朋友搞得多多的，敌人搞得少少的**
- 统一战线不是无原则的团结，是**又联合又斗争**
- 关键是找准**主要矛盾**——当前阶段的主要敌人是谁？

### 法宝二：武装斗争

**核心：没有实力，一切都是空谈。**

- 和平不是求来的，是打出来的
- 枪杆子里出政权——**最终要靠实力说话**
- 武装斗争不是蛮干，是有战略战术的：敌进我退，敌驻我扰，敌疲我打，敌退我追

### 法宝三：党的建设

**核心：打铁必须自身硬。自己不硬，统一不了战线，打不了胜仗。**

- 思想建设：统一认识，明确方向
- 组织建设：建立能打仗的队伍
- 作风建设：纪律严明，说到做到
- 自我革命：不断纠正错误，迭代升级

## 三、三个法宝的内在关系

```
        党的建设（内核）
           ╱ ╲
          ╱   ╲
    统一战线    武装斗争
    （外部资源）  （执行手段）
```

- **党的建设**是根本——自己不硬，一切免谈
- **统一战线**解决"和谁一起干"——单打独斗走不远
- **武装斗争**解决"怎么干"——有策略、有实力、有结果

> 三个法宝是一个**有机整体**，缺一不可。

## 四、对个人成长的启示

### 你的三个法宝

| 法宝 | 原始含义 | 个人成长映射 | 你的工具 |
|------|------|------|------|
| 统一战线 | 团结朋友 | **整合资源**——工具、人脉、信息、平台 | AI 工具链、开源社区、券商/银行、家人支持 |
| 武装斗争 | 武装战斗 | **执行能力**——策略、技能、纪律、执行力 | 交易系统、做T、PDCA、日复盘 |
| 党的建设 | 自我建设 | **内核修炼**——认知、品格、习惯、反思 | 信条、方法论、矛盾论/实践论、OKR、周报 |

### 统一战线：你有哪些"朋友"？

```
你的资源清单：
├── 🤖 AI 工具：Claude/GPT/DeepSeek → 分析、写作、代码
├── 📊 数据平台：东方财富/同花顺/财联社 → 信息优势
├── 📚 知识库：VitePress 站 → 沉淀复用
├── 🏦 机构：华泰/广发/东北 → 交易执行
├── 👨‍⚕️ 医疗：何巧花/省人民医院 → 孕期保障
└── 👨‍👩‍👦 家人 → 精神支柱
```

### 武装斗争：你的"武器"是什么？

```
你的武器库：
├── 💰 交易系统：做T / ETF / 仓位管理
├── 📋 操作系统：PDCA × 知行合一
├── 📖 方法论：矛盾论分析 + 实践论验证
├── 🔧 技能：Python / Git / VitePress / AI 工具链
└── ⚔️ 纪律：红线 / 止损 / 不加杠杆
```

### 党的建设：你的"内核"是什么？

```
你的内核：
├── 🔥 信条：不抛弃不放弃 · 三个不相信 · 岿然不动
├── 🔄 方法论：PDCA × 知行合一
├── 🧠 认知：矛盾论 / 实践论 / 个人操作系统
├── 📋 复盘：日复盘 → 周报 → 月回顾 → 年度 OKR
└── 💪 习惯：每日复盘不缺勤 · FOMC 前控仓 · 不进则退
```
