## 一、止损规则 · 与自有纪律对照

### 外部教学

| 板块 | 软止损 | 硬止损 |
|------|:--:|:--:|
| 主板 | -8~10% | **-20%** |
| 双创（创业板/科创板） | -12~15% | **-20%** |

### 与自有纪律对照

| 维度 | 外部教学 | 自有纪律 | 采纳 |
|------|------|------|:--:|
| 主板个股 | -8~10%软·-20%硬 | -8% | 🟡 接近·自有更严 |
| 双创个股 | -12~15%软·-20%硬 | 不碰双创个股 | ✅ 一致 |
| ETF | 未涉及 | -3~5% | ✅ 自有更保守 |
| 时间止损 | 短线3天·波段1-2周 | 个股5-10天 | ✅ 接近 |

> **采纳：-20%硬止损红线。** 不管主板还是双创，-20%必须无条件下车。这条比自有纪律（-8%）更激进，但补上了「极端情况下的最后一道防线」。

### 新增：硬止损红线

```
所有个股持仓：-20% = 无条件清仓
不适用 ETF（ETF -5% 止损优先）
```

### 金句

> 「谈个恋爱都把你绿了，你还要继续加筹码吗？」—— 亏损加仓 = 致命错误

## 二、止盈规则 · 三个可操作的信号

### 1. 偏离度止盈

```
每日计算5日均线偏离度
偏离度过高 → 主动止盈

→ 与自有 MA5 偏离度脚本完全对应
→ 我们已有: +5%高危·+3%偏热 → 挂减仓单
```

### 2. 三破五信号

```
3日均线跌破5日均线 = 高准确率逃顶信号
→ 与我们的 MA5/MA20 金叉死叉逻辑一致
→ 三破五是更短周期的版本
```

### 3. 三连超级大阳线

```
连续三根大阳线 → 无需等信号·主动减仓
→ 对应: 煤炭 +4.22% 次日 → 我们在高点清仓 ✅
→ 已验证有效
```

### 新增到止盈规则

| 信号 | 行动 |
|------|------|
| 三连大阳线 | 第3根阳线收盘前减仓1/3 |
| 三破五 | 当天清仓 |
| 偏离度>+5% | 挂减仓条件单 |

## 三、资金管理 · 可直接套用

### 分仓法

```
总资金 ÷ 4~10份
每份 = 单只标的首次开仓上限
首笔只用1份

→ 对应自有纪律: 单只行业ETF ≤ 10%·宽基 ≤ 15%·个股 ≤ 5%
→ 两份规则接近·可合并
```

### 加仓规则

```
首笔浮亏 → 不加仓（严禁摊薄成本）
首笔盈利 → 可追加1份（独立核算·不合并成本）

→ 与自有「永不在亏损时加仓」+「盈利后跟踪止损上移」一致 ✅
```

### 开仓限制

```
账户有浮亏持仓 → 优先处理亏损
全部解亏 → 方可新开仓

→ 比自有纪律更严
→ 我们允许同时持有多只（上限12只）·不要求全部解亏
→ 可部分采纳: 单日有止损触发 → 当天不开新仓
```

## 四、量价分析 · 补充到技术面

| 形态 | 含义 | 行动 |
|------|------|------|
| 高位放量阴线 | 主力出货 | 减仓避险 |
| 缩量回调 | 筹码稳定 | 支撑更强·反弹概率高 |
| 缩量回踩50/60日线 | 中期支撑 | 可观察·但ETF不一定适用 |

> 这点我们之前缺少。加入 `ma5-deviation.py` 的量能列已经在用了（量能>1.5放量·<0.6缩量），可以更系统地应用。

## 五、金句入知识库

```
1. 「亏损加仓 = 被绿了还加筹码」→ 对应绝对红线第3条
2. 「杀伐果断」→ 对应止损执行「触发就走·不犹豫」
3. 「慢就是快」→ 对应资金管理·长期践行
```
