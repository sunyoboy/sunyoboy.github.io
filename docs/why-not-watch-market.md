## 盯盘的真正代价

不是"浪费时间"那种轻飘飘的说法——**盯盘直接导致亏钱**。

### 代价一：高潮点做出最差决策

```
你盘中的感官体验：              实际的客观事实：

10:00 有色突然拉涨 +2%         今天成交额情绪 1.34（偏热）
      "来了来了！"              你正在情绪高潮点看盘
                                 而高潮点 = 最佳卖点 + 最差买点
10:30 继续涨 +4%
      "再不买就来不及了！"       踏空焦虑精准攻击盯盘者
      → 点击买入
                                  
次日：有色 -8.36%                昨天买的人全部被埋
      → 账户 -3,000 元
```

**已验证的证据**：

```
6/22 盘中盯盘的人看到了：非银 +5.86%、有色 +3.93%、成交量暴增
       盯盘驱动的感觉：  "大行情来了！"
       当日做的最差决策： 追涨有色
       
       次日结果：         有色 -8.36%
       
6/24 盘中盯盘的人看到了：科创50 上午 +2.48%
       盯盘驱动的感觉：   "AI 回来了！要不要追？"
       
       纪律框架回答：     不要。振幅 9-10% 偏高，PCE 周四落地
       次日结果：         不用验证——框架已经说了答案
```

> 🔑 盘中所有让你"感觉必须立刻行动"的瞬间，回头看都是最不该行动的时刻。因为最强烈的情绪 = 最多人同时情绪化 = 买在最高点。

### 代价二：盘中噪声 vs 收盘信号

同一只股票，盘中看到的和收盘后看到的，是完全不同的东西：

```
盘中（盯盘者视角）：              收盘后（框架视角）：

"跌了 2%！出什么事了？"         今日 -0.2%，振幅正常
"又拉回来了！机会来了！"         成交额情绪 1.10，正常区间
"冲高回落了，完蛋"               MA5 金叉▲多头，趋势未变

同一天，盯盘者经历了三次情绪波动，
收盘后发现——什么都没变。
```

**数据佐证**：

上交所 5,300 万账户研究结论：**散户择时和选股均为负贡献。** 不是因为散户不够聪明——是因为散户**在盯盘时做决策**，而盯盘时人处于最容易被情绪劫持的状态。该数据来自施东辉、张晓燕等人的学术论文及复旦-上交所联合发布的《个人投资者理性指数研究报告》。

### 代价三：吃掉了最重要的资源

CLAUDE.md 里写着的能量分配正在被盯盘消耗：

```
高能量（7:00-9:00）→ 做决策          ← 盯盘没启动，这一块是安全的
中能量（9:30-15:00）→ 上班            ← 盯盘伪装成"重要的事"抢走这个时间
低能量（15:00-16:00）→ 脚本+复盘       ← 盯盘耗尽后，复盘质量下降
低能量（20:30-21:30）→ 第二增长曲线    ← 盯盘导致的焦虑拖延了这个时段
```

盯盘把一个中能量时段**伪装成高能量时段**——你觉得自己在做重要的事（盯盘 = 盯钱），但实际上你在消耗明天做真正决策需要的注意力。

```
盯盘 4 小时的消耗：
  → 无法深度工作（第二增长曲线停滞）
  → 盘后复盘能量不足（复盘质量下降）
  → 晚间焦虑（因为盘中看到太多波动）
  → 第二天开盘前已经累了
```

**而结果呢？** 上交所数据说散户年化 -20.53%。盯盘的时间成本和机会成本全亏掉了，还倒赔钱。

## 框架替代盯盘

你的体系已经给出了替代方案——不是"不关注市场"，是用**收盘后的数据驱动决策替代盘中的情绪驱动决策**：

| 维度 | 盯盘 | 框架 |
|------|------|------|
| 决策时间 | 盘中实时 | 收盘后 15:00-16:00 |
| 决策依据 | 分时图、感觉 | 成交额情绪、MA5 偏离度、波动率 |
| 持有多久 | 几分钟到几小时 | 几天到几周 |
| 操作频率 | 高频 | 低频（每周 ≤ 2 次） |
| 情绪成本 | 极高 | 极低 |
| 与第二增长曲线的关系 | 冲突 | 互补 |
| 历史验证 | -20.53% 年化 | — |

### 例：中证500 要不要买

**盯盘模式（错误）**：

```
盘中看到涨了 1.5%
  → "趋势来了"
  → 市价买入 500 份
  → 没有止损 → 没有仓位计算 → 没有事件检查
  → 周四 PCE 数据暴雷 → 次日低开 2% → 割肉
```

**框架模式（正确）**：

```
收盘后跑脚本
  → 第零层仓位合规：✅
  → 第一层市场状态：成交额情绪 1.10 ✅ / 振幅 9-10% 🟡偏高
  → 第二层技术诊断：偏离低吸区 +2.5% ❌

结论：不买。

→ 设限价单 8.88，止损 8.65，仓位 1,200 份（按公式计算）
→ 等周四 PCE 落地后再评估
→ 如果触发，有止损有止盈有计划
→ 如果没触发，市场没到你的价——不遗憾
```

> 两者唯一的区别是决策时机：一个在盘中情绪高点，一个在收盘后数据齐全时。

---

## 一句话

> 盯盘最大的问题不是浪费时间——是**它让你在情绪最高潮、信息最不完整的时候做出不可撤回的决策。** 你的框架替代盯盘，不是让你对市场视而不见，是让你在所有关键信息到齐之后、在自己能量最稳定的时段、按照提前写好的规则做出决策。
