## 性价比排名（SWE-bench ÷ 有效成本）

### 计算公式

```
性价比指数 = SWE-bench Verified(%) ÷ 有效输出成本($)

有效输出成本 = 输出单价 × token效率系数
```

> 性价比指数 = 每一美元买到多少 SWE-bench 百分点。数字越大越划算。

### 关键发现

**1. DeepSeek 碾压级领先**

DeepSeek V4-Flash 性价比指数 184，是第三名 GLM-5.2（15）的 **12 倍**，是 Opus 4.8（3.5）的 **53 倍**。如果预算有限，这个选择没有悬念。

**2. GPT-5.5 的隐藏性价比**

输出单价 $30 全榜最贵，但 token 效率系数 0.28（同任务只消耗其他模型 1/4 的 token）使其有效成本降至 $8.40——比豆包 2.1 Pro（$6.40）仅高 31%，但 SWE-bench 高 8%。**是排行榜上最反直觉的结果：看似最贵的模型，实际性价比不差。**

**3. 国产最佳性价比：GLM-5.2**

GLM-5.2 有效成本 $5.20，低于豆包 $6.40 和 Qwen $9.18。而且 **MIT 开源可自部署**——有 GPU 的话 API 费用归零，性价比趋向无穷大。

**4. Qwen3.7 Max 的性价比陷阱**

输出 ¥18/M（$2.48）看起来便宜，但 token 效率系数 3.7 倍将有效成本推至 $9.18——实际比 GPT-5.5（$8.40）和豆包（$6.40）都贵。**单价低 ≠ 便宜。**

**5. 性价比 ≠ 最好用**

性价比排名回答的是"每一美元买到多少 SWE-bench 分数"，不是"哪个模型写代码最漂亮"：

| 性能维度 | 模型 | 性价比排名 |
|------|------|:---:|
| 代码审美 / 架构设计 | Claude Opus 4.8 | 第 9 |
| Agent 自动化 / 工具调用 | GPT-5.5 | 第 5 |
| 多文件重构 / 长程推理 | Claude Opus 4.7/4.8 | 第 9/10 |
| 中文场景 / 合规 | 豆包 2.1 Pro / GLM-5.2 | 第 3/4 |
| **每一美元买到的正确率** | **DeepSeek V4-Flash** | **第 1** |

> 🎯 选模型前先想清楚要什么：要性价比就 DeepSeek，要代码品质就 Claude，要平衡就 GPT-5.5（隐藏价位选手）。

## 场景速查矩阵

> 一句话结论：**DeepSeek V4-Flash 是性价比之王。** 输入 0.14、输出 0.28，比 GPT-5.5 便宜 **107 倍**，每一美元买到 56 倍的编码能力。

### 按场景推荐

| 场景 | 推荐模型 | 输入/输出 $ | 理由 |
|------|------|:--:|------|
| **极致便宜/大批量脚本** | DeepSeek V4-Flash | 0.14 / 0.28 | 日均 50M token 仅 $8.4，GPT-5.5 的 1/50 |
| **日常编码主力** | DeepSeek V4-Pro | 0.44 / 0.87 | 80.6% 编码，输出价 = Claude Opus 的 1/29 |
| **开源部署/离线** | GLM-5.2（MIT） | 免费自部署 | 80% SWE-bench，不受 API 限频 |
| **国产合规/企业** | 豆包 2.1 Pro | ¥6 / ¥30 | 82% SWE-bench，阿里云生态 |
| **高难度架构/一次搞定** | Claude Opus 4.8 | 5 / 25 | 88.6% 编码，贵但少返工 |
| **Agent 密集/工具调用** | GPT-5.5 | 5 / 30 | token 效率最高（28% 基准），隐藏性价比 |
| **免费入门** | Gemini CLI / TRAE | 免费 | Gemini 3.1 Pro 或豆包 2.1 驱动 |
| **最低订阅门槛** | GitHub Copilot Pro | $10/月 | 内置多模型切换 |

### 隐藏成本提醒

| 模型 | 陷阱 | 实际影响 |
|------|------|:--:|
| Qwen3.7 Max | 输出 token 量是 GPT-5.5 的 **3.7 倍** | 有效成本翻倍，单价低≠便宜 |
| Claude（新版分词器） | 同文本 token 数多 **35%** | 涨价藏在分词里 |
| GPT-5.5 | 输出极简（28% 基准），可能牺牲设计质量 | 便宜但代码可能不够漂亮 |

### 个人开发者推荐组合

```
日常主力    → DeepSeek V4-Pro（月费 ~$50，约 ¥360）
批量脚本    → DeepSeek V4-Flash（月费 ~$8，约 ¥58）
高难度架构  → Claude Opus 4.8（偶尔用，按次付费）
──────────────────────────────────────────
月费合计    → ~$80-120（约 ¥580-870）
```

> 这个组合覆盖 95% 的编码场景。比单独订阅 Claude Code Max（$200/月）便宜一半，比 GPT-5.5 重度使用（$1,200/月）便宜 10 倍。
