# 🔻 Capitulation（投降）信号 · 底部识别框架

> 如何判断「最恐慌的阶段是不是过去了」——不靠感觉，靠三个可验证的信号。
>
> 来源：2026-07-14 → 07-21 五连跌 -20% 实战追踪总结。

---

## 一、什么是 Capitulation

**Capitulation = 投降。** 金融术语，指市场最后一波恐慌性抛售结束、连最坚定的多头都扛不住割肉了的时刻。

```
正常下跌 → 有人扛、有人抄底、有人换仓
     ↓
恐慌下跌 → 什么都不挑了，只想换成现金
     ↓
Capitulation → 该割的都割完了 → 卖压枯竭 → 底部
```

核心特征：**好股票坏股票一起跌，不分青红皂白。** 这时候不是"市场在区分优劣"，而是"市场只想活命"。

## 二、三个可验证的信号

> 以下三个信号在 2026-07-21 全部确认。这不是理论推演，是实战检验过的。

### 信号 ①：最后一个防御板块补跌

**为什么重要**：如果还有板块在涨，说明资金还在挑挑拣拣——不是真正的恐慌出清。

**怎么判断**：全市场按板块逐个排查——哪个板块还在涨？如果只剩一两个在扛，等它们也倒了 → 信号确认。

```
追踪案例（2026年7月）：
7/16 → 上证50 补跌 ✅
7/17 → 医药补跌 ✅
7/17 → 煤炭翻绿 ✅
7/20 → 银行 +2.41%·最后一个堡垒
7/21 → 银行 -2.23% 🚨 ← 倒了！
```

> **当最后一个防御板块也补跌时，说明连"最安全"的钱都在出逃——无差别出清的最后一环。**

### 信号 ②：黄金止跌

**为什么重要**：黄金下跌通常不是因为黄金不好，而是因为流动性的——投资者被迫卖一切换现金（margin call、爆仓、赎回）。黄金止跌 = 流动性挤兑结束。

**怎么判断**：黄金从连续下跌转为连续翻红。不需要暴涨——止跌本身就是信号。

```
追踪案例：
7/17 → 黄金 -0.62%（两连跌·流动性挤兑中）
7/20 → 黄金 +0.14%（微弱翻红·挤兑减缓）
7/21 → 黄金 +1.16%（加强确认 🎉）
```

### 信号 ③：放量V反 + 下午不跳水

**为什么重要**：一日游和真反弹的核心区别在于：下午是否跳水。一日游是上午涨、下午全吐回去——说明没人真的在接。真反弹是全天维持甚至尾盘继续涨——说明有资金在真金白银地买。

**怎么判断**：
- 反弹日成交量 > 前 5 日均量
- 领涨的是超跌品种（不是防御品种）
- 下午不跳水 → 维持甚至扩大涨幅

```
追踪案例：
7/20 → 上午反弹·下午翻绿 ❌ 一日游
7/21 → 科创+10.73% 全天维持·尾盘未跳水 ✅ 真反弹
```

## 三、三个信号的关系

```
信号① 银行补跌  → 卖压释放（该割的都割了）
信号② 黄金止跌  → 流动性恢复（没人再被强平了）
信号③ 放量V反   → 抄底进场（有人真金白银在接了）
        ↓
  三者齐聚 = 短期底部大概率确认
```

> ⚠️ **capitulation 信号确认 ≠ 明天开始天天涨。** 它是说最恐慌的阶段大概率过去了，不是说不会再有回调。底部是一个区间，不是一个点。

## 四、回踩 MA5 · 底部不追涨的入场策略

### 什么是回踩 MA5

```
价格
 ↑
 |     🚀 大涨              ← 冲太猛，离 MA5 太远
 |    /  \
 |   /    \   ⚠️ 回踩        ← 回调下来"摸一下"MA5
 |  /      \ /
 | /        · ← 碰到 MA5
 |/________/_\______________ → MA5 均线
 |         ↑
 |     在 MA5 附近企稳 = 回踩确认 = 健康的回调
```

### 为什么要在回踩时买而不是追涨

| 追涨 | 回踩 MA5 买 |
|------|------|
| 大涨日冲进去 | 等回调到 MA5 附近 |
| 第二天-3% 就浮亏 | 买入价接近均线支撑 |
| 心态崩 → 割肉 | 止损好设（跌破 MA5 就走） |
| 涨了也拿不住 | 跌了也亏不多 |

> **核心逻辑：capitulation 告诉你底部区间到了，回踩 MA5 告诉你具体买点在哪。前者判断方向，后者控制风险。**

### 操作模板

```
1. capitulation 三信号确认 → 短期底部区间出现
2. 等目标品种回踩 MA5 → 挂条件单
3. 到了买入 → 止损设在 MA5 下方
4. 不到不追 → 等下一个回踩
5. 如果跌破 MA5 → 不入场·说明比预期更弱
```

## 六、速查卡

| 信号 | 确认标准 | 未确认时 |
|------|------|------|
| ① 银行/防御补跌 | 最后一个抗跌板块单日 -1.5%+ | 继续等·不急着抄底 |
| ② 黄金止跌 | 黄金连续 ≥2 日翻红 | 流动性挤兑未结束·警惕 margin call |
| ③ 放量 V 反 | 超跌品种领涨 + 下午不跳水 | 一日游·不能追 |

**一句话**：信号不全 → 现金为王。三信号齐 → 有序回补。不在过热日追，等回踩 MA5 买。
