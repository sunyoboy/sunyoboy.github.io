> 梁文锋：「克制是一种战略。」不舍弃多元，就无法专注主线。
>
> **规则越多 = 每条执行越差。持仓越多 = 注意力越分散。**

---

## 一、梁文锋启示

DeepSeek「不做 3D 生成、不做视频生成、不做世界模型」——商业上都是好生意，但与 AGI 主线无关，一概不做。

> 你的交易同理：**所有与"赚钱"无关的操作，都是噪音。** 条件单、止损线、回踩买入——纪律上的加法 = 决策上的减法。挂完单就不用想了。

## 三、纪律做减法

CLAUDE.md 十八条规则 → 只留三条铁律。

| # | 铁律 | 一句话 |
|:--:|------|------|
| 1 | **收盘后决策** | 盘中不操作·中午不打开 APP |
| 2 | **买入必挂止损** | 入场价 + 止损价 + 仓位 · 缺一不买 |
| 3 | **偏离度到点就做** | 脚本说减仓 → 立刻挂条件单·不等明天 |

> 只盯着这三条，做到比有十八条但只执行三条强。

## 五、三条铁律的落地检查

每日收盘后：

```
□ 今天盘中有操作吗？→ 有 = 违规·记入复盘
□ 今天中午打开过APP吗？→ 有 = 违规
□ 今天新买的东西挂止损了吗？→ 没挂 = 明天开盘前必须挂
□ 脚本有没有触发偏离度信号？→ 有 = 立刻挂条件单
```
