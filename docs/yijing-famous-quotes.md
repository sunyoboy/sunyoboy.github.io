## 六、投资五维映射

| 名句 | 交易等价 |
|------|------|
| 时止则止，时行则行，动静不失其时 | 死叉就止·金叉就行——时机 = 均线信号·不是感觉 |
| 君子藏器于身，待时而动 | 备好弹药（东北账户 41,000）·等三信号齐 |
| 几者，动之微，吉之先见者也 | MA5 偏离从 -3% 开始扩大 = 吉凶变化的最早信号 |
| 变通者，趋时者也 | 不预测趋势·跟随趋势。趋势变了就变 |
| 君子知微知彰，知柔知刚 | 知道什么时候加仓（彰）·什么时候止损（微） |
| 知存而不知亡，知得而不知丧 | "拿着会涨"而不知道"该卖了" = 亏损的根源 |
| 君子以思患而豫防之 | 买入前先想好止损价 = 居安思危 |
| 君子以同而异 | 金叉品种都涨·但天山+4.25%≠讯飞-0.36%——看清同中的异 |
| 安而不忘危，存而不忘亡 | 连涨之后不是加仓·是先收紧止损 |
| 德薄而位尊，知小而谋大，力少而任重，鲜不及矣 | 仓位超过承受力·认知配不上野心 = 迟早爆仓 |
