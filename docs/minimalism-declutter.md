# 断舍离 · 极简生活 · 第欧根尼借鉴

> 自由 = 减法。断舍离的底层逻辑与交易纪律同构。

---

## 一、断舍离 ≠ 扔东西

| 层级 | 行为 | 难点 |
|------|------|:--:|
| 断 | 不买不需要的（入口控制） | 对抗消费冲动·延迟满足 |
| 舍 | 扔掉不再需要的（存量清理） | 对抗"万一用得上"·面对沉没成本 |
| 离 | 放下执念（心态转变） | 对抗攀比/身份焦虑·重新定义自我价值 |

---

## 二、第欧根尼借鉴——借鉴原则，不借鉴程度

```
拥有一件东西 → 需要存放→维护→怕失去→焦虑→买更多→更焦虑
打破循环：不是"拥有更多"，是"需要更少"。
```

| 可借鉴 | 不需要借鉴 |
|------|------|
| 删掉半年没开的 App | 睡木桶 |
| 不买"万一用得上"的东西 | 当众排泄 |
| 不需要用消费证明自己 | 什么都不拥有 |
| 定期审视必需品清单 | — |

## 四、执行方法

**24 小时规则**：任何超过 200 元的非必需品 → 购物车 → 24h 后决定。80% 会忘了它。

**90 天规则**：90 天内没用过 → 默认非必需。除非 10 秒内能说出下次何时用。

**所有权测试**："如果丢了，我会花全价再买吗？" 不会 → 不是必需品。

**第欧根尼练习（每季度）**：空桌·无手机·1h。①哪些焦虑和"拥有"有关？②只能留 20 样东西，留什么？③过去三月买了什么一次没用过？→ 写下来·扔掉/删掉第三题的答案。

## 五、交易映射

| 断舍离 | 交易等价 |
|------|------|
| 断 = 入口控制 | 买入三问·缺一不买 |
| 舍 = 存量清理 | 死叉就卖·不等 |
| 离 = 放下执念 | 沉没成本不是成本 |
| 24h 规则 | 不做计划外交易 |
| 所有权测试 | 如果今天空仓·还会买吗？ |
