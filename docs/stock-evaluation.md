# 买股票就是买公司？— 从第一性原理推导交易评估框架

> 不断追问，直到问出实际要看的东西

---

## 第一层：买股票 = 买公司？

> **是的。** 股票是公司所有权的凭证。买 1 股 = 拥有公司的一小份。

那问题变成：**什么样的公司值得买？**

---

## 第二层：好公司看什么？

巴菲特的答案：三个要素。

```
好公司 = 好生意 + 好管理 + 好价格
```

| 要素 | 问题 |
|------|------|
| 好生意 | 这家公司靠什么赚钱？能赚多久？ |
| 好管理 | 谁在管？管得怎么样？靠不靠谱？ |
| 好价格 | 现在买贵不贵？ |

继续问 ↓

---

## 第三层：好生意怎么判断？

### 3.1 商业模式（怎么赚钱）

| 问题 | 好的回答 | 差的回答 |
|------|------|------|
| 一句话说清怎么赚钱吗？ | "卖白酒给消费者" | "我们是全链路数字化生态平台" |
| 赚的是谁的钱？ | 消费者刚需、企业必须支出 | 政府补贴、一次性项目 |
| 钱能持续赚吗？ | 复购率高、合同长期 | 一锤子买卖 |

### 3.2 护城河（别人能不能抢）

| 问题 | 怎么验证 |
|------|------|
| 给你 100 亿能打倒它吗？ | 打不倒 = 有护城河 |
| 客户会因为便宜 10% 换掉它吗？ | 不会换 = 有转换成本 |
| 新玩家进来容易吗？ | 不容易 = 有壁垒 |

护城河类型：

| 类型 | 例子 |
|------|------|
| 品牌 | 茅台、可口可乐 — 消费者指名购买 |
| 成本 | 中国神华 — 别人成本比你高 |
| 网络效应 | 微信 — 用的人越多越有价值 |
| 转换成本 | 用友/金蝶 — 换系统代价极高 |
| 牌照/特许 | 银行、电信 — 不是谁都能干 |

### 3.3 行业空间（天花板有多高）

| 问题 | 意味着 |
|------|------|
| 这个行业还在增长吗？ | 增长期 = 蛋糕在变大 |
| 渗透率到多少了？ | <30% = 空间大；>70% = 接近饱和 |
| 10 年后这个行业还存在吗？ | 存在 = 值得长期持有 |

## 第四层：好管理怎么判断？

### 4.1 管理层品格

| 问题 | 怎么查 |
|------|------|
| 说话算数吗？ | 看过去 3 年承诺 vs 实际完成 |
| 有没有损害小股东的历史？ | 搜"公司名+违规/处罚/关联交易" |
| 管理层自己买不买自家股票？ | 查高管增减持记录 — 自己都不买说明不看好 |
| 薪酬合理吗？ | 年报查高管薪酬 vs 公司利润 |

### 4.2 管理层能力

| 问题 | 怎么看 |
|------|------|
| ROE 连续 5 年 > 15% 吗？ | 能 = 管理层会赚钱 |
| 毛利率在行业什么位置？ | 前 1/3 = 有竞争力 |
| 负债率健康吗？ | 资产负债率 < 60%（非金融） |
| 自由现金流为正吗？ | 正 = 赚的是真金白银，不是应收账款 |

---

## 第五层：好价格怎么判断？

### 5.1 估值指标

| 指标 | 含义 | 简单标准 |
|------|------|------|
| **PE（市盈率）** | 花多少钱买 1 块钱利润 | < 行业平均 = 相对便宜 |
| **PB（市净率）** | 花多少钱买 1 块钱净资产 | < 1 = 破净，可能低估 |
| **股息率** | 每年分红占股价比例 | > 3% = 有吸引力 |
| **PEG** | PE ÷ 利润增速 | < 1 = 增长没被充分定价 |

### 5.2 买入时机

| 问题 | 判断 |
|------|------|
| 现在是贵还是便宜？ | 对比历史 PE 分位数 — <30% 分位 = 低估 |
| 有没有安全边际？ | 留 20-30% 的余量 — 不在"合理价"买，在"打折价"买 |
| 大盘环境如何？ | 熊市末期买 > 牛市末期买 |

## 第六层：实际交易还要评估什么？

好公司 ≠ 好股票。**还要看时机、仓位、风控。**

### 6.1 技术面（什么时候买）

| 问题 | 工具 |
|------|------|
| 趋势向上还是向下？ | 均线排列（多头/空头） |
| 有没有支撑位？ | 前低、整数关口、均线 |
| 量价配合吗？ | 放量突破 = 有效；缩量突破 = 假突破 |

### 6.2 仓位（买多少）

| 问题 | 规则 |
|------|------|
| 单只占总资金多少？ | 个股 ≤ 10%，ETF ≤ 15% |
| 一次买满吗？ | 不。分批 50% → 25% → 25% |
| 总仓位多少？ | 看行情：高位 ≤ 50%，低位可 70% |

### 6.3 风控（怎么保命）

| 问题 | 规则 |
|------|------|
| 跌多少止损？ | 个股 -5~8%，ETF -3% |
| 涨多少止盈？ | 分批卖或跟踪止损 |
| 买入理由消失了怎么办？ | 不管盈亏，立刻卖 |

## 完整推理链

```
买股票
 └── = 买公司
      └── 好公司 = 好生意 + 好管理 + 好价格
           ├── 好生意 → 商业模式清晰？护城河深？行业空间大？
           ├── 好管理 → 品格靠谱？ROE高？现金流真实？
           └── 好价格 → PE低估？股息率高？有安全边际？
                └── 实际交易还要看
                     ├── 时机 → 趋势/支撑/量价
                     ├── 仓位 → 分批/上限/总仓位
                     └── 风控 → 止损/止盈/理由消失即卖
```

---

## 一页评估清单

每次买入前过一遍：

```
□ 一句话说清它怎么赚钱
□ 护城河是什么（品牌/成本/网络/牌照）
□ ROE > 15%？连续几年？
□ 自由现金流为正？
□ 管理层有没有黑历史？
□ PE 在历史什么分位？
□ 有安全边际吗（打折价）？
□ 大盘趋势向上还是向下？
□ 仓位不超过 10%？
□ 止损设了吗？
```

> **10 个问题答不上 3 个以上 → 不买。用 ETF 替代。**
