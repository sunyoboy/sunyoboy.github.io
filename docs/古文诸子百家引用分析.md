### 3. 《剑来》核心哲学主线

《剑来》本质上是一部以修仙为外壳的**诸子百家思想对话录**，其核心哲学冲突包括：

1. **人性善恶之争**：书中核心事件「三四之争」即文圣（主张性恶，类荀子）与亚圣（主张性善，类孟子）之间的论战
2. **知行合一**：主角陈平安的修行之道，贯穿全文——「知道了道理，然后去做，才是自己的」
3. **破心中贼**：王阳明「破山中贼易，破心中贼难」是主角最大的内心困境
4. **恻隐之心与赤子之心**：孟子与道家的核心概念，反复用于塑造人物抉择
5. **克己复礼与慎独**：儒家修养功夫，主角每日自省的准则

### 4. 《凡人修仙传》引用特点

- 古文引用极少，仅作为**文化底色**点缀
- 核心主题为「弱肉强食」的修仙丛林法则（更近法家而非儒家）
- 儒家「浩然之气」被设定为**儒门功法体系**的一部分（世界观设定）
- 「窈窕淑女，君子好逑」等诗经句仅用于日常生活对话
- 「知己知彼」等孙子兵法的引用服务于战斗策略需要
