## 一、建仓纪律

### 建仓前必问清单

每次建仓前回答 5 个问题。**任何一个答不上来，就不买。**

```
1. 我为什么买它？（一句话说清逻辑）
2. 我打算拿多久？（一天/一周/一月/长期）
3. 跌多少我止损？（具体价格或百分比）
4. 涨多少我止盈？（具体价格或百分比）
5. 我用多少仓位？（占总资金的百分比）
```

### 时机纪律

| ✅ 可以建仓 | 🚫 不能建仓 |
|------|------|
| 重大事件**已落地** | 重大事件**前 1-2 天**（FOMC/CPI/非农） |
| **先清后建**，问题持仓清完 | 问题持仓还没处理 |
| **尾盘 14:45-14:55** | **开盘 30 分钟内** |
| 指数在**支撑位**附近 | 连涨 3 天以上（短期超买） |
| 缩量**止跌**信号 | 全市场恐慌性暴跌中 |
| 你很冷静 | 你很激动（兴奋/恐惧/愤怒） |

### 仓位纪律

| 规则 | 上限 |
|------|:---:|
| 单只个股 ≤ 总资金 | **10%** |
| 单只 ETF ≤ 总资金 | **15%** |
| 单次买入 ≤ 目标仓位 | **50%** |
| 当日总买入 ≤ 总资金 | **20%** |
| 现金永远保留 | **≥ 30%** |

### 品种纪律

| 优先级 | 品种 |
|:---:|------|
| 🥇 | 宽基 ETF（沪深300/500） |
| 🥈 | 行业 ETF（科技/红利/基建） |
| 🥉 | 龙头个股（研究透彻 + 仓位严控） |
| ❌ | 垃圾股/ST/纯概念炒作 |

### 分批建仓法

```
目标仓位 2 万元：
  第 1 笔：1 万（50%），尾盘买入，观察 2-3 天
  第 2 笔：5000（25%），趋势确认后加
  第 3 笔：5000（25%），回调不破止损再加
```

> **永远不要一次性满仓。**

## 二、持仓纪律

| 场景 | 操作 |
|------|------|
| 买入理由**仍然成立** | 继续持有，不因短期波动卖出 |
| 买入理由**已消失** | 不管盈亏，**立刻卖出** |
| 持仓涨了但**不确定要不要卖** | 看止盈规则，而不是看感觉 |
| 持仓跌了但**不想止损** | 执行止损，不做"下一次会反弹"的赌注 |
| 持仓一直**横盘不动** | 给自己设一个观察期限（如 2 周），到期未动则减仓换标的 |
| **盘中频繁看持仓** | 关掉 APP。你已经设了止损条件单，不需要盯盘 |

> **持仓的核心：不是"拿着"，而是"有理由地拿着"。**

## 三、止损纪律

### 止损是必须的

```
没有止损 = 没有安全带的驾驶
止损不是认输，是保留战斗力
```

### 止损规则

| 品种 | 止损线 | 说明 |
|------|:---:|------|
| 宽基 ETF | **-3%** | 大盘 ETF 波动小，-3% 已异常 |
| 行业 ETF | **-5%** | 行业波动大一些 |
| 个股（短线） | **-5%** | 短线追求快进快出 |
| 个股（中长线） | **-8%** | 给更大的波动空间 |
| 做 T 单次 | **-0.8%** | 日内必须严格 |

### 止损执行

| 规则 | 说明 |
|------|------|
| **买入后立刻挂条件单** | 不靠意志力止损，靠系统自动执行 |
| **触发就走，不犹豫** | "再等等看"是亏损扩大的开始 |
| **不移动止损位** | 止损线一旦设定，只能上移（跟踪止损），不能下移 |
| **止损后不立刻回补** | 至少等 1 个交易日冷静 |
| **单日止损 2 次 → 当日停手** | 说明今天判断出错，不再操作 |

### 跟踪止损（盈利后保护利润）

```
买入价 10.00，止损设在 9.50（-5%）
涨到 11.00 → 上移止损到 10.45（保住 4.5% 利润）
涨到 12.00 → 上移止损到 11.40
回落到 11.40 → 自动卖出，锁定 +14% 利润
```

> **跟踪止损 = 让利润奔跑，同时锁住已有收益。**

## 四、止盈纪律

### 止盈比止损更难

```
止损难在"不愿认错"
止盈难在"还想更多"
```

### 止盈规则

| 策略 | 止盈方式 |
|------|------|
| **目标价止盈** | 买入时设目标价，到了就卖，不贪 |
| **分批止盈** | 涨 5% 卖 1/3，涨 10% 再卖 1/3，剩 1/3 用跟踪止损 |
| **技术面止盈** | 跌破 5 日线 / 出现顶背离 / 放量滞涨 → 减仓 |
| **时间止盈** | 持有超过预定期限且未达目标 → 卖出换标的 |
| **做 T 止盈** | 0.5%-1.0%，到了就走，不恋战 |

### 止盈心态

| ❌ 错误想法 | ✅ 正确想法 |
|------|------|
| "再拿一会儿赚更多" | 到了目标就执行，没人能卖在最高点 |
| "卖了万一继续涨怎么办" | 卖掉的那一刻利润就是你的，之后的涨跌跟你无关 |
| "这次不一样" | 每次都"不一样"，但纪律是一样的 |

## 五、加仓与减仓纪律

### 加仓纪律（加在对的地方）

| ✅ 可以加仓 | 🚫 不能加仓 |
|------|------|
| 趋势向上 + 回调到支撑位 | 趋势向下 + "便宜了再买点摊成本" |
| 首笔盈利 + 逻辑不变 | 首笔亏损 + 想通过加仓回本 |
| 加仓后总仓位仍在上限内 | 加仓后超过单只 10% / 总仓位上限 |

> **核心：只加盈利的仓，不加亏损的仓。越跌越买 = 越套越深。**

### 减仓纪律（何时减）

| 信号 | 操作 |
|------|------|
| 涨幅达到止盈目标 | 减 1/3 到 1/2 |
| 成交量异常放大但价格不涨（放量滞涨） | 减 1/2 |
| 重大利好兑现（利好出尽） | 全部卖出 |
| 重大事件前（FOMC/CPI） | 减至安全仓位 |
| 基本面发生负面变化 | 不管盈亏，坚决减仓 |

## 六、做T纪律

| 规则 | 说明 |
|------|------|
| 只做 A/B 级信号 | C 级弱信号放过，宁可错过 |
| 单日最多做 **2 次** | 做多错多 |
| 止盈 **0.5%-1.0%** | ETF 波动小，目标不能高 |
| 止损 **-0.8%** | 跌破坚决走，日内不补 |
| **收盘必须平仓** | 14:50 前所有做T仓位归零 |
| **连错两把停手** | 当天判断出错，不再操作 |
| 重大事件日**不做T** | FOMC/CPI/GDP 公布日观望 |
| 窄幅震荡（±0.3%）**不做** | 没有空间，硬做只亏手续费 |

> 详见 [宽基ETF做T策略](index-etf-trading)

## 七、心态纪律

### 交易前检查情绪

```
问自己：我现在冷静吗？
  冷静 → 可以交易
  兴奋 → 可能追高，暂停 30 分钟
  恐惧 → 可能恐慌卖出，暂停 30 分钟
  愤怒（刚亏了） → 今天不交易
```

### 不见兔子不撒鹰

> 猎人的逻辑，不是赌徒的逻辑。这是交易心态纪律最精炼的七个字。

```
猎人（框架交易）              赌徒（情绪交易）
─────────────                ─────────────
看见兔子 → 放鹰               没看见兔子 → 先放鹰再说
确定目标 → 出手               怕错过 → 出手
一击必中                      十次九空
```

**六个字拆解：**

| 字 | 对应 | 含义 |
|------|------|------|
| 兔 | 目标价 / 入场条件 | 不是感觉，是具体数字。中证500 的 8.70、中证1000 的 3.42 |
| 鹰 | 限价单 / 资金 | 提前挂好，不是盘中手动追 |
| 见 | 触发确认 | 价格到了、条件满足了——不是"快到了" |
| 撒 | 执行 | 条件触发→机械执行，不犹豫不修改 |

**三个问号自检：**

```
问自己：兔子在哪？
  → 中证500 目标 8.70，现在 8.94 → 兔子没出来 ❌

问自己：为什么想撒鹰？
  → 因为科创50 +4.41%、全市场在涨 → FOMO ❌

问自己：现在撒鹰能打中吗？
  → 追涨进去，止损空间不够 → 不能 ❌
```

**和框架的映射：**

| 框架层 | 不见兔子不撒鹰 |
|------|------|
| 第二层（技术诊断）| 兔子 = 入场价/止损价，没算出来就不动手 |
| 第七层（执行指令）| 鹰 = 限价单，挂上去就不改，没触发就等 |
| 心态纪律 | 见 = 条件确认，不是"我觉得要涨" |

> 🎯 6/26 等了六天才等到兔子。今天下午全市场都在涨，系统1 在尖叫"再不买就没了"——但猎人知道：**兔子没出来，撒鹰就是浪费箭。限价单挂在那儿，到了自然成交。不到，不动。**

### 财不入急门

> 越急越穷。不是鸡汤——是交易数学。

```
急 → 怕错过 → 追涨 → 买在日内高点 → 第二天低开 → 止损 → 更急
                                                    ↑____循环____|
```

**急的四种表现 & 数学后果：**

| 急的表现 | 数学后果 |
|------|------|
| 追涨（不等限价单触发）| 买到的比等到的贵，止损空间被压缩，盈亏比变差 |
| 频繁交易 | 摩擦成本（手续费+滑点）每次约 0.1-0.3%，十次就吃掉 1-3% |
| 上午跌下午就补仓 | 没给市场足够时间确认方向——不是在抄底，是在赌底 |
| 刚止损就买下一个 | 情绪驱动，不是框架驱动——下一个大概率也止损 |

**和「不见兔子不撒鹰」的分工：**

```
不见兔子不撒鹰 → 管住手（行为纪律）
财不入急门     → 管住心（情绪纪律）
两个加一起     → 框架完整执行
```

**三个场景自检：**

```
场景1：科创50 +4.41%，想追
  → 为什么想买？"因为涨了" → 急门，不进 ✅

场景2：中证500 8.94，差 0.24 到目标，想改限价单
  → 为什么想改？"怕明天直接飞了" → 急门，不改 ✅

场景3：中证1000 3.47，距 3.42 只差 0.05，子弹上膛
  → 兔子快出来了，但还没出来 → 急门，再等 ✅
```

> 🎯 市场每天都有机会。今天错过了明天还有，明天错过了下周还有。**但每一次追进去亏掉的钱，不会因为"怕错过"而退回来。财不入急门——七个字，值你交易账户里的每一分钱。**

### 心态红线

| 🚫 绝不做 | 说明 |
|------|------|
| **报复性交易** | 亏了想马上赚回来 → 往往亏更多 |
| **攀比交易** | 别人赚了我也要买 → FOMO 是最贵的情绪 |
| **证明自己** | "我就不信它不涨" → 对手是市场，不是你的自尊 |
| **频繁看盘** | 一天看 50 次账户 → 每次都消耗决策能量 |

### 亏损后的操作流程

```
亏损发生
  → 执行止损（系统自动/手动）
  → 关掉交易软件
  → 等至少 1 个交易日
  → 复盘：为什么亏？买入理由对不对？止损位合理吗？
  → 提炼一条规则写入交易纪律
  → 下次遇到相同情况，系统自动避开
```

## 八、账户管理纪律

| 规则 | 说明 |
|------|------|
| 三个账户**各有定位** | 华泰=ETF底仓、广发=做T主力、东北=观察仓 |
| 逆回购**每日操作** | 闲置资金当天做逆回购，不让一分钱闲着 |
| 每周**统计一次总资产** | 填入净值跟踪表 |
| 每月**对账一次** | 三账户合并，核对持仓与记录是否一致 |
| 密码/设备**安全** | 不在公共 WiFi 交易，定期改密码 |

## 九、复盘纪律

| 频率 | 做什么 | 工具 |
|------|------|------|
| **每日** | 填复盘模板：指数/板块/操作/学到了什么 | 日复盘 |
| **每周** | PDCA 周报：P→D→C→A | 个人周报 |
| **每月** | 月回顾：OKR 进度 + 净值曲线 + 持仓对比 | 月回顾 |
| **每笔交易** | 记录：品种/价格/理由/结果/反思 | 做了什么 |

### 复盘必问三个问题

```
1. 这笔交易，我遵守纪律了吗？
2. 如果重来一次，我会怎么做？
3. 需要新增或修改哪条纪律？
```

> **复盘不是为了后悔，是为了迭代。每一次复盘都让系统更强。**

## 十、绝对红线

```
🚫 永不加杠杆（融资/配资/借钱炒股）
🚫 永不满仓（至少保留 30% 现金）
🚫 永不在亏损时加仓摊成本
🚫 永不追涨停板
🚫 永不因为"别人在赚"就买入
🚫 永不在情绪激动时下单
🚫 永不挪动止损位（只能上移，不能下移）
🚫 永不让单笔亏损超过总资金的 2%
🚫 永不同时持有超过 5 只个股
🚫 永不在不了解的行业重仓
```

## 十一、一页速查

```
┌─────────────────────────────────────────────┐
│                  交易纪律速查                 │
├──────────┬──────────────────────────────────┤
│ 建仓     │ 5问必答 · 尾盘14:45 · 分批50/25/25 │
│ 持仓     │ 有理由才拿 · 理由消失即卖          │
│ 止损     │ 买入即挂 · 触发就走 · 不移不补      │
│ 止盈     │ 分批卖 · 到目标就走 · 跟踪止损      │
│ 加仓     │ 只加盈利仓 · 不摊亏损成本          │
│ 做T      │ 日内≤2次 · 0.5-1%止盈 · 收盘归零   │
│ 心态     │ 不冷静不交易 · 亏后停一天          │
│ 红线     │ 不杠杆 · 不满仓 · 单笔≤2%         │
│ 复盘     │ 每日写 · 每笔记 · 每次提炼一条规则  │
└──────────┴──────────────────────────────────┘
```

> 🔑 **交易纪律的本质：用规则代替情绪做决策。系统越完善，心态越不重要。**
