## 一、建仓前必问清单

每次建仓前，回答这 5 个问题。**任何一个答不上来，就不买。**

```
1. 我为什么买它？（一句话说清逻辑）
2. 我打算拿多久？（一天/一周/一月/长期）
3. 跌多少我止损？（具体价格或百分比）
4. 涨多少我止盈？（具体价格或百分比）
5. 我用多少仓位？（占总资金的百分比）
```

> 如果只能凭"感觉"回答，说明研究不够——不要买。

## 二、时机纪律

### 什么时候可以建仓

| 条件 | 说明 |
|------|------|
| ✅ 重大事件**已落地** | FOMC/CPI/GDP 公布后，不是公布前 |
| ✅ **先清后建** | 问题持仓清理完毕后，才用回收资金建新仓 |
| ✅ **尾盘 14:45-14:55** | 全天方向已定，价格最真实，情绪最小 |
| ✅ 指数在**支撑位**附近 | 均线支撑、整数关口、前期低点 |
| ✅ 缩量**止跌**信号 | 不是放量暴跌时抄底 |

### 什么时候不能建仓

| 条件 | 说明 |
|------|------|
| 🚫 重大事件**前 1-2 天** | FOMC/CPI/非农前不建新仓 |
| 🚫 **开盘 30 分钟内** | 情绪波动最大，容易追涨杀跌 |
| 🚫 **连涨 3 天以上** | 短期超买，回调概率大 |
| 🚫 **全市场恐慌性暴跌** | 等第二天确认止跌再考虑 |
| 🚫 **看到别人赚钱就急** | FOMO 是最贵的情绪 |
| 🚫 **连续亏损后** | 心态已失衡，先停一天 |

## 三、仓位纪律

### 单次建仓上限

| 规则 | 数值 |
|------|------|
| 单只个股 ≤ 总资金 | **10%** |
| 单只 ETF ≤ 总资金 | **15%** |
| 单次买入 ≤ 目标仓位 | **50%**（先买半仓，观察再加） |
| 当日总买入 ≤ 总资金 | **20%** |

### 分批建仓法（核心）

```
目标仓位 2 万元：
  第 1 笔：买入 1 万（50%），观察 2-3 天
  第 2 笔：确认趋势后加 5000（25%）
  第 3 笔：回调不破止损再加 5000（25%）
```

> **永远不要一次性满仓。** 分批买入的好处：即使第一笔买贵了，后面还有摊低成本的机会。

### 不同行情的总仓位

| 行情 | 总股票仓位上限 |
|------|:---:|
| 低位震荡 | 60-70% |
| 上升趋势 | 50-60% |
| **高位震荡** | **30-50%** |
| 下跌趋势 | ≤ 20% |

## 四、品种纪律

### 优先级

| 优先级 | 品种 | 理由 |
|:---:|------|------|
| 🥇 | **宽基 ETF**（沪深300/500） | 分散风险，不会踩雷，适合压舱石 |
| 🥈 | **行业 ETF**（科技/红利/基建） | 分散个股风险，捕捉行业趋势 |
| 🥉 | **龙头个股** | 仅限研究透彻的，仓位严格控制 |
| ❌ | **垃圾股/ST/概念炒作** | 绝不碰 |

### 个股建仓额外要求

- 只买你**能用一句话说清楚商业模式**的公司
- 只买**行业前三**的龙头
- 只买你**愿意持有 1 年以上**的
- 看不懂的行业/公司 → 用 ETF 替代

## 五、执行纪律

### 建仓当天必做

| 步骤 | 动作 |
|:---:|------|
| 1 | 写下买入理由（一句话） |
| 2 | 设定止损价（买入后立即挂条件单） |
| 3 | 设定止盈价（心中有数，不必挂单） |
| 4 | 记录到复盘：品种/价格/数量/理由 |

### 条件单先行

```
买入 → 立刻设止损条件单
  例：买入沪深300ETF 4.10元
  止损：跌破 4.05（-1.2%）自动卖出

没有止损 = 没有安全带的驾驶
```

## 六、建仓后纪律

| 情况 | 操作 |
|------|------|
| 买入后当天就跌 | 不加仓，观察。只有跌到止损位才卖 |
| 买入后连涨 3 天 | 不追加，等回调再考虑加第 2 笔 |
| 买入后横盘不动 | 持有观察，不因为"没涨"就卖 |
| 买入后触发止损 | **无条件执行**，不犹豫不侥幸 |
| 买入理由消失 | 不管盈亏，立刻卖出 |

## 七、绝对红线

```
🚫 永不加杠杆（融资/配资/借钱炒股）
🚫 永不满仓（至少保留 30% 现金）
🚫 永不在亏损时加仓摊成本（越跌越买 = 越套越深）
🚫 永不追涨停板
🚫 永不因为"别人在赚"就买入
🚫 永不在情绪激动时下单（愤怒/兴奋/恐慌）
```

---

> 🔑 **建仓的核心不是"买什么"，而是"怎么买"。好的建仓纪律让你：买对了能拿住，买错了亏得少。**
>
> **巴菲特：投资第一要务是不输钱，其次才是赚钱。**
