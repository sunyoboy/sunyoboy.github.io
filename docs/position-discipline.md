## 一、建仓纪律

### 建仓前必问清单

每次建仓前回答 5 个问题。**任何一个答不上来，就不买。**

```
1. 我为什么买它？（一句话说清逻辑）
2. 我打算拿多久？（一天/一周/一月/长期）
3. 跌多少我止损？（具体价格或百分比）
4. 涨多少我止盈？（具体价格或百分比）
5. 我用多少仓位？（占总资金的百分比）
```

### 买入前 · 避雷三问

> 2026-07-08 西部材料教训沉淀。**以下三个问题任一个答不上来，就不买。**

```
1. 买的是价值还是故事？
   → PE<20 且有利润支撑 = 价值
   → PE>100 或利润同比大降 = 故事（大概率是出货期）
   → 故事可以涨200%，但跌80%的时候你还在里面

2. 主力在买还是在卖？
   → 近10日主力净流入 = 可以跟
   → 近10日主力净流出超5亿 = 不碰
   → 大股东在减持 = 最可靠的空头信号

3. 公司自己觉得贵不贵？
   → 公司发了「非理性炒作/异常波动」公告 = 离场信号
   → 公司发了「股东减持」公告 = 离场信号
   → 扣非净利润同比大降超50% = 基本面恶化
```

**实战案例：西部材料（002149）**

| 三问 | 答案 | 该不该买 |
|------|------|:--:|
| 价值还是故事？ | PE 270倍·扣非利润-96.53% = **故事** | ❌ |
| 主力在买还是卖？ | 10日净流出10.97亿·大股东减持 = **卖** | ❌ |
| 公司觉得贵吗？ | 发了「非理性炒作」公告 = **贵** | ❌ |

> 三个问题全部红灯，但当时还是买了。100股亏19%不是运气，是三个红灯同时亮着还往里冲的结果。

### 时机纪律

| ✅ 可以建仓 | 🚫 不能建仓 |
|------|------|
| 重大事件**已落地** | 重大事件**前 1-2 天**（FOMC/CPI/非农） |
| **先清后建**，问题持仓清完 | 问题持仓还没处理 |
| **尾盘 14:45-14:55** | **开盘 30 分钟内** |
| 指数在**支撑位**附近 | 连涨 3 天以上（短期超买） |
| 缩量**止跌**信号 | 全市场恐慌性暴跌中 |
| 你很冷静 | 你很激动（兴奋/恐惧/愤怒） |

### 仓位纪律

| 规则 | 上限 |
|------|:---:|
| 单只个股 ≤ 总资金 | **10%** |
| 单只 ETF ≤ 总资金 | **15%** |
| 单次买入 ≤ 目标仓位 | **50%** |
| 当日总买入 ≤ 总资金 | **20%** |
| 现金永远保留 | **≥ 30%** |

### 品种纪律

| 优先级 | 品种 |
|:---:|------|
| 🥇 | 宽基 ETF（沪深300/500） |
| 🥈 | 行业 ETF（科技/红利/基建） |
| 🥉 | 龙头个股（研究透彻 + 仓位严控） |
| ❌ | 垃圾股/ST/纯概念炒作 |

### 「不碰」≠「不看」· 候选池跟踪纪律

> 2026-07-10 科创50教训：从1925→2080涨了+8%，我们全程看戏。
> 不是因为判断错了，是因为「不碰」变成了「不看」。

```
❌ 错误: 决定不碰某个品种 → 不再看它
✅ 正确: 决定不碰 → 每天扫一眼涨跌 → 条件变了就重新评估

谁破了这条规则：科创50
  SOX企稳后我们没有更新判断
  对科创50的跟踪停留在「最弱宽基」的标签上
  8%的反弹是一个信号——不是追涨，是提醒我们需要重新评估
```

**候选池扫一遍只需要 30 秒：**

```bash
# 每天盘后顺手看一眼候选池涨跌
python3 ~/.claude/skills/query-indicator/query_indicator.py queryIndicator \
  --query "科创50ETF588000、恒生科技ETF513180、纳指ETF513100最新涨跌幅"
```

**更新判断的触发条件：**

| 原来判断 | 如果出现 | 重新评估 |
|------|------|------|
| SOX拖累·不买科创 | SOX连涨3天 | ✅ 重新看 |
| 死叉·不买黄金 | MA5上穿MA20 | ✅ 重新看 |
| 等回踩·不追能源 | 回踩3-5% | ✅ 动手 |
| FDA压制·不加创新药 | 创新药连涨+放量 | ✅ 重新看 |

> **不是推翻原判断，是在条件变化时更新判断。不碰 = 暂时不买 + 每天看一眼。**

### 分批建仓法

**按仓位比例分批：**

```
目标仓位 2 万元：
  第 1 笔：1 万（50%），尾盘买入，观察 2-3 天
  第 2 笔：5000（25%），趋势确认后加
  第 3 笔：5000（25%），回调不破止损再加
```

> **永远不要一次性满仓。**

**按价格区间分批（兔子不只有一只）：**

> 6/26-30 中证500 实战修正：只设一个目标价 8.70 → 差一毛钱完全踏空。修正为区间分批。

```
中证500 目标区间 8.70-8.85：

  区间上沿 8.85  → 300 份（半仓试探）     ← 轻仓上车
  区间下沿 8.70  → 300 份（重仓出手）     ← 理想价位重仓
  不到 8.85      → 不买（现有仓位足够）

止损统一：8.65
```

| 对比 | 单目标价 | 区间分批 |
|------|:--:|:--:|
| 6/26 8.86 差 0.01 | ❌ 完全踏空 | ✅ 8.85 触发，入 300 |
| 6/30 9.15 | ❌ 踏空到底 | ⏸️ 第一批在赚钱，不追 |
| 错失成本 | 600份全踏空 | 只踏空 300份重仓 |
| 纪律保持 | ✅ | ✅ |

> 🎯 **区间分批 = 不见兔子不撒鹰的升级版。** 兔子不止一只——区间上沿是小兔子（轻仓），区间下沿是大兔子（重仓）。既不过度保守（比单目标多一个上车机会），也不追高（超过上沿绝对不买）。

### 选对 · 拿得住 · 上仓位

> 交易盈利的三个台阶。大部分人死在第一步，少数人到第二步，极少数跨过第三步。

```
选对股  → 只做大概率正确的事
拿得住 → 用系统代替意志力
上仓位 → 正确的策略需要足够的子弹
```

#### 选对：只做 ETF，不碰个股

| ✅ 可以买 | ❌ 不能买 |
|------|------|
| 宽基 ETF（中证500/1000/沪深300/科创50）| 任何 A 股个股 |
| 行业 ETF（煤炭/电网/红利/有色）| 朋友推荐的股票 |
| 黄金 ETF | "研究透了"的股票 |
| | 认为"这次不一样"的股票 |

> 🎯 **自检**：过去半年的盈亏记录中，ETF 和个股各自的胜率是多少？如果 ETF 全面盈利、个股全面亏损——你已经有了答案。

#### 拿得住：条件单代替意志力

```
拿不住不是因为心态差——是因为没有条件单。

建仓后 30 秒内 → 挂止损条件单
买完不挂止损 = 裸奔
挂完不看盘 = 系统在替你拿
```

> 🎯 **不挂止损不建仓。** 这不是建议——是纪律。

#### 上仓位：分批到上限，永不超过红线

| 参数 | 数值 | 为什么 |
|------|:--:|------|
| 当前 | 21% | 过于保守，逆回购利息跑不赢通胀 |
| 第一阶段目标 | **30%** | 宽基触发第一买点后 |
| 第二阶段目标 | **40%** | 宽基触发第二买点后 |
| 永不超过 | **50%** | 留一半现金应对生活+应急 |
| 每批 | 总资产 10% | 分 2-3 批，不一次性满仓 |

**个人上限计算公式：**

```
不可动用底线 = 孕期支出预估 + 生活应急 + 交易安全垫
             = 9万 + 5万 + 5万 ≈ 15万

可投入上限 = 总资产 - 15万
上仓位上限 = 可投入上限 ÷ 总资产 × 100%

→ 实际执行：不超过 50%
```

> 🎯 **三个台阶不是平级的。** 如果第一步（选对）做错了，后面的拿得住和上仓位只会放大错误。选错股 + 上仓位 = 灾难。选对 ETF + 上仓位 = 复利。

### 双轨制：短线试错 vs 长线宽基

> 两套完全不同的逻辑，共用同一个框架外壳，但参数完全不同。**混淆了会两边都亏。**

```
短线试错 = 探路兵             长线宽基 = 主力部队
────────────                  ────────────
目的：验证方向，不是赚钱        目的：耐心持有，赚趋势的钱
周期：3-5 个交易日             周期：3 周 - 3 个月
仓位：≤ 总资产 2%              仓位：≤ 总资产 10%
选品：行业 ETF（高弹性）        选品：宽基 ETF（均衡）
入场：连涨 3 日 + 放量          入场：回踩 MA10/MA20 + 缩量企稳
止损：-5% 或破 MA10             止损：-3% 或破 MA20
止盈：+8% 或有信号反转          止盈：MA5 死叉 或 偏离 >+5%
加仓：不加                      加仓：回调不破止损时分批加
退出后：清仓不留                 退出后：等下次买点再入
```

#### 短线试错 · 参数表

| 参数 | 数值 | 为什么是这个数 |
|------|:--:|------|
| **总仓位上限** | 总资产 × 2%（≈ 5,600 元）| 错了亏 280 元，不影响全局 |
| **单笔试错** | 1,500-3,000 元 | 分散 2-3 个方向探路 |
| **同时持有上限** | 3 个试错仓位 | 再多盯不过来 |
| **入场条件** | ① 行业连涨 ≥ 3 日 ② 放量（≥1.2x 均量）③ PE 不在 90 分位以上 | 三缺一不试 |
| **入场方式** | 限价单（开盘前挂，不盘中追）| 和长线一样——不见兔子不撒鹰 |
| **止损** | **-5%** 或破 MA10（哪个先到执行哪个）| 短线不扛单，快死快埋 |
| **止盈** | **+8%** 或出现首根放量阴线 | 短线吃到就走，不贪 |
| **最大持仓天数** | **5 个交易日** | 到期不赚也走——时间到了说明判断错了 |
| **退出后** | 清仓，不补，不回头看 | 试错 ≠ 建仓。退出后即使涨了也别后悔——它已经完成了探路的使命 |

#### 长线宽基 · 参数表

| 参数 | 数值 | 为什么是这个数 |
|------|:--:|------|
| **总仓位上限** | 总资产 × 10%（≈ 28,000 元）| 核心仓位，重点持有 |
| **单次建仓** | 目标仓位 × 50%（首笔），分批加 | 不一次性满仓 |
| **入场条件** | ① 七层全绿 ② 回踩 MA10/MA20 ③ 缩量企稳 | 等最好的价格 |
| **入场方式** | 区间分批限价单（上沿轻仓+下沿重仓）| 兔子不止一只 |
| **止损** | **-3%** 或破 MA20 | 宽基波动小，止损位可以更紧 |
| **止盈** | MA5 死叉 或 偏离 >+5% | 趋势走坏才走 |
| **最大持仓天数** | **3 个月**（可滚动）| 宽基拿得住比买得低更重要 |

#### 两轨不能混的铁律

```
❌ 试错亏了 → "我再拿一拿，万一反弹" = 把短线变长线 = 纪律崩了
❌ 长线涨了 → "快卖快卖，别跌回去" = 把长线变短线 = 捡了芝麻丢西瓜
❌ 试错赚了 → "再加点，这个方向对了" = 试错变建仓 = 仓位从 2% 暴增到 10%
❌ 长线亏了 → "再试一个方向" = 长线亏钱后报复性试错 = 连亏

✅ 试错就按试错出，长线就按长线拿。两套参数、两个账户心态、两种退出标准——绝不混用。
```

### 第三轨：战略底仓（国家硬目标 · 不清零 · 不挂战术止损）

> 短线试错、长线宽基**都设止损**。但有第三类仓位——**国家硬目标战略底仓**，规则完全不同：不挂战术止损、不清零、扛波动，赚国运的钱。

**什么进战略底仓**：有国家承诺 + 明确时间表的十五五硬目标赛道——载人登月(2030)/航天、芯片自主可控、双碳(2030)、建军百年(2027)。对应 ETF 见 [对标美股·十五五 ETF](fifteen-five-us-benchmark.md)。

| 维度 | 战略底仓 | 战术仓（试错/宽基） |
|------|------|------|
| 止损 | **不挂战术止损** | 严格止损·触发必执行 |
| 仓位 | 每赛道 5-10%·**永不清零** | 试错 2% / 宽基 10% |
| 退出 | 不清零；极端过热可减仓锁利·回调补回 | 触发即清 |
| 心态 | 扛波动·赚国运的钱 | 快进快出 / 趋势 |

```
❌ 把战略底仓挂战术止损 → 一个技术破位就割掉国运赛道 = 卖飞（卫星/天山 W28 教训）
❌ 用"卖出后涨停"反推"卖错了" → 结果论·单次结果是噪声·看框架不看次日 K 线
❌ 追高买入当战略底仓（卫星 6.77）→ 战略看好 ≠ 任何价格都能买·入场仍要择时
✅ 战略底仓：择时入场 · 不挂战术止损 · 不清零 · 可换标的但不丢敞口
```

> ⚠️ **最常见的错**：把战略底仓错放进战术账本、挂了止损 → 该扛的波动被止损割掉。**分对账本，才不会在国运赛道上卖飞。**

#### 每日检查清单

```
收盘后：
  1. 跑 shenwan-monitor.py → 看领涨/领跌
  2. 跑 ma5-deviation.py → 看持仓信号
  3. 试错仓位检查：

  4. 长线仓位检查：
     → 中证500 有没有到 8.85/8.70？→ 挂限价单
     → 有没有触发 MA5 减仓信号？→ 明天执行
```

## 二、持仓纪律

| 场景 | 操作 |
|------|------|
| 买入理由**仍然成立** | 继续持有，不因短期波动卖出 |
| 买入理由**已消失** | 不管盈亏，**立刻卖出** |
| 持仓涨了但**不确定要不要卖** | 看止盈规则，而不是看感觉 |
| 持仓跌了但**不想止损** | 执行止损，不做"下一次会反弹"的赌注 |
| 持仓一直**横盘不动** | 给自己设一个观察期限（如 2 周），到期未动则减仓换标的 |
| **盘中频繁看持仓** | 关掉 APP。你已经设了止损条件单，不需要盯盘 |

> **持仓的核心：不是"拿着"，而是"有理由地拿着"。**

## 三、止损纪律

### 止损是必须的

```
没有止损 = 没有安全带的驾驶
止损不是认输，是保留战斗力

止损 = 价格 × 时间，缺一不可
价格到了你认，时间到了你不认 → 大多数亏损是等出来的
```

### 止损规则（双维度）

| 品种 | 价格止损 | 时间止损 | 说明 |
|------|:--:|:--:|------|
| 宽基 ETF | **-3%** | 不设 | 大盘ETF·越跌越买 |
| 行业 ETF | **-5%** | **20个交易日** | 横盘20天没方向→换 |
| 个股（短线） | **-5%** | **5个交易日** | 短线不做成长线 |
| 个股（中长线） | **-8%** | **10个交易日** | 10天不涨→逻辑可能错了 |
| 做 T 单次 | **-0.8%** | **当日收盘前** | 不过夜 |

### 时间止损的优先级

```
价格止损 vs 时间止损 → 谁先触发谁执行

买入第3天: 跌了2%（未触价格止损）→ 继续持有
买入第5天: 跌了3%触价格止损 → 走人
买入第6天: 横盘没涨 → 时间止损触发 → 走人（哪怕没亏钱）

→ 时间止损的核心：不给"再等等"任何借口
```

### 止损执行

### 止损规则

| 品种 | 止损线 | 说明 |
|------|:---:|------|
| 宽基 ETF | **-3%** | 大盘 ETF 波动小，-3% 已异常 |
| 行业 ETF | **-5%** | 行业波动大一些 |
| 个股（短线） | **-5%** | 短线追求快进快出 |
| 个股（中长线） | **-8%** | 给更大的波动空间 |
| 做 T 单次 | **-0.8%** | 日内必须严格 |

### 止损执行

| 规则 | 说明 |
|------|------|
| **买入后立刻挂条件单** | 不靠意志力止损，靠系统自动执行 |
| **触发就走，不犹豫** | "再等等看"是亏损扩大的开始 |
| **不移动止损位** | 止损线一旦设定，只能上移（跟踪止损），不能下移 |
| **止损后不立刻回补** | 至少等 1 个交易日冷静 |
| **单日止损 2 次 → 当日停手** | 说明今天判断出错，不再操作 |

### 跟踪止损（盈利后保护利润）

```
买入价 10.00，止损设在 9.50（-5%）
涨到 11.00 → 上移止损到 10.45（保住 4.5% 利润）
涨到 12.00 → 上移止损到 11.40
回落到 11.40 → 自动卖出，锁定 +14% 利润
```

> **跟踪止损 = 让利润奔跑，同时锁住已有收益。**

## 四、止盈纪律

### 止盈比止损更难

```
止损难在"不愿认错"
止盈难在"还想更多"

止盈 = 价格 × 时间，缺一不可
赚够了不走 = 把利润还给市场
```

### 止盈规则（双维度）

| 品种 | 价格止盈 | 时间止盈 | 方式 |
|------|:--:|:--:|------|
| 宽基 ETF | 涨+8% | 不设 | 分批：5%卖1/3·8%卖1/3·剩1/3跟踪 |
| 行业 ETF | 涨+10% | **20个交易日** | 分批：5%卖1/3·10%卖1/3·剩1/3跟踪 |
| 个股（短线） | +5% | **5个交易日** | 到了就走·不恋战 |
| 个股（中长线） | +8% | **20个交易日** | 跟踪止损上移 |
| 做 T 单次 | +0.5~1% | **当日收盘前** | 到了就走 |

### 止盈策略

| 策略 | 方式 |
|------|------|
| **目标价止盈** | 买入时设目标价，到了就卖，不贪 |
| **分批止盈** | 涨 5% 卖 1/3，涨 10% 再卖 1/3，剩 1/3 用跟踪止损 |
| **技术面止盈** | 跌破 5 日线 / 出现顶背离 / 放量滞涨 → 减仓 |
| **时间止盈** | 持有超过预定期限且未达目标 → 卖出换标的 |
| **做 T 止盈** | 0.5%-1.0%，到了就走，不恋战 |

### 双维度触发原则

```
价格 vs 时间 → 谁先触发谁执行

做T买入当天: 收盘前未达+0.5%止盈 → 时间到了 → 走人（哪怕+0.2%）
个股买入第8天: 涨了+6%触止盈 → 卖1/3锁利
个股买入第11天: 横盘没涨 → 时间止损触发 → 全走
```

### 止盈心态

| ❌ 错误想法 | ✅ 正确想法 |
|------|------|
| "再拿一会儿赚更多" | 到了目标就执行，没人能卖在最高点 |
| "卖了万一继续涨怎么办" | 卖掉的那一刻利润就是你的，之后的涨跌跟你无关 |
| "这次不一样" | 每次都"不一样"，但纪律是一样的 |

## 五、加仓与减仓纪律

### 建仓 ≠ 加仓 ≠ 补仓

> 三个词只差一个字，账户结果差十倍。

| | 建仓 | 加仓 | 补仓 |
|------|------|------|------|
| **动作** | 0 → 有仓位 | 有仓位 → 更多 | 亏了 → 再买摊低成本 |
| **前提** | 七层全绿 | 盈利中 + 趋势确认 + 回调不破止损 | "觉得还会涨回来" |
| **风险** | 可控（设了止损）| 可控（未超仓位上限）| **不可控（放大亏损）** |
| **框架态度** | ✅ 允许 | ✅ 允许（有条件）| 🔴 **绝对禁止** |
| **例子** | 6/26 入中证500 900份@8.86 | 回踩 8.70 → 加 600 份 | 跌到 8.50 再加 1000 份 ← 不行 |

**最容易犯的错：**

```
建仓后跌了 → "再加点摊低成本"
                ↑
              这不是加仓，是补仓。
              补仓 = 把一个小错放大成一个大错。

正确做法：
  建仓后跌了 → 到止损 → 清仓
  建仓后涨了 → 回调到买点 → 加仓
  建仓后横盘 → 不动，等方向
```

### 加仓纪律（加在对的地方）

**加仓的三个条件，缺一不叫加仓：**

```
① 现有仓位在盈利 — 不在盈利的加仓 = 补仓 = 错
② 趋势方向确认 — MA5/MA20 金叉 + 量能配合
③ 价格回到合理区间 — 回调不破止损位
```

| ✅ 可以加仓 | 🚫 不能加仓 |
|------|------|
| 趋势向上 + 回调到支撑位 | 趋势向下 + "便宜了再买点摊成本" |
| 首笔盈利 + 逻辑不变 | 首笔亏损 + 想通过加仓回本 |
| 加仓后总仓位仍在上限内 | 加仓后超过单只 10% / 总仓位上限 |

> **核心：只加盈利的仓，不加亏损的仓。** 建仓是框架允许你的第一步，加仓是趋势奖励你的第二步，补仓是情绪惩罚你的无底洞。

### 减仓纪律（何时减）

| 信号 | 操作 |
|------|------|
| 涨幅达到止盈目标 | 减 1/3 到 1/2 |
| 成交量异常放大但价格不涨（放量滞涨） | 减 1/2 |
| 重大利好兑现（利好出尽） | 全部卖出 |
| 重大事件前（FOMC/CPI） | 减至安全仓位 |
| 基本面发生负面变化 | 不管盈亏，坚决减仓 |

## 七、心态纪律

### 交易前检查情绪

```
问自己：我现在冷静吗？
  冷静 → 可以交易
  兴奋 → 可能追高，暂停 30 分钟
  恐惧 → 可能恐慌卖出，暂停 30 分钟
  愤怒（刚亏了） → 今天不交易
```

### 不见兔子不撒鹰

> 猎人的逻辑，不是赌徒的逻辑。这是交易心态纪律最精炼的七个字。

```
猎人（框架交易）              赌徒（情绪交易）
─────────────                ─────────────
看见兔子 → 放鹰               没看见兔子 → 先放鹰再说
确定目标 → 出手               怕错过 → 出手
一击必中                      十次九空
```

**六个字拆解：**

| 字 | 对应 | 含义 |
|------|------|------|
| 兔 | 目标价 / 入场条件 | 不是感觉，是具体数字。中证500 的 8.70、中证1000 的 3.42 |
| 鹰 | 限价单 / 资金 | 提前挂好，不是盘中手动追 |
| 见 | 触发确认 | 价格到了、条件满足了——不是"快到了" |
| 撒 | 执行 | 条件触发→机械执行，不犹豫不修改 |

**三个问号自检：**

```
问自己：兔子在哪？
  → 中证500 目标 8.70，现在 8.94 → 兔子没出来 ❌

问自己：为什么想撒鹰？
  → 因为科创50 +4.41%、全市场在涨 → FOMO ❌

问自己：现在撒鹰能打中吗？
  → 追涨进去，止损空间不够 → 不能 ❌
```

**和框架的映射：**

| 框架层 | 不见兔子不撒鹰 |
|------|------|
| 第二层（技术诊断）| 兔子 = 入场价/止损价，没算出来就不动手 |
| 第七层（执行指令）| 鹰 = 限价单，挂上去就不改，没触发就等 |
| 心态纪律 | 见 = 条件确认，不是"我觉得要涨" |

> 🎯 6/26 等了六天才等到兔子。今天下午全市场都在涨，系统1 在尖叫"再不买就没了"——但猎人知道：**兔子没出来，撒鹰就是浪费箭。限价单挂在那儿，到了自然成交。不到，不动。**

### 财不入急门

> 越急越穷。不是鸡汤——是交易数学。

```
急 → 怕错过 → 追涨 → 买在日内高点 → 第二天低开 → 止损 → 更急
                                                    ↑____循环____|
```

**急的四种表现 & 数学后果：**

| 急的表现 | 数学后果 |
|------|------|
| 追涨（不等限价单触发）| 买到的比等到的贵，止损空间被压缩，盈亏比变差 |
| 频繁交易 | 摩擦成本（手续费+滑点）每次约 0.1-0.3%，十次就吃掉 1-3% |
| 上午跌下午就补仓 | 没给市场足够时间确认方向——不是在抄底，是在赌底 |
| 刚止损就买下一个 | 情绪驱动，不是框架驱动——下一个大概率也止损 |

**和「不见兔子不撒鹰」的分工：**

```
不见兔子不撒鹰 → 管住手（行为纪律）
财不入急门     → 管住心（情绪纪律）
两个加一起     → 框架完整执行
```

**三个场景自检：**

```
场景1：科创50 +4.41%，想追
  → 为什么想买？"因为涨了" → 急门，不进 ✅

场景2：中证500 8.94，差 0.24 到目标，想改限价单
  → 为什么想改？"怕明天直接飞了" → 急门，不改 ✅

场景3：中证1000 3.47，距 3.42 只差 0.05，子弹上膛
  → 兔子快出来了，但还没出来 → 急门，再等 ✅
```

> 🎯 市场每天都有机会。今天错过了明天还有，明天错过了下周还有。**但每一次追进去亏掉的钱，不会因为"怕错过"而退回来。财不入急门——七个字，值你交易账户里的每一分钱。**

### 心态红线

| 🚫 绝不做 | 说明 |
|------|------|
| **报复性交易** | 亏了想马上赚回来 → 往往亏更多 |
| **攀比交易** | 别人赚了我也要买 → FOMO 是最贵的情绪 |
| **证明自己** | "我就不信它不涨" → 对手是市场，不是你的自尊 |
| **频繁看盘** | 一天看 50 次账户 → 每次都消耗决策能量 |

### 亏损后的操作流程

```
亏损发生
  → 执行止损（系统自动/手动）
  → 关掉交易软件
  → 等至少 1 个交易日
  → 复盘：为什么亏？买入理由对不对？止损位合理吗？
  → 提炼一条规则写入交易纪律
  → 下次遇到相同情况，系统自动避开
```

> 详见 [想赢怕输的心态](fear-greed) · [保持好心态](peace-of-mind)

|------|------|------|

```

```

> **复盘不是为了后悔，是为了迭代。每一次复盘都让系统更强。**

## 十、绝对红线

```
🚫 永不加杠杆（融资/配资/借钱炒股）
🚫 永不满仓（至少保留 30% 现金）
🚫 永不在亏损时加仓摊成本
🚫 永不追涨停板
🚫 永不因为"别人在赚"就买入
🚫 永不在情绪激动时下单
🚫 永不挪动止损位（只能上移，不能下移）
🚫 永不让单笔亏损超过总资金的 2%
🚫 永不同时持有超过 5 只个股
🚫 永不在不了解的行业重仓
🚫 个股浮亏达-20% = 无条件清仓（最后的硬止损·任何板块通用）
🚫 连续三根大阳线后不追涨·已持仓的主动减仓1/3
🚫 月初/月末/大节假日前不开新仓（见下方详解）
```

### 为什么月初/月末/大节假日不做？

| 时段 | 原因 | 风险 |
|------|------|------|
| **月初** | 资金面偏紧（逆回购到期）+ 机构调仓换股 + 方向不明 | 新开仓胜率天然偏低 |
| **月末** | 季末效应 + 窗口粉饰 + 基金排名扰动 | 假信号多，真突破少 |
| **大节假日** | 避险情绪 + 交投清淡 + 假期可能有黑天鹅 | 节后跳空风险不可控 |

```
实战验证：
  7/1（月初）→ 新刀第一天，高开被砸
  7/2（月初）→ 全线暴跌，被迫清仓三只
  7/2 清仓黄金 → 冷静期5天 → 躲过了黄金从9.02→8.60的跌幅

→ 这三类时段不开新仓，只做持仓管理和条件单维护
→ 原持仓的止损/减仓仍然正常执行（不受此条限制）
→ 如果已经持有，出现止损信号 → 照常执行，不因时间窗口而拖延
```
