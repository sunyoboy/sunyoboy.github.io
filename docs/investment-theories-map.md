## 三、交易系统的分级对应

```
第一层：大类配置（格伦姆）
  → 今天股票 60% or 40%？债券补了吗？有没有浪费现金？

第二层：买什么（博格 + 巴菲特 + 林奇）
  → 宽基指数就够了。你懂 ETF 就买 ETF，不懂的不碰。

第三层：买多少（范·撒普）
  → 每笔风险 ≤ 总资产 2%。港股通互联网 1,170 元 ok。

第四层：怎么判断（马克斯 + 芒格）
  → 钟摆在极左吗？capitulation 信号到了吗？反过来想。

第五层：怎么扛（达利欧 + 塔勒布）
  → 全天候配置·国债是对冲·国债是生存底线。

第六层：怎么改（加文·贝克 + 曹名长）
  → 亏钱=错了→止损。跨低栏杆→买算得清的。
```

## 六、你的实战组合

```
格伦姆 50/50（默认）
  ├── 股票 50%：沪深300·中证500·科创50·创业板·行业ETF（博格式宽基）
  ├── 债券 50%：511090 30年国债
  └── 加一点：黄金ETF + 有色ETF（达利欧·全天候）

单笔纪律：
  └── 范·撒普：每笔风险 ≤ 2%

仓位偏离（只在极端时）：
  capitulation 三信号确认 → 偏股 60/40（钟摆极左·马克斯）
  六行业过热 → 回到 50/50 或偏债（钟摆极右）

纠错机制（加文·贝克）：
  亏了 = 错了 → 止损 → 写复盘 → 不改方向改方法
```
