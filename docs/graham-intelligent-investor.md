## 二、四大核心思想

### 1. 市场先生

每天有一个人叫"市场先生"，敲门给你报个价——愿意按这个价买你的股票，也愿意按这个价卖给你。有时候他报价很高（亢奋），有时候很低（沮丧）。

> **你不能被他影响，你只能利用他。** 他报高价你就卖给他，报低价你就从他手里买。其他时间，不理他。

**映射**：CLAUDE.md「不盯盘」——市场先生每天都在你耳边喊价，你每听一次，就多一次被他影响的机会。条件单 = 不等他喊价，你自己定好价。

### 2. 安全边际

**用 0.5 元买 1 元的东西。** 你估出来一家公司值 100 块，不是 100 块买——等跌到 50 块再买。这 50 块的差就是安全边际。你估错了、出事了——有这 50% 缓冲你也死不了。

**映射**：MA5 回踩买入法——不在大涨日追，等回踩均线（打折）再买。止损线也是安全边际——亏 5% 就走，不让你亏 50%。

### 3. 25-75% 配置规则

**股票仓位永远在 25% 到 75% 之间，默认 50/50，极端时向一端倾斜。**

```
正常市场：50% 股票 + 50% 债券
股票太贵了：25% 股票 + 75% 债券（减仓锁利）
股票太便宜了：75% 股票 + 25% 债券（抄底·债券是弹药库）
```

**这不是择时策略——是让你在极端时保持清醒的锚。** 股市涨上天 → 不是天才，降股票到 50%。股市崩盘 → 不是世界末日，加股票到 75%。

> ⚠️ **债券 ≠ 逆回购。** 逆回购是现金，不是债券。格伦姆的债是长期国债（30年）。逆回购年化 1.4% 跑不赢通胀，等于在贬值。

### 4. 不要亏钱

**"投资第一原则是不要亏钱，第二原则是不要忘记第一条。"** 不是说每笔交易都不能亏——而是亏 50% 要涨 100% 才能回本。**控制回撤比追求收益更重要。**

**映射**：你的 -5%/-8%/-20% 三级止损体系——不是让你赚钱的，是让你不被单笔交易毁灭的。

## 三、配置纪律（核心）

### 铁律三条

| # | 纪律 | 说明 |
|:--:|------|------|
| 1 | **股票永远在 25-75% 之间** | 每个账户单独适用·不满仓股票·不空仓 |
| 2 | **债券仓位 ≥ 25%** | 格伦姆底线——对三账户均适用 |
| 3 | **不浪费现金** | 逆回购 = 闲置资金 ≠ 债券。需要用的钱放逆回购等机会，不需要用的钱买国债赚票息 |

### 当前市场定位

```
capitulation 三信号确认（7/21） → 股票相对便宜
  → 可以偏股，不用偏债
  → 推荐 60% 股票 / 40% 债券
  → 不是抄底——是回到偏股配置
```

### 三账户对照

| 账户 | 当前 股/债 | 目标 股/债 | 操作 |
|------|:--:|:--:|------|
| 广发 | 69% / 0% 🚨 | 60% / 40% | 降股票 9pp→买国债 |
| 东北 | 32% / 0% 🚨 | 55% / 45% | 加股票 23pp→按回补计划·逆回购转国债 |
| 省心投 | 54% / 0% 🚨 | 55% / 45% | 逆回购→国债 |

> 🚨 三账户债券全为零——今天卖国债反了格伦姆方向。

### 怎么调仓

```
不是明天全部完成。分两周：
  第一周：逆回购到期后→先买国债把债券仓位补到 25%
  第二周：按回补计划加股票·东北加仓

日常维护：
  每月最后一个周末检查一次 股/债 比例
  如果股票涨太多→偏离 > 10pp → 再平衡（卖股买债）
  如果股票跌太多→偏离 > 10pp → 再平衡（卖债买股）
```
