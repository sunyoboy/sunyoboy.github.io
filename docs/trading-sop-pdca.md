# 交易执行 SOP · PDCA 循环

> 2026-07-08 · 基于 `position-discipline.md` 全部纪律条款，整合为可执行的日常流程。
>
> 核心理念：**纪律靠系统不靠意志力。条件单 > 手动下单。检查清单 > 盘中判断。**

---

## 一、PDCA 总览

```
P (Plan)   盘前 30min  → 跑脚本·看信号·挂条件单
D (Do)     盘中 0min   → 条件单自动执行·不盯盘
C (Check)  盘后 30min  → 复盘·检查偏离率·记录
A (Act)    周末 2h     → 周复盘·调整策略·校准参数
```

---

## 二、P · 盘前 SOP（9:00-9:30）

### Step 1: 跑脚本（5min）

```bash
python3 scripts/ma5-deviation.py     # 持仓均线诊断
python3 scripts/shenwan-monitor.py   # 申万31行业全景
```

### Step 2: 检查信号（10min）

| 检查项 | 来源 | 行动 |
|------|------|------|
| 是否有高危/偏热信号？ | MA5偏离度 表一 | 有→挂减仓条件单 |
| 是否有偏冷/冰点信号？ | MA5偏离度 + 申万 | 在计划内→挂买入条件单 |
| 金叉/死叉变化？ | MA5偏离度 表二 | 金叉变死叉→停止加仓 |
| 是否触发止损/止盈？ | 条件单系统 | 已触发→确认·不反悔 |

### Step 3: 挂条件单（15min）

> **条件单 = 你的离合器。挂完后关掉APP，盘中不需要看。**

| 条件单类型 | 适用场景 | 券商工具 |
|------|------|------|
| **止损卖出** | 个股-8%·行业ETF-5%·宽基-3% | 华泰/广发 条件单→止损卖出 |
| **止盈卖出** | 涨+5%卖1/3·+10%再卖1/3 | 条件单→分批止盈 |
| **限价买入** | 中证500@8.50·证券ETF@1.120 | 条件单→限价买入 |
| **跟踪止损** | 盈利后锁定利润 | 条件单→跟踪止损（上移） |
| **时间止盈/止损** | 到期未触发→撤单重新评估 | 条件单→到期日设置 |

### Step 4: 网格交易（如有）

```
适用于: 横盘震荡品种（如30年国债、可转债、银行ETF）
不适用: 趋势品种（煤炭暴涨暴跌时不开网格）

参数设置:
  基准价: 现价
  网格间距: 行业ETF 2%·宽基1.5%·债券0.3%
  每格金额: 总仓位上限 ÷ 网格层数（至少5层）
  触发条件: 只买跌不买涨方向的网格（单边网格）
```

### 盘前检查清单

```
□ ma5-deviation.py 已跑
□ shenwan-monitor.py 已跑
□ 高危/偏热 → 减仓条件单已挂
□ 偏冷/冰点 → 买入条件单已挂（计划内才挂）
□ 止损/止盈条件单仍在有效期内
□ 今日是否有 FOMC/CPI/非农/节假日？
  → 有 → 不挂新买单
  → 无 → 正常挂单
□ 单日已挂条件单不超过3个新单
```

## 三、D · 盘中 SOP（9:30-15:00）

### 核心原则：不盯盘

```
✅ 条件单在替你工作 → 你不需要看
✅ 只处理极端情况 → 跌停/涨停/放量暴跌才看一眼
❌ 手动刷新持仓 → 每看一次消耗一次决策能量
❌ 盘中临时起意下单 → 这是偏离率的来源
```

### 极端情况处理

| 触发条件 | 行动 |
|------|------|
| 持仓跌停 | 检查是否触止损 → 触发了自动卖·没触发设新的 |
| 持仓涨停 | 检查止盈条件单 → 触发分批卖 |
| 大盘暴跌>3% | 不恐慌卖出·不加仓（等次日评估） |
| 大盘暴涨>3% | 不追涨·检查止盈单是否触发 |
| 条件单成交通知 | 确认·记录·不再手动追加 |

### 盘中纪律

```
🚫 盘中不新开仓位（除非条件单自动触发）
🚫 盘中不修改条件单（除非止损上移）
🚫 盘中不看账户盈亏（收盘后再看）
🚫 盘中不追涨杀跌
```

## 四、C · 盘后 SOP（15:00-16:00）

### Step 1: 跑脚本（5min）

```bash
python3 scripts/fetch-market-data.py  # 生成复盘模板
```

### Step 2: 填写复盘（15min）

按 `review/` 模板顺序填写：指数→标的→盘面→驱动→板块→技术面→后市

### Step 3: 执行检查（10min）

#### 偏离率检查

```
今日操作共 ___ 笔
其中按计划执行的 ___ 笔
偏离计划的 ___ 笔

偏离率 = 偏离笔数 ÷ 总笔数 × 100%
目标：偏离率 < 10%
如果 > 20% → 明天全天不手动下单，只靠条件单
```

#### 止损止盈检查

```
□ 今日是否有止损触发？→ 有·确认已卖出·不反悔
□ 今日是否有止盈触发？→ 有·记录盈利
□ 今日是否有时间止损到期？→ 有·评估是否撤单
□ 今日是否有品种触发买入条件单？→ 有·确认成交
```

#### 现有持仓审查

```
□ 每只持仓的买入理由是否仍然成立？
  → 不成立 → 明天挂卖出条件单
  → 成立 → 维持持有
□ 有没有需要上移跟踪止损的？
  → 盈利>5% → 上移止损到成本价
  → 盈利>10% → 上移止损到+5%
```

### Step 4: 挂明日条件单（10min）

```
□ 明日买入挂单（限价单·不超过3个）
□ 明日卖出挂单（止损/止盈/减仓）
□ 确认所有挂单有效期（默认5个交易日·到期提醒）
```

## 五、A · 周末 SOP（周六/周日）

### 每周检查

| 检查项 | 内容 | 时间 |
|------|------|:--:|
| **胜率统计** | 本周几笔盈利/几笔亏损/偏离率 | 15min |
| **条件单审计** | 哪些触发了·哪些过期了·哪些该撤 | 10min |
| **纪律遵守** | 本周偏离率 vs 目标10% | 5min |
| **持仓优化** | 品种是否超过12只·有无重复 | 10min |
| **网格参数** | 网格是否正常运行·间距是否需要调整 | 5min |
| **周复盘** | P→D→C→A 完整记录 | 30min |

### 每月检查

| 检查项 | 内容 |
|------|------|
| **净值曲线** | 月度盈亏·最大回撤·夏普比率（如有） |
| **OKR对齐** | 量化/选股/写作各飞轮进度 |
| **纪律更新** | 本月出现过的新错误→是否需要新增纪律条款 |
| **低吸区间** | 更新 `fetch-market-data.py` 的 BUY_ZONES |
| **止损线** | 根据行情调整关键价位的止损参数 |
