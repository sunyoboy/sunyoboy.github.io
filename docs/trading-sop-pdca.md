## 一、盘前 · 9:00-9:25 · 25分钟

### 1.2 过信号清单（5min）

| 信号 | 来源 | 行动 |
|------|------|------|
| 🔴 正偏高危（>+5%） | MA5 表一 | **挂减仓条件单**（如紫金上次+6.6%→挂29减仓） |
| 🔴 死叉 + 中期空头 | MA5 表二 | 停止加仓·已有持仓设止损 |
| 🟡 正偏偏热（+2~5%） | MA5 表一 | 观察·偏离扩大=危险·收窄=正常 |
| 🟡 负偏偏冷（-2~-5%） | MA5 表一 | 金叉+中期多头→可挂低吸单·死叉→不买 |
| 🔵 申万冰点（-5%以下） | 申万监测 | 等企稳信号（下影线/十字星）不抄底 |
| 🔴 止损线被触发 | 条件单 | 确认已成交·不反悔·不等"反弹" |

## 二、盘中 · 9:30-15:00 · 0分钟

### 一原则：不盯盘

条件单在替你执行。打开APP看行情 = 消耗决策能量 = 增加手动偏离概率。

### 只看三种极端

| 情况 | 行动 |
|------|------|
| 持仓跌停 | 检查止损单是否触发·没触发补挂 |
| 大盘暴跌>3% | 不恐慌卖出·不加仓（第二天评估） |
| 条件单成交通知 | 看一眼确认·不再手动追加 |

### 盘中铁律

```
🚫 不新开仓位（条件单触发除外）
🚫 不修改条件单（止损上移除外）
🚫 不因为"跌多了"就补仓
🚫 不因为"别人涨了"就追
🚫 不做T（上班族没时间盯盘）
```

## 六、买入前的避雷三问

> 任何新买入之前，三个问题必须全部答上来。任一个答不上来 → 不买。
> 来源：position-discipline.md 「买入前·避雷三问」+ 西部材料教训

```
1. 价值还是故事？
   □ PE < 20 且有利润支撑 → 价值 → 可以买
   □ PE > 100 或利润崩塌 → 故事 → 不买
   □ ETF 看 PE 分位（<20% = 价值）

2. 主力在买还是卖？
   □ 近10日主力净流入 → 可以跟
   □ 近10日主力净流出超5亿 → 不碰
   □ 大股东在减持 → 最可靠的空头信号

3. 公司/品种自己觉得贵吗？
   □ 发了「非理性炒作」公告 → 不买
   □ 大股东减持公告 → 不买
   □ 扣非利润同比大降 >50% → 不买
```

## 七、卖出决策树

```
持仓跌了 → 触止损了？
  ├── 触了 → 立刻卖（不犹豫）
  └── 没触 → 买入理由还在？
            ├── 在 → 持有
            └── 不在 → 立刻卖（不管盈亏）

持仓涨了 → 触止盈了？
  ├── 触了 → 分批卖（目标价/跟踪止损）
  └── 没触 → 买入理由还在？
            ├── 在 → 上移跟踪止损·继续持有
            └── 不在 → 卖

持仓横盘 → 时间止损到了？
  ├── 到了 → 卖（不给"再等等"借口）
  └── 没到 → 维持·到期日提醒
```

## 八、品种数量管控

```
总持仓:  ≤ 12只（当前19只·超标7只）
每账户:  省心投≤4·东北≤4·广发≤6
单品种:  宽基≤15%·行业ETF≤10%·个股≤5%
单行业:  ≤ 25%

超标的处理:
  → 不在候选池的 → 先出
  → 两账户重复的 → 出一份
  → 买入理由已消失的 → 立刻出
  → 超小仓（<1%）→ 清掉·回收注意力
```
