## 三、画风诊断：实物资产 + 电力 + 卡脖子，不是 AI 成长

- ⚠️ 矿业 / 有色 / 石油 / 煤炭 / 船舶 / 交运（**一半仓位**）在美股不是 AI 主升浪，是"通胀交易 + 红利防御"逻辑，属能源 / 材料板块的价值修复
