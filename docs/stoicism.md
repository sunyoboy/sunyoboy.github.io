# 斯多葛主义（Stoicism）

> 古希腊/罗马最具影响力的哲学流派之一 · 控制二分法是核心

---

## 一句话

**有些事取决于你，有些事不取决于你。** ——爱比克泰德《手册》第一条

---

## 控制二分法

| 取决于你的事 | 不取决于你的事 |
|------|------|
| 你的判断 | 别人的看法 |
| 你的选择 | 天气、疾病、死亡 |
| 你的欲望与厌恶 | 财富、名声、权力 |
| 你的行动意图 | 结果 |

痛苦来自试图控制不可控之事。自由来自**只关注可控之事，坦然接受不可控之事**。

---

## 四大美德

| 美德 | 含义 |
|------|------|
| **智慧** | 分清什么可控制、什么不可控制 |
| **勇气** | 在知道结果不确定时仍然行动 |
| **正义** | 公平对待他人，记住我们是共同体 |
| **节制** | 不被欲望和恐惧支配 |

## 美德是练出来的，不是天生的

> 核心认知纠正：人们习惯把伟人的高尚品格归为天赋出身，这会让普通人放弃自我提升。**亚里士多德 + 爱比克泰德共识：美德不是天生拥有，而是靠反复行动练习习得的技能。**

### 爱比克泰德成长行动公式

```
第一步：明确自己想成为什么样的人
第二步：持续做对应行动
```

| 想成为 | 就持续做 |
|------|------|
| 慷慨 | 主动多付出 |
| 勇敢 | 主动直面恐惧 |
| 诚实 | 不回避真实观点 |
| 自律早起 | 定闹钟，到点就起 |

> 成长没有复杂公式，只有持续行动。

### 习惯养成科学依据

- **伦敦大学学院（UCL）研究**：平均 **66 天**就能把行为变成不用刻意坚持的自动习惯
- **时间锚点**：现在开始培养一个习惯，到 9 月就能稳定成型——适合给自己设短期成长目标

### 坚持难题解决方案

| 痛点 | 对策 |
|------|------|
| 独自自律容易放弃 | 找同伴、社群互相监督打卡 |
| 不知道自己有没有进步 | 定期前后状态对比，清晰看见变化 |
| 觉得"我天生不够好" | 品格不是天赋，是练出来的——不用自我否定 |

## 与交易/生活的精确映射

### 交易
- **可控制**：入场价、止损线、仓位大小、研究深度
- **不可控制**：市场走势、北向资金、政策突发
- **核心原则**：赚钱亏钱是结果，不是决策质量的唯一标准
  > 「赚钱 ≠ 决策对」就是斯多葛主义在交易中的精确表达

### 孕期
- **可控制**：关心方式、就医安排、用药纪律
- **不可控制**：激素水平、检查结果
- **核心原则**：把力用在能改变的事情上，坦然面对不能改变的结果

### 第二增长曲线
- **可控制**：今天有没有推进 25 分钟
- **不可控制**：飞轮何时产生收入、市场何时验证
- **核心原则**：最小推进原则——1 行代码 > 0 行。1 段话 > 空白。
