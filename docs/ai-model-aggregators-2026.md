## 一句话结论

```
追求极致便宜       → DeepInfra 直连
国内模型高频调用    → 硅基流动（国产开源，官方价4-6折）
需要海外旗舰模型    → OpenRouter（透传价+5.5%，60+供应商容灾）
国内+海外全都要     → 七牛云 AI（双协议兼容，国内直连）
组合打法最省钱      → DeepInfra主用 + LiteLLM自托管 + OpenRouter备用
```

## 选型指南

| 你的实际场景 | 推荐 | 原因 |
|------|:---:|------|
| Claude Opus 4.8 重度使用 | 直接 **Claude Code** 订阅 | 聚合平台没 Claude API 便宜 |
| DeepSeek V4 大批量调用 | **DeepInfra** 直连 | 地板价 $1.30/$2.60 |
| 国产模型日常编码 | **硅基流动** | 官方价4-6折，推理加速66% |
| 需要 GPT-5.5 + Claude + 国产 | **OpenRouter** | 350+模型，一次对接 |
| 需要连接国内+海外模型 | **七牛云 AI** | 双协议兼容，国内直连 |
| 自建统一 API，零手续费 | **LiteLLM** 自托管 | MIT协议 |
| 入门体验，成本可预期 | **302.AI** | 订阅制 |

### 最佳组合

```
DeepInfra 直连（主力，最便宜）
    +
OpenRouter 备用（DeepInfra 挂了自动切换，5.5% 买容灾）
    +
LiteLLM 自托管（有服务器就上，零手续费统一API）
```
