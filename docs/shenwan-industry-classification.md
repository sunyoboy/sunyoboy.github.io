## 是什么

申万宏源研究所制定的一套 A 股行业分类标准。把全市场 5,000+ 只股票按主营业务归入不同的行业，每只股票只能归入一个行业。层级结构：

```
一级行业（31个）         → 最粗，看大方向
  二级行业（100+个）      → 中等粒度，看赛道
    三级行业（300+个）    → 最细，看细分产业链
```

**例**：

```
一级：食品饮料
  二级：饮料
    三级：白酒
    ├── 贵州茅台 → 归入"白酒"
    └── 五粮液   → 归入"白酒"
  二级：食品
    三级：乳制品 → 伊利股份

一级：有色金属
  二级：工业金属
    三级：铜 → 紫金矿业
    三级：铝 → 天山铝业
  二级：贵金属
    三级：黄金 → 山东黄金
```

> 贵州茅台和伊利股份同属申万一级"食品饮料"，全市场资金讨论"食品饮料板块今天涨了"时，说的就是申万口径下这个行业里所有股票的平均表现。

## 为什么申万分类是标准

A 股有三套主流分类：**申万**、**中信**、**证监会**。

| 维度 | 申万 | 中信 | 证监会 |
|------|:---:|:---:|:---:|
| 制定者 | 申万宏源研究所 | 中信证券 | 中国证监会 |
| 分类逻辑 | **产业链逻辑** | 偏投资风格 | **行政统计逻辑** |
| 更新频率 | 每年调整 | 不定期 | 基本不变 |
| 业内认可 | ✅ 研报统一基准 | 🟡 少数使用 | ❌ 仅监管用途 |

**例**：电池产业链（上游锂→中游电芯→下游整车）。申万把它拆成三个子行业分别归入"有色金属"、"电力设备"、"汽车"。证监会可能全混在"制造业"里——你无法从证监会分类中区分"做电池的"和"做螺丝的"。

> 几乎所有券商研报都用申万行业做比较基准。你每天看的"有色板块 +3.93%"，就是申万口径下的"有色金属"行业指数。

## 对你评估框架的五个益处

### 1. 你已经在用，只是没意识到

```
每天复盘里写的：
  "有色 -8.36%"    → 申万一级"有色金属"
  "券商 +5.86%"    → 申万一级"非银金融"下面的二级"证券"
  "医药 +2.23%"    → 申万一级"医药生物"
  "6月行业分化 17.9%" → 用申万行业口径统计

把这些散落在各处的板块分析结构化 → 
你的复盘从"今天这个涨那个跌"升级为
可比较、可积累、可回溯的数据。
```

### 2. 避免"看着分散、实际集中"

```
你以为的持仓：
  科大讯飞 + 天山铝业 + 紫金矿业 + 黄金ETF + 500ETF

申万分类下的真实暴露：
  科大讯飞  → 申万一级"计算机"（AI软件）
  天山铝业  → 申万一级"有色金属"（铝）
  紫金矿业  → 申万一级"有色金属"（铜金）
  黄金ETF   → 申万一级"有色金属"（贵金属）
              ↑ 三个全在同一个一级行业！
  500ETF    → 宽基，不归入任何行业

你的"分散持仓"在申万标准下暴露了：
有色金属集中度超 20%。
不是紫金和天山的问题——
是你买了三只在申万体系下被归为同一行业的东西。
```

> **这正是决策框架第七层（相关性校验）的基础数据**——没有申万分类，你无法精确量化板块集中度。

### 3. 低吸板块的量化筛选

今天上午你问"哪个板块估值低且企稳可以低吸"——

```
没有申万分类：靠感觉翻几个行业指数，随便挑

有申万分类：
  31 个一级行业 → 逐层过滤
  → 排除 PE >60% 分位（电子/AI/电力设备/通信/计算机）
  → 排除仍在单边下跌（有色/军工/化工）
  → 排除筑底未确认（房地产）
  → 留下候选 → 食品饮料（PE<10%分位+止跌反弹）

这个筛选是可复现的——因为行业分类是标准化的。
```

### 4. 轮动规律的发现

```
历史规律（必须用申万口径统计）：
  6月：电子 +6.26%、电力设备 +4.30%、通信 +4.30%
  7月：有色金属 +4.6%、国防军工 +3.3%、钢铁 +3.5%

这不是"我感觉7月有色会涨"——
是一个基于 10 年数据的可验证假设。
```

你自己复盘里写的"6月行业分化 17.9%"就是用申万行业做统计算出来的。继续用同一套口径，月度策略才能跨年对比。

### 5. 帮你发现"你以为安全其实不安全"的东西

```
紫金矿业：你以为它是"黄金股"
申万分类：一级行业"有色金属"，二级"铜"
→ 它在和铜矿/铝业共享同一个行业风险因子

黄金ETF：你以为它和紫金是"同一个方向的对冲"
申万分类：它跟踪的是黄金本身，不是有色金属股票
→ 你持有的铜（紫金）和铝（天山）高度相关
→ 黄金ETF 反而和它们相关性弱
→ 但你仓位里把黄金当对冲，真正冲的是铜+铝
```
