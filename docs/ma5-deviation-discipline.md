# 5 日线偏离度 · 持仓纪律

> **不是单一指标，是三维决策框架。**

只盯着偏离度一个数字毫无意义——+3% 的偏离在单边上涨中是加仓信号，在震荡市中却是减仓信号。区别就在于另外两个维度：**持续了多久**、**价格处于什么位置**。

## 三维框架

### ① 偏离方向（Direction）—— 「价格相对于均线在哪」

| 偏离度 | 命名 | 含义 |
|:------:|------|------|
| **> +5%** | 正偏·高危 | 价格大幅高于 MA5，均线引力极强，回调概率高 |
| **+2% ~ +5%** | 正偏·偏热 | 短期偏强，但尚未极端，注意是否加速或衰竭 |
| **-2% ~ +2%** | 贴合·均衡 | 价格围绕均线，均线无牵引力，方向待选 |
| **-5% ~ -2%** | 负偏·偏冷 | 短期偏弱，超跌反弹概率在累积 |
| **< -5%** | 负偏·冰点 | 价格大幅低于 MA5，技术性反弹条件成熟，但趋势破坏风险也最高 |

> **关键认知**：正偏离越大，均线「向下拉」的引力越强；负偏离越大，均线「向上吸」的磁铁效应越强。但引力不等于反转——需要结合持续时间判断是「刚开始偏离」还是「偏离到极限了」。

### ② 持续时间（Duration）—— 「偏离了多久，在加速还是减速」

这个维度由两个指标组成：

| 指标 | 含义 | 数据来源 |
|------|------|------|
| **连续偏向天数** | 收盘价连续在 MA5 上方/下方的交易日数 | `ma5-deviation.py` 输出 |
| **偏离趋势** | 最近 3 日 |偏离度| 的增减：扩大 = 加速偏离，收窄 = 动能衰竭 | 同上 |

两个指标交叉形成 4 种状态：

```
                    偏离扩大（加速）
                        │
          🔴 趋势加速     │    🟢 主升浪
          连跌 + 扩大     │    连涨 + 扩大
          → 等反弹减仓    │    → 持有不动
    ──────────────────────┼──────────────────────
      负偏 ←              │              → 正偏
    ──────────────────────┼──────────────────────
          🟡 超跌修复     │    🟡 动能衰竭
          连跌 + 收窄     │    连涨 + 收窄
          → 等反弹到均线   │    → 考虑止盈减仓
                        │
                    偏离收窄（减速）
```

**持续时间维度的核心规则**：

| 状态 | 操作含义 |
|------|------|
| **负偏 + 扩大 + 连跌 ≥3 日** | 下跌趋势在加速，不要在此处接飞刀。等偏离开始收窄再考虑任何操作 |
| **负偏 + 收窄 + 连跌 ≥5 日** | 超跌后偏离在修复，反弹至 MA5 的概率最高。这是减仓的最佳窗口期，不是抄底 |
| **正偏 + 扩大 + 连涨 ≥3 日** | 主升浪特征，均线托着价格往上走。此时减仓是逆势——等偏离收窄再动 |
| **正偏 + 收窄 + 连涨 ≥5 日** | 动能见顶信号。价格虽然还在均线上方，但已无力继续拉开距离。减仓窗口 |

### ③ 价格位置（Position）—— 「价格在整体格局中的什么位置」

这是三维框架中最被忽视、但最重要的维度。偏离 3% 在底部区域和顶部区域的含义截然不同。

| 位置指标 | 判断标准 | 含义 |
|------|------|------|
| **MA5/MA20 交叉** | MA5 > MA20 = 金叉（短期多头），MA5 < MA20 = 死叉（短期空头） | 短期趋势的确认信号 |
| **MA20/MA60 关系** | MA20 > MA60 = 中期多头，MA20 < MA60 = 中期空头 | 决定了大方向——在中期空头中做多是逆水行舟 |
| **距 20 日低点** | 现价 vs 20 日最低价的距离 | < 3% = 靠近近期支撑，破位风险高；> 10% = 远离支撑，回调需求大 |
| **距 20 日高点** | 现价 vs 20 日最高价的距离 | > -3% = 接近前高，突破或双顶；< -10% = 深度回调中 |
| **量能比** | 5 日均量 ÷ 20 日均量 | > 1.5 = 放量（趋势确认），< 0.6 = 缩量（方向不明） |

**价格位置的「叠加效应」**——单看偏离度没意义，必须叠加位置：

> **同样的 -5% 偏离，三种完全不同的决策：**
>
> | 价格位置 | 决策 |
> |------|------|
> | 距 20 日低 +1%，MA5 刚死叉 MA20，中期空头 | 🔴 刚破位，加速下跌概率高，**立即减仓** |
> | 距 20 日低 +15%，MA5 金叉 MA20，中期多头 | 🟢 牛市回调，均线支撑有效，**持有或加仓** |
> | 距 20 日低 +8%，MA5 走平，量能萎缩 | 🟡 震荡格局，方向不明，**不动** |

## 决策矩阵

把三维度合并，得到以下决策矩阵：

### 负偏离区（价格低于 MA5）

| 偏离度 | 持续时间 | 价格位置 | 操作 | 优先级 |
|:------:|------|------|------|:--:|
| **-2%~-5%** | 连跌 1-2 日，偏离扩大 | 死叉 + 中期空头 | 反弹至 MA5 减 1/4 | 🟡 |
| **-2%~-5%** | 连跌 1-2 日，偏离收窄 | 金叉 + 中期多头 | 牛市回调，持有 | 🟢 |
| **-2%~-5%** | 连跌 ≥5 日，偏离收窄 | 任何 | 超跌修复中，反弹即减仓窗口 | 🟡 |
| **-5%~-8%** | 连跌 ≥3 日，偏离扩大 | 死叉 + 中期空头 | **反弹至 MA5 减 1/2** | 🔴 |
| **-5%~-8%** | 连跌 ≥3 日，偏离收窄 | 距 20 低 < 3% | 等反弹，反弹到 MA5 减半 | 🟡 |
| **-5%~-8%** | 连跌 1 日（首日破位） | 死叉 + 中期空头 | **次日开盘减仓，不等** | 🔴 |
| **< -8%** | 任何 | 任何 | **清仓或减至观察仓** | 🔴🔴 |

### 正偏离区（价格高于 MA5）

| 偏离度 | 持续时间 | 价格位置 | 操作 | 优先级 |
|:------:|------|------|------|:--:|
| **+2%~+5%** | 连涨 ≥3 日，偏离扩大 | 金叉 + 中期多头 | 主升浪，持有 | 🟢 |
| **+2%~+5%** | 连涨 ≥5 日，偏离收窄 | 接近 20 日高 | 动能衰竭，减 1/3 | 🟡 |
| **+2%~+5%** | 连涨 1-2 日 | 死叉 + 中期空头 | 反弹行情，减仓机会 | 🟡 |
| **> +5%** | 偏离收窄 | 金叉 + 中期多头 | 减 1/4，锁定部分利润 | 🟡 |
| **> +5%** | 偏离扩大 | 距 20 低 > 15% | 加速赶顶，分批减仓 | 🔴 |

## 减仓细则

### 第一步：确认「不是主升浪」

正偏离时，先排除主升浪，否则容易卖飞：

| 排除条件 | 说明 |
|------|------|
| MA5 > MA20 > MA60 且三线发散向上 | 标准多头排列，回调即加仓，不减 |
| 偏离扩大 + 量能 > 1.5x | 放量推升，趋势有效，不减 |
| 连涨 < 3 日 | 行情刚开始，观察 |

> **如果以上三条全不满足，进入减仓评估。**

### 第二步：选择减仓时机

**正偏离减仓（止盈 / 减亏）**：

```
时机 = 偏离收窄的当日或次日
     ≠ 偏离还在扩大的时候

原因：偏离扩大说明趋势还在惯性运行，
     提前减仓 = 主动放弃最后一段利润。
     等偏离开始收窄再动，大概率卖在阶段高点附近。
```

**负偏离减仓（止损）**：

```
时机 = 反弹至 MA5 附近（偏离回正或收窄到 ±1% 以内）
     ≠ 大跌当天（除非触发清仓条件）

原因：超跌后均线有磁铁效应，价格大概率会技术性反弹。
     反弹到 MA5 附近是离场的最佳时机——既不是恐慌低点，
     也不是"再等等看"的幻想区间。
```

### 第三步：确定减仓比例

| 场景 | 减仓比例 | 条件 |
|------|:---:|------|
| 正偏 + 动能衰竭 | **1/4** | 偏离收窄但 MA5 仍金叉 MA20 |
| 正偏 + 接近前高 | **1/3** | 偏离 > +3% 且距 20 日高 < 3% |
| 正偏极端 + 死叉形成 | **1/2** | 偏离 > +5% 且 MA5 即将下穿 MA20 |
| 负偏 + 破位确认 | **1/2** | 偏离 < -5% 且连跌 ≥3 日且死叉 |
| 负偏 + 反弹无力 | **1/2** | 反弹触及 MA5 后回落，成交量未放大 |
| 负偏 + 首次大跌破位 | **1/2（次日执行）** | 单日偏离 > -5% 且跌破关键支撑 |

## 清仓细则

清仓 = 不设反弹条件，不等。满足以下任何一条直接执行：

### 清仓条件（满足任一即触发）

| # | 条件 | 指标组合 | 逻辑 |
|:--:|------|------|------|
| 1 | **三重空头共振** | 死叉 + 中期空头 + 偏离 < -5% + 连跌 ≥5 日 | 短中长期均确认下行，持仓逻辑已不成立 |
| 2 | **极端超跌加速** | 偏离 < -8% + 偏离扩大 + 放量 | 恐慌盘涌出，等反弹的代价可能超过直接清仓 |
| 3 | **反弹确认失败** | 反弹触及 MA5 后再度回落 + 破 20 日低点 | 均线从支撑变阻力，下跌进入第二波 |
| 4 | **逻辑破坏** | 基本面/行业逻辑改变 + 偏离 < -5% | 技术面止损应该让位于逻辑止损——如果当初买入的理由已经不成立，任何价格都是卖点 |
| 5 | **被迫清仓** | 触及预设止损线（如天山 11.5、卫星 1.35） | 纪律就是纪律，不找理由 |

### 清仓后纪律

```
清仓后 5 个交易日内不买回同一标的。
防止「卖完就见底」的焦虑驱动报复性买回。
```

## 常见错误

### ❌ 错误 1：「偏离大 = 要反转了」

| 错误 | 纠正 |
|------|------|
| 偏离 -6% 就觉得到底了，开始抄底 | 偏离大只说明「价格离均线远」，不说明「均线不会继续往下走」。均线本身是移动的——如果明天继续大跌，MA5 会跟着下移，偏离度可能反而收窄。**偏离大 ≠ 到底，偏离收窄才是到底的领先信号。** |

### ❌ 错误 2：「偏离小 = 安全」

| 错误 | 纠正 |
|------|------|
| 贴均线就不操作 | 贴均线 + 死叉 + 中期空头 = 均线即将成为向下阻力，不是安全区。价格贴着 MA5 在下跌中 = 「均线压着价格往下走」，是最危险的形态之一。 |

### ❌ 错误 3：「连跌 N 天了该反弹了」

| 错误 | 纠正 |
|------|------|
| 连续下跌 5 天就买 | 连续下跌天数没有「应该反弹」的阈值。只有在偏离开始收窄时，反弹概率才真正上升。连跌 5 天 + 偏离还在扩大 = 下跌加速，不是抄底信号。 |

### ❌ 错误 4：「涨多了就卖」

| 错误 | 纠正 |
|------|------|
| 正偏 > 3% 立刻卖 | 正偏 + 扩大 + 中期多头 = 主升浪中，减仓会踏空。等偏离开始收窄（动能衰竭）再卖。 |

### ❌ 错误 5：「用偏离度单独做决策」

| 错误 | 纠正 |
|------|------|
| 只看到偏离 -4%，不看不看不看不看持续时间、不看价格位置 | **永远三维一起看。** 偏离告诉你「偏离了多少」，持续时间告诉你「偏离在加速还是减速」，价格位置告诉你「偏离发生的背景是熊是牛」。缺一个维度，判断就不成立。 |

## 工具

运行 `python3 scripts/ma5-deviation.py` 输出全景诊断表，包含三维度全部指标。

| 输出段落 | 内容 |
|------|------|
| **表一：三维指标全景** | 偏离度、方向信号、连续偏向天数、偏离趋势、MA5/MA20 交叉、中期背景、距 20 低、量能 |
| **表二：均线与关键价位** | MA5/MA20/MA60 具体数值、偏离 20/偏离 60、20 日高低点、关键支撑位 |
| **表三：综合操作建议** | 按紧急程度排列，分 🔴 紧急操作和 📋 常规操作 |

---

> **最后一条规则：当三维指标和直觉冲突时，信指标不信直觉。**
>
> 直觉会说「跌了这么多该反弹了」，指标会说「偏离还在扩大，还没跌完」。
> 直觉会说「涨了这么多该卖了」，指标会说「主升浪中，偏离收窄前不动」。
> 纪律的价值就在于——在你情绪最不稳定的时刻，给你一个不需要情绪的决策框架。
