# 🦁 荀子 · 天人相分 + 化性起伪 + 礼法并重

> 先秦儒家的「异端」——孟子说性善，荀子说性恶。两个人吵了 2300 年，其实说的是同一件事的两面。
>
> **孟子向内求（扩充善端），荀子向外建（制度改造）。一个都不能少。**

---

## 一、天人相分：天是规律，不是裁判

**出处**：《荀子·天论》

### 核心论点

「天行有常，不为尧存，不为桀亡。」——天道运行有固定规律，不会因为尧是圣君就多给好天气，也不会因为桀是暴君就用天灾惩罚他。

「故明于天人之分，则可谓至人矣。」——能分清天的职分和人的职分，才是最高明的人。

### 与「天人合一」对立什么

| | 孔孟「天人合一」 | 荀子「天人相分」 |
|------|------|------|
| **天是什么** | 有道德意志的天·天会赏善罚恶 | 纯粹的自然规律·天不干预人事 |
| **人怎么做** | 修德配天·感应天意 | 认识规律·利用规律 |
| **灾异解释** | 天降灾异警示人君 | 纯粹自然现象·和人没关系 |
| **一句话** | 天在看 | 天不管 |

> **这是中国思想史上第一次「去魅化」——把天从道德裁判的位置上拉下来，变成纯粹的客观规律。**

### 处世落地

- **不等天意等规律。** 不指望"天助我也""吉人自有天相"——研究规律比祈祷管用。
- **成事靠人不靠天。** 荀子的逻辑：天提供土壤，人决定收成。同样的市场（天），不同的人（交易系统）结果完全不一样。

### 与投资体系的映射

| 荀子 | 投资实战 |
|------|------|
| 天行有常 | 市场有规律·不以你的仓位为转移 |
| 不为尧存不为桀亡 | 市场不关心你努不努力、委不委屈 |
| 明于天人之分 | 分清什么是你控制的（仓位/止损），什么是你控制不了的（涨跌） |
| 制天命而用之 | 识别规律 → 顺应规律 → 利用规律 |

> 这和 `docs/stoicism.md` 的「控制二分法」几乎同构——荀子比爱比克泰德早 200 年说了同样的话。

## 二、化性起伪：人性本恶，靠制度改造

**出处**：《荀子·性恶》

### 核心论点

「人之性恶，其善者伪也。」——人的本性是恶的（自私、贪欲、好逸恶劳），表现出来的一切善行都是人为改造的结果。

**「伪」≠ 虚伪。** 荀子的「伪」是「人为」（亻 + 为），指后天的人为改造——教育、礼仪、法律、制度。

### 逻辑链

```
人性本恶（饿了想吃、累了想歇、看到利益想抢）
     ↓
如果放任自流 → 争夺、残贼、淫乱 → 社会崩溃
     ↓
所以需要「化性起伪」
     ↓
┌────────────────────┬────────────────────┐
│  礼（内部约束）       │  法（外部约束）        │
│  教化、习惯、道德     │  制度、规章、惩罚      │
│  "你不想做但应该做"   │  "你不敢不做"         │
└────────────────────┴────────────────────┘
     ↓
善行 = 本性被改造后的结果
```

### 孟子 vs 荀子：不是对错问题，是角度问题

| | 孟子·性善论 | 荀子·性恶论 |
|------|------|------|
| **人性的初始** | 善端（四端之心，人人都有） | 恶（好利、疾恶、好声色） |
| **恶从哪来** | 环境遮蔽了善端 | 天生就是这样 |
| **善从哪来** | 扩充善端·向内求 | 化性起伪·向外建 |
| **怎么做** | 存心养性·找回本心 | 隆礼重法·制度改造 |
| **信任什么** | 人的内在良知 | 制度的外在约束 |
| **适用场景** | 做人——你有良知，向内觉察 | 做事——你会犯错，靠制度兜底 |

> **2300 年后的结论：都对，但各管一半。** 做人靠孟子（信条+内省），做事靠荀子（纪律+SOP+止损线）。缺孟子容易冷酷，缺荀子容易散漫。

## 三、礼定伦，法至霸：双轨制

**出处**：《荀子》礼论篇 + 王制篇

### 礼的职能：内部秩序

礼 = 一套完整的伦理规范系统，涵盖仪式、等级、行为准则。作用是让人**自觉**遵守秩序。

> 「礼者，法之大分，类之纲纪也。」——礼是法律的根本依据，是所有规范的纲领。

### 法的职能：外部约束

荀子是儒家中最强调「法」的人物（他的学生韩非和李斯后来成为法家代表人物）。单纯靠礼的教化不够——总有人不自觉——所以需要法的强制。

> 「法者，治之端也。」——法是治理的起点。

### 双轨逻辑

```
礼 → 让你「想做好人」（羞耻感+荣誉感）
法 → 让你「不敢做坏人」（惩罚+制度约束）

缺礼 → 社会冷漠·人人自危但没人主动行善
缺法 → 道德说教·对不自觉的人完全无效
```

### 与投资体系的映射

| 荀子 | 你的交易体系 |
|------|------|
| **礼**（内部） | CLAUDE.md 信条「三个不相信」「岿然不动」、修身内省 |
| **法**（外部） | 止损线、条件单、SOP、MA5偏离度纪律、CLAUDE.md 强制执行规则 |
| **礼法并重** | 有好心态（礼）+ 但也要设止损（法）·缺一不可 |

> **最好的例子：7/21 银行ETF 买入。** 如果你有"法"——买入后立刻挂条件单——那光靠"礼"（"我觉得银行到底了"）做决策也没关系，因为法会保护你。**礼让你敢出手，法让你不失控。**

## 四、与孔孟老庄韩非的关系

```
儒家：
  孔子 → 仁 + 礼（奠基）
    ├── 孟子 → 性善·仁政·向内求
    └── 荀子 → 性恶·礼法·向外建
              ├── 韩非 → 法家（只留法，不要礼）
              └── 李斯 → 秦制（法变成暴政工具）

道家：
  老子 → 反者道之动·无为
  庄子 → 逍遥·齐物

荀子站在儒家和法家之间：
  继承了孔子的「礼」→ 礼定伦
  批判了孟子的「性善」→ 化性起伪
  启发了韩非的「法」→ 法至霸
  但反对法家的「纯法无礼」→ 礼法必须并重
```
