## 一、已安装的本地技能

| Skill | 类型 | 功能 |
|------|------|------|
| `/pywencaistock` | Claude Code Skill | 通过 pywencai 库调用问财 API，返回 DataFrame |

**基本用法：**

```python
# 查涨停股
result = skill('pywencai-stock').search(query='A股涨停')

# 查主力资金流向
result = skill('pywencai-stock').search(query='主力净流入最多的股票')

# 查板块
result = skill('pywencai-stock').search(query='芯片概念股')
```

## 二、⭐⭐⭐ 核心六技能（替代每日脚本）

> 这六个技能覆盖了 `fetch-market-data.py` + `shenwan-monitor.py` + `ma5-deviation.py` 的核心功能。

### 1. 行情数据查询

| 属性 | 内容 |
|------|------|
| 出品 | 同花顺官方 |
| 功能 | 个股/ETF/指数实时价格、涨跌幅、成交量、主力资金流向、大小单、技术指标 |
| 替代 | `fetch-market-data.py` 的标的行情抓取 |

**典型查询：**
```
中证500ETF今天收盘价和涨跌幅
宁德时代主力资金净流入
科创50ETF成交量和技术指标
```

### 2. 指数数据查询

| 属性 | 内容 |
|------|------|
| 出品 | 同花顺官方 |
| 功能 | 上证指数、沪深300、创业板指、恒生指数、纳斯达克等指数行情 |
| 替代 | `fetch-market-data.py` 的 7 大指数抓取 |

**典型查询：**
```
今天上证指数、深证成指、创业板指、科创50的涨跌幅
近5日沪深300走势
纳斯达克指数最新点位
```

### 3. 行业数据查询

| 属性 | 内容 |
|------|------|
| 出品 | 同花顺官方 |
| 功能 | 行业估值、财务、盈利、行情、板块排名 |
| 替代 | `shenwan-monitor.py` 的 31 行业全景 |

**典型查询：**
```
申万31个行业今天涨跌幅排名
医药生物行业估值分位
银行板块资金流入情况
```

### 4. 宏观数据查询

| 属性 | 内容 |
|------|------|
| 出品 | 同花顺官方 |
| 功能 | GDP、CPI、PPI、利率、汇率、社融、M2、PMI、工业增加值、消费、投资、进出口 |
| 用途 | 月度市场概况数据源 |

**典型查询：**
```
最新M2同比增速
6月CPI和PPI数据
最新LPR利率
```

### 5. 财务数据查询

| 属性 | 内容 |
|------|------|
| 出品 | 同花顺官方 |
| 功能 | 营收、净利润、ROE、负债率、现金流、毛利率、净利率 |
| 用途 | 中报季业绩暴雷筛查·38候选池基本面跟踪 |

**典型查询：**
```
科大讯飞最新季度营收和净利润
天山铝业ROE和负债率
沪深300成分股中报净利润增速排名
```

### 6. 问财选A股

| 属性 | 内容 |
|------|------|
| 出品 | 同花顺官方 |
| 功能 | 行情/技术形态/财务/行业概念多条件组合选股 |
| 用途 | 38候选池动态筛选·条件变化时重新评估 |

**典型查询：**
```
PE<20且ROE>15%且市值>100亿的A股
金叉且放量的沪深300成分股
医药行业市盈率低于历史30%分位的股票
```

## 六、安装与配置

### 已完成的配置

```bash
# 环境变量已写入 ~/.zshrc
IWENCAI_API_KEY=sk-proj-00-...
IWENCAI_BASE_URL=https://openapi.iwencai.com
```

### 本地 Skill 使用

在 Claude Code 中直接使用 `/pywencaistock`，或通过 Bash 调用：

```bash
python3 scripts/iwencai_query.py "今天A股涨停的股票"
```

### SkillHub 在线技能

在 [SkillHub](https://www.iwencai.com/skillhub) 页面点击技能 → 复制 Prompt → 粘贴到 AI 助手中使用。
