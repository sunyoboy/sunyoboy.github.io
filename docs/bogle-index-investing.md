## 一、核心理念

### 为什么选股是徒劳的

博格统计了 1970-2005 年美国所有主动基金：**80% 跑输标普 500。** 不是基金经理不聪明——是扣掉管理费（~1.5%）和交易成本后，超额收益归零。

> 「在投资中，你得到的不是你所付出的——你得到的是你所不付出的。**成本是确定的，收益是不确定的。**」

### 博格公式

```
市场总回报 = 企业盈利增长 + 股息收益 + 估值变化
投资者实际到手 = 市场总回报 - 费用

费用的影响：
  年化 7%，扣 2% 管理费，30 年后：
    不扣费：100 万 → 761 万
    扣 2%： 100 万 → 432 万
    费用吞噬了 43% 的收益
```

> ETF 管理费 0.15%-0.5%。**同样的指数收益，省下的费用就是你的 alpha。**
