## 二、债券ETF如何赚钱？

### 收益 = 票息收入 + 资本利得

```
债券ETF总收益
    ├── 票息收入（稳定持续）
    │   国债每年付息约2.5-3.0%
    │   ETF把利息累积到净值里
    │   持有一年 ≈ 净值自然增长2.5-3.0%
    │
    └── 资本利得（利率下行时爆发）
        利率↓ → 旧债券（锁定高利息）更值钱 → 价格涨
        涨价幅度 ≈ 利率降幅 × 久期
        30年国债久期≈18年
        利率降0.3% → 价格涨约5.4%
```

### 具体算账

```
场景：买入100,000元30年国债ETF，持有一年

票息收入：  100,000 × 2.8%   = +2,800
资本利得：  100,000 × 5.4%   = +5,400 （假设利率降0.3%）
──────────────────────────────────────
年化收益：                    ≈ +8,200（约8.2%）

对比逆回购（当前1.36%）：     +1,360
差额：                        +6,840 元
```

## 四、关键概念

### 久期

```
久期 = 债券价格对利率变化的敏感度

30年国债久期 ≈ 18年
含义：利率每降0.1%，价格涨约1.8%

对比：
  短融ETF久期 ≈ 0.5年 → 利率降0.1%，只涨0.05%
  30年国债久期 ≈ 18年 → 利率降0.1%，涨1.8%

→ 看好利率下行 = 买长久期（511090）
→ 不确定方向   = 买短久期（511360）
```

### 收益率 vs 价格（反向关系）

```
收益率（利率）  债券价格
    ↓            ↑
    ↑            ↓

原因：新发债券利率降了→旧债券的高利息更值钱→旧债券涨价
```

## 七、操作纪律

```
✅ 买入后不用每天看 — 债券波动极小，盯它浪费精力
✅ 每月看一眼利率方向 — 确认降息周期未逆转
✅ 拿一年以上 — 债券收益靠持有不靠择时
✅ 用闲置资金买 — 不要占用股票仓位的钱
❌ 不做T — 债券没弹性，手续费吃掉收益
❌ 不频繁买卖 — 每笔交易成本约0.01%
❌ 不在加息周期买 — 等降息信号出现再动手
```
