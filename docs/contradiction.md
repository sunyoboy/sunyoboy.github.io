## 一、核心观点

### 1. 矛盾的普遍性

**矛盾存在于一切事物的发展过程中，贯穿始终。**

没有矛盾就没有世界。任何事物、任何阶段都包含矛盾——差异就是矛盾。不要幻想存在一个"没有问题"的理想状态。

### 2. 矛盾的特殊性

**每种矛盾都有自己的特点，必须具体问题具体分析。**

不同事物的矛盾不同，同一事物在不同阶段的矛盾也不同。不能用一种公式套所有问题。**具体情况具体分析**是马克思主义活的灵魂。

### 3. 主要矛盾与次要矛盾

**复杂事物中，必有一种矛盾是主要的，起领导和决定作用。**

- 抓住了主要矛盾，其他问题就迎刃而解
- 主要矛盾变了，事物的性质就变了
- 不能平均用力，"眉毛胡子一把抓"

### 4. 矛盾的主要方面与次要方面

**同一矛盾内部，必有一方是主要的，另一方是次要的。**

矛盾双方不是均衡的，主要方面决定事物的性质。当力量对比发生根本变化时，事物性质随之改变。

### 5. 矛盾的同一性与斗争性

| 属性 | 含义 | 关系 |
|------|------|------|
| **同一性** | 矛盾双方互相依存、互相转化 | 有条件的、暂时的、相对的 |
| **斗争性** | 矛盾双方互相排斥、互相对立 | 无条件的、绝对的、贯穿始终 |

同一性中有斗争，斗争性中有同一。**有条件的相对同一性和无条件的绝对斗争性相结合，构成事物的矛盾运动。**

### 6. 对抗性矛盾与非对抗性矛盾

- **对抗性矛盾**：必须通过激烈冲突才能解决（如敌我矛盾）
- **非对抗性矛盾**：可以通过和平方式、渐进调整解决（如人民内部矛盾）
- 不同性质的矛盾，用不同方法解决

## 二、如何使用矛盾论处理现实问题

### 第一步：承认矛盾，不回避

- 遇到问题先告诉自己：**矛盾是正常的，没有矛盾才不正常**
- 不要试图"消灭"所有矛盾——矛盾是推动事物发展的动力
- 把问题当成分析对象，而不是情绪负担

### 第二步：找出主要矛盾

问自己三个问题：

1. 当前面临的所有问题中，**哪个问题是其他问题的根源**？
2. 解决了这个问题，**哪些次要问题会自然消失**？
3. 如果只能做一件事，**应该做什么**？

> 关键方法：把所有问题列出来，用网状图找出因果关系。箭头指向最多的那个，大概率是主要矛盾。

### 第三步：分析矛盾的主要方面

对于已确定的主要矛盾，进一步分析：

- 矛盾双方谁占主导？谁是被动方？
- 力量对比是在加强还是在削弱？
- **转化的临界点在哪**？

### 第四步：推动矛盾转化

- 创造条件让主要矛盾向有利方向转化
- 不是消灭矛盾，而是**改变矛盾双方的力量对比**
- 当主要矛盾解决后，及时识别新的主要矛盾

### 第五步：注意矛盾的特殊性

- 同一类问题，不同阶段的处理方式不同
- 不要机械套用过去的成功经验
- **具体问题具体分析**，不断修正判断

## 三、应用举例

### 投资中的应用

| 步骤 | 示例 |
|------|------|
| 承认矛盾 | 市场永远存在多空分歧，涨跌是常态 |
| 找主要矛盾 | 科技股暴跌：是流动性问题（暂时性）还是产业逻辑破坏（根本性）？ |
| 分析主要方面 | 当前谁主导？空方力量是否在衰竭？ |
| 推动转化 | 等恐慌盘出清、缩量企稳时入场，而非在暴跌中硬扛 |
| 注意特殊性 | 这次的下跌和上次的下跌原因不同，不能简单类比 |

### 工作中的应用

| 步骤 | 示例 |
|------|------|
| 承认矛盾 | 项目延期是多个因素共同作用，不是一个人的问题 |
| 找主要矛盾 | 是需求不清？资源不足？还是技术瓶颈？ |
| 分析主要方面 | 瓶颈在哪一方？能否短期内改变力量对比？ |
| 推动转化 | 砍掉次要需求，集中资源突破关键瓶颈 |
| 注意特殊性 | 不同项目的瓶颈不同，不能每次都"加人" |

---

## 四、一句话总结

> **承认矛盾是普遍的 → 找到起决定作用的主要矛盾 → 分析矛盾的主要方面 → 创造条件推动矛盾向有利方向转化 → 具体问题具体分析，不搞一刀切。**

**矛盾本身不是问题，用错误的方法处理矛盾，才是真正的问题。**
