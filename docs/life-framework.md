# 人生完整维度框架

> 做人 × 做事 × 修身 = 底层三核心 · 延伸八大板块 · 极简总框架四象限

---

## 一、基础三核心

### 1. 做人（与他人的关系）
心性·品德·分寸·人情·底线·识人·情绪·人际关系·取舍·格局

### 2. 做事（与世界的关系）
逻辑·执行力·复盘·取舍·效率·风险·责任·目标规划·解决问题·分寸博弈

### 3. 修身（与自己的关系）— 底层根基
身体·作息·欲望管理·心态·认知·自省·独处·定力·内耗化解

---

## 二、延伸八大板块

### 1. 认知与思维（决定人和事的上限）
底层逻辑·独立思考·破除偏见·信息筛选·元认知·长短思维·概率思维·辩证看待得失·避免盲从·区分表象与本质·终身学习·接受自身局限

### 2. 欲望与取舍（一生都在做选择）
得失观·知足·克制贪欲·断舍离（物质/社交/执念）·平衡理想与现实·自由与责任·付出与回报·顾家与事业

### 3. 关系圈层（所有社交分类）
- **亲情**：父母·伴侣·子女相处之道
- **友情**：识人·边界·淡交长久·远离消耗型人
- **职场人际**：上下级·同事·合作·利益往来分寸
- **泛社交**：酒局·人脉·无用社交取舍·人情往来规则

### 4. 财富与生存（现实落地）
赚钱逻辑·储蓄·抗风险·消费观·资产思维·打工·副业·创业·合作分钱·避坑·不赌不投机·贫富心态：有钱不张扬·低谷不自卑

### 5. 苦难与心态（应对起伏）
挫折·低谷·失去·遗憾·委屈·孤独·失败如何自渡·接纳无常·降低期待·减少内耗·自愈能力·看淡得失

### 6. 选择与命运
关键节点选择：学业·城市·工作·婚姻·交友·何为命·何为运·努力和时机的关系·不回头后悔·为自己每一个决定承担后果

### 7. 时间与自我规划
时间管理·拒绝拖延·优先级排序·短期谋生·长期成长·年轻时沉淀·中年稳守·晚年自在·区分消耗时间/增值时间/休息时间

### 8. 生死与终极通透（顶层总结）
接纳衰老·离别·失去·放下执念·何为活着的意义：不是名利·是体验·心安·无愧于心·看淡评价：他人声名褒贬不必放在心上

## 六、完整七维框架

> 做人 + 做事 = "立德 + 立功"。但《左传》还有"立言"，古希腊还有"美"，庄子还有"超越"。

### 七维总览

```
         做事（立功）         立言（创造）
               \               /
                \             /
    认知 ←───  修身（底层根基）  ───→ 审美
                /             \
               /               \
         做人（立德）         超越（终极关怀）
```

| 维度 | 核心问题 | 你已有的对应 |
|------|------|------|
| 🧘 **修身** | 我怎么对自己？ | 断舍离·第欧根尼·斯多葛 |
| 🤝 **做人** | 我怎么对别人？ | 论语·围炉夜话·孕期关怀清单 |
| 🔨 **做事** | 我怎么对世界？ | 交易纪律·复盘 SOP·决策矩阵 |
| ✍️ **立言** | 我留下了什么？ | VitePress 知识库·每日复盘 |
| 🎨 **审美** | 什么让我纯粹感到活着？ | [审美清单 + 五项纪律](aesthetics) ✅ 已补 |
| 🌌 **超越** | 这一切最终为了什么？ | 易经·道德经 |
| 🧠 **认知** | 我如何理解这一切？ | 枪炮病菌·加文贝克·矛盾论 |

### 做人做事之外：三个被忽略的维度

**立言**：不是在做事——因为你不靠它赚钱。是你创造过的东西在你消失后还在。你的 VitePress 站点已经是立言。

**审美**：不是奢侈品——是确认自己还活着的能力。第欧根尼晒太阳不是做事，亚历山大的征服是做事，晒太阳是活着。

**超越**：所有的"做人做事"都在回答"怎么活"——只有超越在回答"为什么活"。你不需要有答案，但需要一个框架来容纳这个问题。
