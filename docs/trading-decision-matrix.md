## 买入前三问（缺一个 = 不买）

```
① PE < 20？（ETF 看 PE 分位 < 20%）        □
② 近 10 日主力净流入？                       □
③ 公司没发炒作风险/减持公告？                  □
④ 入场价 ___ + 止损价 ___ + 仓位 ___ = 全填了？ □
```

## 不做清单（任何时候）

```
❌ 月初/月末/大节假日前 → 不开新仓
❌ 个股 PE > 100 或利润崩塌 → 不碰
❌ 大股东减持中 → 不碰
❌ 金叉已破的品种 → 不加仓
❌ 盘中临时决定买卖 → 操作全部盘前挂单
❌ 周五下午追高 → 周末不确定性
❌ 连跌中补仓 → 去杠杆行情不等企稳不加仓
```

## 仓位规则

## final capitulation 三信号

```
① 银行补跌（最后一个未补跌板块）-2%+
```
