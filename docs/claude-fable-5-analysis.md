## Fable 5 与 Mythos 5 的关系

两者是**同一底层权重**。Fable 5 面向公众，敏感领域自动降级到 Opus 4.8；Mythos 5 是满血版，仅对通过审核的机构开放。

## 代码能力

| 基准 | Fable 5 | Opus 4.8 | GPT-5.5 | 优势幅度 |
|------|:---:|:---:|:---:|:---:|
| **SWE-bench Verified** | **95.0%** | 88.6% | — | |
| **SWE-bench Pro** | **80.3%** | 69.2% | 58.6% | 比 GPT-5.5 高 **22pp** |
| **FrontierCode Diamond** | **29.3%** | 13.4% | 5.7% | 比 GPT-5.5 高 **5倍** |
| **Terminal-Bench 2.1** | **88.0%** | — | 83.4% | |

> FrontierCode 上 5 倍差距意味着**最难的代码任务上，其他模型基本做不了**。

### 实战验证

| 任务 | 表现 | 对照 |
|------|------|------|
| **Stripe 代码迁移** | 5000 万行 Ruby，**1 天完成** | 人工需 2 个月+ |
| **自主科研** | 跨 138 物种单细胞基因组学，1 周 | 训练出的模型击败 Science 论文 |
| **宝可梦通关** | 仅凭**屏幕截图**，无需辅助框架 | 之前所有模型都需要复杂 scaffolding |

## 五个核心优势

### 1. 自主长程推理

```
跨会话记忆: 失败→调查→验证→提炼为规则→在后续任务中调用
规则正确调用率: 73%

不再是"更聪明的模型"，是"能记住自己犯过的错并自觉改正的模型"。
```

### 2. 代码断层领先

FrontierCode Diamond 29.3% vs GPT-5.5 的 5.7%——不是 10% 的领先，是 5 倍。

### 3. 视觉理解新天花板

从截图重建 Web 应用源码，从图表提取精确数据，仅凭屏幕截图完成复杂游戏通关。

### 4. 知识工作广度

GDPval-AA Elo 1932，比 GPT-5.5 高 163 分。编写从零构建 $100M 公司的全套计划得分 38.6%。

### 5. 安全护栏产品化

不靠"拒答"，靠**自动路由**——敏感请求静默降级到 Opus 4.8。95%+ 的会话不触发降级。

## 禁令的连锁破坏

### 1. 差距被锁定

```
Fable 5 发布时: SWE-bench Pro 领先 GPT-5.5 22pp
被禁后:       最强可用模型 = Opus 4.8（落后 Fable 5 11pp）
              竞争对手（OpenAI/DeepSeek/智谱）持续迭代
              差距正在被追平
```

**讽刺**: 禁令名义是"防止中国获得先进 AI"。但中国企业根本不需要 Fable 5——他们有 DeepSeek V4 和 GLM-5.2，持续迭代中。禁令的唯一效果是**阻止美国人自己使用**。

### 2. 成本无法摊薄

```
正常路径:
  发布 → 大量用户 → 推理优化 → 硬件利用率↑ → 成本↓ → 正循环

Fable 5 现实:
  发布 → 3天 → 下架 → 零用户 → 零反馈 → 推理设施闲置
       → 数亿美元研发投入 → 可营利窗口3天 → ROI ≈ 负无穷
```

### 3. 市场份额被抢

```
2026 H2 → 竞品推出新模型 → Anthropic 只能用 Opus 4.8 回应
2027 → 企业客户: "我们过去一年都在用 GPT 和 DeepSeek"
      → Anthropic: "但我们有一个从未上线的 Fable 5！"
```

### 4. 禁令反噬禁令制定者

```
禁令目的:   阻止中国获得先进 AI
禁令实际效果:
  ✅ 阻止了英国人用 Fable 5
  ✅ 阻止了加拿大人用 Fable 5（Legion 公司已起诉美国政府）
  ✅ 阻止了美国人自己用 Fable 5
  ❌ 对中国完全没影响（他们有 DeepSeek/GLM）

反噬:
  → Anthropic IPO 估值受损
  → 美国 AI 产业自断最强刀刃
  → 下一次 Anthropic 还会做前沿模型吗？
```

## 总结

| 维度 | Fable 5 正常路径 | Fable 5 被禁后 |
|------|------|------|
| **能力差距** | 领先 22pp，护城河拓宽 | 差距被锁死，竞品在追 |
| **成本摊薄** | 大规模使用 → 优化 → 降价 | 零用户 → 研发沉没 |
| **市场份额** | 抢占高端客户 | 客户流向竞品 |
| **ROI** | 正循环 | **负无穷** |
| **中国影响** | 本来也不依赖 | 不受影响 |
| **美国自身** | 拥有最强可用模型 | **自愿放弃最强可用模型** |

> 🎯 美国政府用一纸禁令，把自己最领先的 AI 公司的最强产品变成了博物馆展品——"看，我们有最强的模型！不好意思，没人能用。" 这可能是 AI 军备竞赛史上最精准的**自残**。
