## 为什么宽基 ETF 更适合做T

### 优势

| 维度 | 宽基 ETF | 个股 |
|------|------|------|
| **风险** | 分散一篮子，不会踩雷 | 单一公司，可能暴雷 |
| **流动性** | 日成交额数十亿，滑点小 | 中小盘股有流动性风险 |
| **手续费** | 免印花税，佣金万分之 0.5~1 | 印花税 0.05%，佣金更高 |
| **可预测性** | 跟踪指数，技术分析有效性强 | 受消息/公告/资金炒作干扰大 |
| **心态** | 不用盯个股新闻，只看指数 | 容易被个股消息影响情绪 |
| **T+0 品种辅助** | 对应期权/期货可做领先指标 | 无 |

### 劣势

- 波动幅度小（日内通常 1%~2%，个股可能 3%~5%）
- 所以做T需要**更大仓位**或**更高频率**才能有可观的绝对收益

## 做T信号体系

### 一、大盘择时（做不做）

**✅ 适合做T的日子：**

- 前一交易日指数涨跌 **±0.5% 以内**（没有强烈方向）
- 预期当天**震荡为主**（无重大消息、无重大数据公布）
- 指数处于**震荡区间**（上有压力下有支撑，非单边趋势）
- 开盘后前 30 分钟**没有跳空缺口**，或跳空后快速回补

**❌ 不适合做T的日子：**

- 前一交易日指数暴涨暴跌 **±2% 以上**（趋势强，容易单边）
- 重大事件日：FOMC、CPI 公布、中央经济工作会议等
- 大幅跳空低开/高开超过 1%，且不回补
- 恐慌性普跌或普涨（全市场涨跌比 >4:1 或 <1:4）

### 二、日内买卖点（什么时候做）

**分时图信号（核心工具）：**

| 级别 | 买入信号 | 卖出信号 |
|------|------|------|
| **A级（强信号）** | 分时 MACD 底背离 + 放量止跌 + 接近日内均线下方 1%+ | 分时 MACD 顶背离 + 放量滞涨 + 远离日内均线上方 1%+ |
| **B级（中等信号）** | 突破分时下降趋势线 + 回踩确认 | 跌破分时上升趋势线 + 反弹确认 |
| **C级（弱信号）** | 日内均线附近缩量止跌 | 日内均线附近缩量滞涨 |

**辅助参考（加分项）：**

- 对应指数期权/期货的隐含波动率突然收缩 → 日内可能反转
- 北向资金盘中流入流出出现拐点
- 同花顺全A指数出现日内放量止跌/滞涨

## 三种典型日内形态与操作

### 形态A：低开高走（正T机会）

```
分时图：
     ┌─────────────────
     │        ╱
     │      ╱
     │────╱   ← 日内均线
     │  ╱
     │╱
─────┘
开盘低开 → 横盘 → 拉升
```

| 步骤 | 操作 | 判断条件 |
|------|------|------|
| 1 | 确认不是恐慌性低开 | 低开幅度 <1.5%，前日无大利空 |
| 2 | 等待底部 | 分时 MACD 底背离、缩量止跌 |
| 3 | **买入**（正T） | 分时突破下降趋势线，或放量反弹 |
| 4 | 挂卖单 | 日内均线附近或买入价 +0.8%~1.2% |
| 5 | 卖出平仓 | 反弹到目标位，不贪 |

### 形态B：高开低走（反T机会）

```
分时图：
╲
  ╲────  ← 日内均线
    ╲
      ╲
        └────
高开 → 横盘 → 跳水
```

| 步骤 | 操作 | 判断条件 |
|------|------|------|
| 1 | 确认不是突破性高开 | 高开幅度 <1%，前日无大利好 |
| 2 | 等待顶部 | 分时 MACD 顶背离、缩量滞涨 |
| 3 | **卖出底仓**（反T） | 分时跌破上升趋势线，或放量跳水 |
| 4 | 挂买单 | 日内均线下方或卖出价 -0.8%~1.2% |
| 5 | 买入接回 | 跌到目标位回补 |

### 形态C：窄幅震荡（不操作）

```
────────────────  ← 日内均线
～~～~～~～~～~～
────────────────
```

全天在 ±0.3% 之内窄幅波动，没有明确方向。**不做。** 宁可错过，不要硬做。

## 仓位与止盈止损

| 参数 | 建议值 | 说明 |
|------|------|------|
| 底仓 | 总资金的 50%~70% | 底仓太少做T空间小 |
| 单次做T仓位 | 底仓的 20%~30% | 例如底仓 10 万元，每次用 2~3 万 |
| 单次目标收益 | **0.5%~1.0%** | ETF 波动率低，目标不能太高 |
| 止损 | **买入价 -0.8%** | 跌破坚决走，日内不补 |
| 单日最大做T次数 | ≤2 次 | 做多错多 |
| 单日最大亏损 | 总资金的 0.3% | 达到就停手 |

> 📌 ETF 做T的核心不是"赚一笔大的"，而是**积小胜为大胜**。每天 0.3%~0.5% 的稳定盈利，一个月下来就是 5%~8%。

## 每日操作流程

| 时间 | 动作 |
|------|------|
| **9:15-9:25** | 看集合竞价，判断开盘方向与力度。观察竞价量能 |
| **9:30-10:00** | **只看不做。** 确认当日走势形态（A/B/C），不抢开盘 |
| **10:00-11:30** | 形态明朗后，寻找 A/B 级信号，执行第一次操作 |
| **11:30-13:00** | 午间复盘：上午操作是否正确？下午预期？ |
| **13:00-14:30** | 寻找第二次机会（如果没有好信号，不做） |
| **14:30-15:00** | **只平仓不开新仓。** 确保收盘持仓恢复底仓数量 |
| **15:00 后** | 记录：做T次数、盈亏、做对/做错的信号，复盘 |

## 核心纪律

1. **不做第三种形态**：只有低开高走和高开低走两种有明显方向的形态才操作，震荡不做
2. **只做看得懂的信号**：A/B 级信号才动手，C 级弱信号放过
3. **先止损后止盈**：买入后马上设好 0.8% 止损，不要心存侥幸
4. **收盘必须平仓**：14:50 前所有做T仓位必须归零，不留隔夜做T仓
5. **重大事件日不做**：FOMC、CPI、GDP 公布日，只观望
6. **连错两把停手**：连续两次止损，说明当天判断出错，当日不再做
7. **盈亏比 >1.5:1**：预期止盈空间至少是止损空间的 1.5 倍才入场

## 一句话总结

> 宽基 ETF 做T = **震荡市 + A/B级信号 + 0.5%~1%目标 + 严格止损 + 收盘归零**。不求一笔暴利，求每月稳定复利。
