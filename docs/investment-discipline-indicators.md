## 验证窗口

以 2026/6/22—6/23 两日极端行情为检验窗口：

```
6/22 上证 +1.78%，有色 +3.93%，成交 3.74万亿（历史第二高）
6/23 上证 -1.37%，有色 -8.36%，近5000只下跌
```

---

## 建议一：管住手

### 是否合理

**极其合理。** 6/22 高潮日追有色 → 6/23 亏 8%+。管住手不是不交易，是不在情绪高潮点交易。

| 标的 | 6/22 | 6/23 | 如6/22追进去 |
|------|:----:|:----:|------|
| 天山铝业 | +7.05% | **-8.53%** | 两天亏1.5% |
| 紫金矿业 | +2.53% | **-8.84%** | 两天亏6.3% |
| 有色ETF | +5.26% | **-8.31%** | 两天亏3.1% |

### 指标化

| 指标 | 计算公式 | 红灯（不开仓） | 绿灯（可操作） |
|------|------|:---:|:---:|
| **成交额情绪** | 当日成交额 ÷ 20日均成交额 | **> 1.3** | 0.8-1.2 |
| **个股三日振幅** | (3日最高-3日最低) ÷ 3日前收盘 | **> 15%** | < 8% |
| **追涨惩罚率** | 昨日涨停股今日平均涨幅 | **< 0%** | > +2% |
| **周交易频次** | 每周开仓次数 | **> 3次** | ≤ 2次 |

### 验证

```
6/22 成交额情绪 = 3.74万亿 ÷ ~2.8万亿 = 1.34  ← 红灯，不开仓
天山铝业 6/22 三日振幅 = 7.9% ← 当日未超标，次日飙到21%

按"情绪 > 1.3 不开新仓"→ 6/22 规避所有有色买入
```

## 建议二：控仓位

### 是否合理

**合理。** 你仓位 ~15%，6/23 有色 -8.36% 仅拖累总资产约 0.5%。若有人 50% 仓位在有色，一天亏掉总资产 4%。

### 指标化

| 指标 | 计算公式 | 红灯 | 绿灯 |
|------|------|:---:|:---:|
| **单品种仓位上限** | 单一持仓 ÷ 总资产 | **> 20%** | ≤ 10% |
| **关联品种合计** | 同板块持仓 ÷ 总资产 | **> 30%** | ≤ 20% |
| **高风险品种识别** | 近20日波动率 > 3% | 仓位减半 | 正常 |
| **现金最低水位** | 现金 ÷ 总资产 | **< 40%** | ≥ 50% |

### 你的当前值

| 指标 | 数值 | 判定 |
|------|:---:|:---:|
| 单品种仓位 | 科大讯飞 ~5.7% | ✅ |
| 关联品种（有色系） | ~11% | ✅ |
| 现金水位 | ~85% | ⚠️ 偏保守，可适度提至 50-60% |

## 建议三：别加杠杆

### 是否合理

**致命级合理。** 6/22 两融余额 29,950亿创历史新高，单日+295亿涌入有色和电子。6/23 有色 -8.36%。

| 情景 | 本金10万投有色ETF | 6/22 | 6/23 | 两日净盈亏 |
|------|:---:|:---:|:---:|:---:|
| 不加杠杆 | 10万 | +5,260 | **-8,310** | -3,050 |
| 1倍杠杆（融资10万） | 20万 | +10,520 | **-16,620** | **-6,100** |
| 2倍杠杆 | 30万 | +15,780 | **-24,930** | **-9,150（逼近强平）** |

> 机构调一次预估（高盛：金价目标 5400→4900），你的杠杆安全垫就少一截。唯一正确的杠杆倍数：**0**。

### 指标化

| 指标 | 计算公式 | 红线 |
|------|------|:---:|
| **个人杠杆倍数** | 融资余额 ÷ 净资产 | **0** |
| **市场杠杆水位** | 两融余额 ÷ A股流通市值 | **> 3%** → 市场风险积累 |
| **维持担保比例** | 总资产 ÷ 总负债 | **< 200%** → 危险 |
| **单日最大回撤×杠杆** | 近20日最大单日跌幅 × 杠杆倍数 | **> 20%** → 单日可能被强平 |

## 建议四：黄金 = 配置非投机

### 是否合理

**完全合理。** 2026上半年黄金 -28%，投机者被屠杀，配置者只需季度看一眼。

```
国际金价：1月 $5,600 → 6月 $4,150（振幅 > 35%）

投机者：1月追高 → 浮亏 -28%
        6/22 买入 → 两天亏 -2%
        试图抄底 6/15 → 五天后又跌 -4.5%

配置者：
  目标：黄金 = 总资产 7%
  6/15 发现占比跌到 5% → 补到 7%
  金价跌了也不慌——因为不是因为"看涨"才买的
```

### 投机 vs 配置识别

| 指标 | 投机模式 | 配置模式 |
|------|------|------|
| **持仓周期** | < 30 天 | > 1 年，仅季度调整 |
| **买入依据** | "金价要涨" | "黄金占比 < 目标" |
| **卖出依据** | 止损/止盈线 | 偏离目标 ±2% 触发再平衡 |
| **盯盘频率** | 每天 | 每季度 |
| **仓位占比** | > 20%（赌方向） | **5-10%** |
| **情绪波动** | 金价决定当天心情 | 只是其中一个持仓 |

### 再平衡规则

```
目标配置：黄金 = 总资产 7%

每季度第一天检查：
  黄金 > 9% → 卖出差额
  黄金 < 5% → 用现金补足到 7%
  5% ≤ 黄金 ≤ 9% → 不动

不需要判断金价走势。规则自动生效。
```

### 黄金不适合投机的数据证据

| 特征 | 黄金 | 适合投机的品种 |
|------|:---:|:---:|
| 近半年最大回撤 | **-28%** | 正常 |
| 6月单日 > 3% 天数 | ≥ 5天 | 较少 |
| 受美联储表态影响 | 一次鹰派 → 3%+ | 有限 |
| 受地缘政治影响 | 美伊谈判进展 → 2% | 有限 |
| 受油价/美元/美债联动 | **三重联动** | 单一 |
| 变量数量 | 6+ 宏观变量，极难预测 | 较少 |

> 黄金是受最多宏观变量联动的资产——任何一个短期投机者都扛不住其中任意一个变量翻脸。

## 自动化脚本

`scripts/discipline-check.py` — 一键输出上述指标。

### 用法

```bash
# 市场公共指标（成交额情绪、两融水位、板块波动率）
python3 scripts/discipline-check.py

# 个人持仓合规检查（需配置 portfolio.json）
python3 scripts/discipline-check.py --portfolio
```
