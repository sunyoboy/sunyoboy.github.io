## 一、筛选标准

> **副业 = 用最少可支配时间产出「工资外收入」。** 你的约束是时间（周末 3h 深度块 + 孕期陪伴优先），不是缺工具。

三条过滤器，任一不满足就砍：

1. **能直接产出内容/产品** — 可变现，不是自嗨效率工具
2. **免费或极低成本** — 契合你的偏好
3. **契合现有栈 + 有限时间** — 不引入高产能开销

## 三、短视频：能做，天花板更高——但是 Phase 2

> 结论：**不要现在做，也不要永远不做。** 正确姿势是「文字先行 · 视频复用」。

### ✅ 为什么值得做

- 财经/投资是短视频高变现赛道（视频号可绑已有公众号、抖音、B站），流量天花板远高于文字
- AI 已把制作成本打下来：数字人口播（蝉镜/有言）+ AI 剪辑（即梦/剪映）+ AI 配音，**不用露脸、不用真拍**
- 你每天产出复盘 = 现成脚本原料，边际成本低

### ⚠️ 为什么不是现在

| 约束 | 说明 |
|------|------|
| 时间 | 周末 3h 已给量化/写作，孕期陪伴优先，再开一条产线会挤爆 |
| 一致性 | 短视频算法吃「周更/日更」持续性，现阶段保证不了 |
| 0 基础 | 写作已立飞轮，视频从 0 起，学习曲线成本高 |
| 🔴 **合规红线** | 在职员工做投资内容，**荐股/预测有合规与法律风险** |

### 🎯 正确姿势：文字先行 · 视频复用

```
第一步（现在）  ：写作变现跑通 — 选题验证 + 建立个人 IP
第二步（触发后）：AI 把验证过的文章「二次加工」成短视频
                → 数字人口播 / 图文视频，边际成本极低
```

**不要从 0 同时开两条产线。**

### 🟢 触发条件（满足任一，即开 Phase 2）

- 公众号过千粉
- 某篇知乎/掘金破万赞
- 出现付费咨询/约稿

### 🛠️ 最小成本做法（真到 Phase 2 时）

| 环节 | 选择 | 理由 |
|------|------|------|
| 形式 | 不露脸 → 数字人口播（蝉镜/有言）或图文视频（即梦/剪映） | 规避露脸的合规 + 精力问题 |
| 内容 | 60 秒一个观点，直接从日复盘/文章切 | 复用现成原料 |
| 平台 | 视频号（绑公众号）起步 | 和现有内容打通 |
| 频率 | 周更 1 条起步，不追日更 | 可持续 |
| 🔴 合规 | 只讲方法论/认知/复盘，标注「不构成投资建议」，不荐股不预测 | 底线 |
