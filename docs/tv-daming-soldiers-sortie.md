# 📺 《大明王朝1566》《士兵突击》·权力与坚持

> 两部剧一政一兵，一古一今，讲同一件事：**人在极端环境里怎么守住自己。**
>
> 大明王朝：嘉靖把帝王术玩到极致，最后成了孤家寡人——**聪明过头害了自己。**
> 士兵突击：许三多笨到不会放弃，最后改变了所有人——**笨到极致赢了人生。**

---

## 一、《大明王朝1566》：权力天花板上的困局

中国最好的历史剧。表面是倒严嵩，内核是嘉靖——一个把自己玩成孤家寡人的皇帝。

### 1. "长江水清灌溉数省，黄河水浊也灌溉数省"

全剧最核心的台词。嘉靖说：长江水清，黄河水浊——但两江都灌溉了两岸。清官（海瑞）有清官的用，贪官（严嵩）有贪官的用。**水清水浊不重要，灌溉了才是关键。**

> 嘉靖的帝王术：不用完人，只用有用之人。但这也是他最大的问题——**他把人当工具，最后自己也被制度反向囚禁。**

### 2. "再苦一苦百姓"

严嵩每遇到难题，开口就是"再苦一苦百姓"。清流和严党斗法，最后买单的都是老百姓。

> **你有没有对"未来的自己"说过类似的话？** "再扛一扛这笔亏损"，"再忍一忍这个坏习惯"——**未来的自己，就是现在被你牺牲的百姓。**

### 3. "任何人答应你的事都不算数，只有你自己能做主的事才算数"

嘉靖临终前对裕王说的最后遗言。一个玩弄所有人一生的皇帝，最后留下的不是帝王术，是这句话。

> CLAUDE.md「不等不靠不要」——嘉靖用一辈子验证了这六个字。

### 4. "朕御极四十年，乾纲独断，乃至于斯"

他以为乾纲独断 = 掌控一切。事实是：**他越想掌控一切，越被严嵩、吕芳、陈洪、徐阶反向掌控。** 每个人都在利用他的多疑和自负——他最信任的人恰恰是骗他最深的。

> **盯盘就是嘉靖式"乾纲独断"**——你以为在看分时图就是掌控，实际是在被分时图玩弄情绪。

### 5. "云在青天水在瓶"

嘉靖在修道，海瑞在坐牢，严嵩在数钱，徐阶在等，杨金水在装疯——所有人都在等嘉靖死，但所有人都在演给他看。**权力最大的孤独：你不知道谁是真心、谁是演戏、谁是真心地演着戏。**

## 二、《士兵突击》：连队的朴素真理

最好的军旅剧。一个"傻子"怎么改变了所有人。

### 1. "有意义就是好好活，好好活就是做有意义的事"

许三多的循环论证——听起来是废话。但他用行动证明了这不是废话：一个人修了一条路，一个人守了半年空营房，一个人做了 333 个腹部绕杠。**"有意义"不是想出来的，是做出来的。**

> 第二增长曲线——1 行代码 > 0 行。不要问"学 Python 有没有意义"，做了才有意义。

### 2. "想到和得到之间，还有一个做到"

钢七连最重要的信条。想到是 0%，得到是 100%，做到就是中间的每一步。

> OKR 14 个 KR ——想到了。做到了吗？格伦姆 25-75% 配置想好了——国债买了吗？

### 3. "光荣在于平淡，艰巨在于漫长"

许三多在草原看守输油管道——没人管、没人在乎、没有任何成绩。但他每天叠被子、站岗、跑步、一个人修路——**没有观众。** 真正的考验不是上战场的那几分钟，是那之前的三千个日日夜夜。

> 你的交易不是每天都有科创+10.73%。大部分日子是科创+1.54%→-2.26%的无聊跳水。**老老实实跑脚本、填复盘、挂条件单——没人鼓掌，也不需要人鼓掌。**

### 4. "现在你混日子，小心将来日子混了你"

成才——钢七连最聪明的兵，枪法最好、脑子最快、最会来事。也是最混的——因为他太聪明，所以不信笨功夫。最后被扔进许三多待过的荒原，才真正开始成长。

> **成才 = 你的反面教材：有天赋但不守纪律，有小聪明没有笨功夫。** 许三多 = 你没有他的钝感力，但可以学他的笨功夫。

### 5. "不抛弃，不放弃"

不是天赋异禀，是笨到不会放弃。

> `growth/信条.md` 已经有这句话了——六个字，做到了就是人生赢家。

### 6. "我才三十岁，还没玩够呢"——这句话是嘉靖说的。

三十岁的许三多在看守荒原，三十岁的嘉靖在修道玩人。前者不知道自己能走多远，后者以为自己能掌控一切。**人不能太聪明，太聪明了就容易看不上笨功夫。**

## 三、两部互补

| | 大明王朝1566 | 士兵突击 |
|------|------|------|
| 核心问题 | 在权力中不失良心 | 在平庸中不放弃坚持 |
| 主人公 | 嘉靖（聪明过头·害了自己） | 许三多（傻到家·改变了所有人） |
| 警示 | 聪明+不信任=孤家寡人 | 聪明+没信仰=混日子 |
| 鼓励 | 海瑞：明知不可为而为之 | 许三多：不知道能不能做到·先做 |

> **两剧一起看：嘉靖的反面就是许三多。** 一个什么都算到了——最后什么都没得到。一个什么都不去想——最后什么都有了。
