## 一、筛选标准

| 维度 | 标准 |
|------|------|
| **需求来源** | 来自消费者/企业真实需求，不是政府投资/GDP 增速依赖 |
| **盈利质量** | 净利润含金量高（经营现金流 > 净利润），不是应收账款堆出来的 |
| **分红能力** | 连续多年分红，股息率 > 3%，分红率 > 30% |
| **护城河** | 品牌/牌照/成本/网络效应，不容易被新进入者颠覆 |
| **资本开支** | 低 CAPEX，赚的钱大部分可以分掉，不需要持续大规模再投入 |

## 二、A股符合标准的行业

### 🥇 白酒（品牌消费品）

| 为什么不是投资驱动 | 为什么能赚钱分红 |
|------|------|
| 需求来自消费者——送礼、宴请、自饮 | 品牌护城河极深，毛利 70%+ |
| 经济好坏都要喝酒，和 GDP 增速弱相关 | 几乎不需要资本开支，赚的钱 70-80% 分掉 |

- **股息率**：茅台 ~2.5%，五粮液 ~3.5%，泸州老窖 ~4%
- **逻辑**：白酒是中国 A 股最符合巴菲特标准的行业——强品牌、轻资产、高分红

### 🥇 银行（牌照垄断 + 利润丰沛）

| 为什么不是投资驱动 | 为什么能赚钱分红 |
|------|------|
| 存贷差是基本盘，经济在就有需求 | 金融牌照稀缺，竞争格局稳定 |
| 房地产风险出清后，银行坏账改善 | 国有大行股息率 5-7%，每年稳定分红 |

- **股息率**：农行 6.2%、工行 5.8%、建行 5.5%、招行 4.8%
- **逻辑**：大行是"准债券"——股价波动小，分红稳定。买农行 ≈ 买票面利率 6% 的永续债

### 🥇 煤炭（资源型现金牛）

| 为什么不是投资驱动 | 为什么能赚钱分红 |
|------|------|
| 电力/钢铁/化工刚性需求，不是基建投资驱动的 | 资源稀缺+供给侧改革后格局出清 |
| 和工业用电/居民取暖挂钩，需求稳定 | 长协价锁定利润，资本开支少，分红率 50-70% |

- **股息率**：中国神华 6-7%，陕西煤业 7-8%，兖矿 5%+
- **逻辑**：神华是煤炭 + 电力 + 铁路一体化，现金奶牛。煤价波动有长协托底

### 🥈 水电（永续电站）

| 为什么不是投资驱动 | 为什么能赚钱分红 |
|------|------|
| 电站建好后，水是免费的 | 大坝折旧完还在发电，永续经营 |
| 和经济周期几乎无关 | 现金流极度稳定，年年分红 |

- **股息率**：长江电力 ~3.5%，华能水电 ~3%
- **逻辑**：长江电力 = 长江上一台印钞机。缺点是增长慢，股息率不如银行高

### 🥈 家电（品牌消费品）

| 为什么不是投资驱动 | 为什么能赚钱分红 |
|------|------|
| 空调/冰箱/洗衣机，家家都要买 | 格力/美的寡头格局稳定 |
| 和住房销售关联度降低，更多是存量更换 | 毛利 30%+，现金流好，分红高 |

- **股息率**：格力电器 6-8%，美的集团 3.5-4.5%
- **逻辑**：格力是分红之王。缺点是行业增长趋缓——但分红够高，当债券持有

### 🥈 通信运营商

| 为什么不是投资驱动 | 为什么能赚钱分红 |
|------|------|
| 每个人都用手机，和基建投资无关 | 三巨头格局稳定，牌照保护 |
| 5G 资本开支高峰已过，进入收成期 | 移动承诺分红率 > 70% |

- **股息率**：中国移动 ~4.5%，中国电信 ~4%，中国联通 ~3.5%
- **逻辑**：通信服务是刚需——经济再差，手机费也得交

### 🥉 高速公路/铁路

| 为什么不是投资驱动 | 为什么能赚钱分红 |
|------|------|
| 路建好后，汽车流量是持续的 | 垄断路段，无竞争 |
| 物流/出行是刚需 | 30 年收费权到期前尽量分红 |

- **股息率**：宁沪高速 ~5%，招商公路 ~4.5%，京沪高铁 ~3%
- **逻辑**：宁沪高速长三角最优质路段，股息率吊打理财产品
