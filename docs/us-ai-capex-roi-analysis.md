## 核心矛盾

```
美国四大厂 2026 年 AI 资本开支合计 >$6,500 亿美元
                vs
全行业 AI 服务年收入目前仅 ~$500 亿（约 1/13）

与此同时：
  DeepSeek V4-Flash 推理成本 = GPT-5.5 的 1/35
  GLM-5.2 在华为昇腾芯片上训练 → 证明不需要 Nvidia
  模型能力差距从 2 年 → 2.7%（Stanford 2026 AI Index）
```

如果开源模型用 1/35 的价格达到 90% 的能力，那美国厂商花了 35 倍的钱买的那 10% 溢价，市场愿意买单吗？

**成本差距的结构性来源**：

| 维度 | 美国路线 | 中国路线 |
|------|------|------|
| **芯片** | Nvidia H100/B200，$25,000-50,000/卡 | 华为昇腾 910C，管制倒逼自研 |
| **架构** | Dense Transformer | MoE（Mixture of Experts），每次推理仅激活 ~5% 参数 |
| **训练策略** | 大力出奇迹 | 稀疏注意力 + 知识蒸馏 + 数据效率优化 |
| **电力** | 美国电网瓶颈，SMR 核电尚未投产 | 东数西算，西部廉价电力 |
| **商业模式** | 卖模型访问权（高溢价） | 开源权重（MIT），靠 API 走量 |

> 🔑 **核心差异不是技术，是哲学**：美国用算力换能力，中国用算法换效率。DeepSeek V3 训练成本 $557 万 vs GPT-4 数亿美元——**同能力级别，成本差 50-100 倍**。

## 为什么高额投入可能入不敷出

### 论据 1：摩尔定律的 AI 版本正在压缩硬件价值

```
AI 算力经济学的核心循环：
  新一代芯片（更贵）→ 训练更大模型 → 模型更强 → 收入增长

但如果：
  新一代芯片 → 开源模型用旧芯片达到 90% 能力
            → 新一代芯片的 10% 溢价无法覆盖成本
            → 循环断裂
```

Goldman Sachs 的量化：
> 一张 $50,000 的加速卡，5 年折旧 = $10,000/年。如果第 2 年新一代卡让它失去经济价值，运营商就持有一个不再产出的资产。贬值不是线性的，是悬崖式的。

### 论据 2：模型定价的"两极分化"正在挤压中间层

JPMorgan 的分析框架：

```
只有两种模型能活：

1. 极致的（Claude Opus 4.8 / GPT-5.5）→ 靠能力吃溢价
2. 极便宜的（DeepSeek V4-Flash）    → 靠量吃规模

中间层（定价 $2-5/M，能力 85-90 分位）→ 会被从两端同时挤压。
而中间层恰好是美国大部分 AI 初创公司的所在。
```

### 论据 3：开源正在吃掉闭源的护城河

GLM-5.2 在华为昇腾上训练，MIT 开源——意味着任何人都可以：
- 下载权重，用自己的 GPU 跑
- 微调，适配自己的业务场景
- **完全不经过 API 付费通道**

如果开源模型每 6 个月追近 1 个百分点，到 2027 年差距可能归零或反超。届时：

```
企业决策者的算账：
  GPT-5.5 API：$10/百万 token
  自部署 GLM-5.2：$0（电费 + GPU 折旧）

  选哪个？ → 除非 GPT 确实好出可感知的差距
```

### 论据 4：推理成本正在从云端向终端迁移

Nvidia DGX Spark（$4,700，128GB 统一内存）可以本地运行 200B 参数模型。3-4 年后，"大多数人每天接触的最强模型会跑在自有设备上"——假设成立，云端的万亿级推理基础设施投资将被绕开。

## 为什么高额投入也可能赚回来

### 论据 1：AI 的远期市场规模可能远超想象

$500 亿年收入 → 需要到 $1.6 万亿才能让投入回报 15%。但如果 AI 最终替代 10-20% 的白领工作，市场规模可能是**数十万亿**——当前投入在那一刻看就是便宜的。

### 论据 2：基础设施折旧 ≠ 价值归零

即使算力单价下降，总需求仍在爆发。**token 消耗量的增速（年化 10x+）远超单价下降（年化 -50%）**——总收入仍在涨。就像云计算：单价降了 10 年，AWS 收入翻了 100 倍。

### 论据 3：赢家通吃的格局值得赌

如果通用人工智能（AGI）是 winner-take-all 的市场，那花 $6,500 亿赌一个价值数万亿的市场，**从期望值看是合理的**——即使 80% 概率全赔，20% 概率赚 $5 万亿，期望值仍是正的。

### 论据 4：不是所有的 AI 投入都是"可替代的"

前沿能力（多步推理、Agent 自动化、科学发现）仍然需要巨大的训练算力。DeepSeek 和 GLM 证明了在已知路线上可以用更少资源跑出好结果，但探索未知路线（新的架构、新的训练范式）仍然需要烧钱试错。

## 最可能的结局

```
美国 AI 投入不会是 "全赔" 也不会是 "全赚"

最可能的三种资产命运：

🥇 赢家（30% 概率）：
  Google 的 TPU 定制芯片 + 垂直整合 → 单位成本优势 → ROI 最好
  Meta 的 AI 广告增收已验证（$600 亿年化）→ 投入有直接产出

🥈 打平（40% 概率）：
  Microsoft 的 Copilot 收入增长但 Azure AI 利润率被压缩
  Amazon 的 AWS AI 增长但 FCF 被投入拖累

🥉 亏损（30% 概率）：
  大量数据中心在 2-3 年后因芯片迭代而提前减值
  开源模型让闭源 API 定价权瓦解
  GPU 租赁市场供过于求，价格暴跌
```

> 中国路线的最大冲击不是技术上的——是**价格信号**。当市场发现同样的事情可以用 1/35 的成本做到，整个 AI 软件层的定价天花板就被锁死了。而硬件投入是锁死的——你买了 $50,000 的 GPU，折旧就是折旧，不会因为竞争对手降了价而减少。

## 与 A 股投资的关联

### 直接影响

| 品种 | 传导机制 |
|------|------|
| **人工智能板块** | 如果 DeepSeek/GLM 继续开源降价 → 全球 AI 软件层利润被压缩 → 国内 AI 应用公司毛利率承压 |
| **半导体/算力** | 如果华为昇腾被验证可规模训练前沿模型 → 利好国产芯片链（中芯国际/寒武纪/海光） |
| **光通信/CPO** | 如果数据中心建设放缓 → 利空光模块出口（中际旭创/天孚通信/新易盛） |
| **电力设备** | 如果美国数据中心需求降温 → 利空电网设备出口 |

### 操作含义

> 你当前持仓中信创/有色/黄金均不直接暴露于这个风险。但如果你考虑加仓 AI 相关标的，需要意识到：**国内的 AI 算力叙事（光模块出口、服务器代工）高度依赖美国数据中心资本开支的持续性。开源模型的低成本路线对这个叙事是结构性的利空。**
