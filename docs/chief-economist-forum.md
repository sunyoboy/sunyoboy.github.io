# 中国首席经济学家论坛（CCEF）

> 2012 年成立 · 97+ 位首席经济学家 · 非营利智库平台
>
> 官网：[chinacef.cn](https://www.chinacef.cn/#/home)

---

## 是什么

中国首席经济学家论坛（China Chief Economist Forum）由国务院参事夏斌于 2012 年 11 月在上海发起，成员来自国内外一流金融机构的首席经济学家。国务院副总理刘鹤寄语："**贵在恒，望成为有较大影响力的智库。**"

---

## 为什么值得关注

| 价值 | 说明 |
|------|------|
| **宏观判断力** | 97 位首席经济学家的共识与分歧，相当于免费买了一张"宏观投票图" |
| **政策敏感度** | 成员中多人是央行/发改委前官员，对政策转向有先验敏感 |
| **行业覆盖面** | 银行/券商/外资行/互联网——不同立场形成交叉验证 |
| **年度年会** | 每年 1 月 50 位经济学家集中发声，相当于全年宏观路演浓缩版 |

## 与 KnowingDoing 体系的结合

### 如何使用

```
盘前 9:00-9:25（高能量）
  → 快速扫一眼论坛首页，看今天有没有首席发重要文章
  → 如果有关于货币政策/FOMC/汇率的文章 → 记录到当日复盘「宏观背景」栏

周末深度块（周六上午 3h）
  → 精读一篇年度报告或年会纪要 → 与自己的持仓方向交叉验证
  → 对比多个经济学家对同一问题的分歧 → 理解不确定性边界

月度回顾
  → 对照经济学家共识 → 检查持仓是否与宏观方向一致
```

### 避坑

- 不盲从：首席的判断也会错，用他们来**补充视角**，不是替代决策
- 不追热：论坛更新频率不高（非财经新闻），适合做**慢变量参考**而非短线决策
- 区分"预测"与"解释"：事后解释每个人都会，事前预测才体现水平——关注年会和季度展望
