# 投资教训与根因分析

> 每次犯错都是系统升级的机会。不是归因于运气，是找到可修正的模式。

---

## 一、核心发现：能算出来的赚钱·感觉出来的亏钱

### 五笔交易对照

| 品种 | 买入理由 | 可验证？ | 结果 | 盈亏 |
|------|------|:--:|:--:|:--:|
| 煤炭ETF | PE低·高股息·夏季用电 | ✅ 三个都可查 | 获利了结 | +2,300 |
| 卫星ETF | 军工概念·「跌够了」 | ❌ 无法验证 | 持有中亏损 | -2,741 |
| 天山铝业 | 周期股·「便宜」 | ❌ 没算PE分位 | 持有中亏损 | -2,452 |
| 西部材料 | 「跌了总能涨回来」 | ❌ 纯感觉 | 止损离场 | -1,027 |
| 稀土ETF | 定价权 | ❌ 没算利润 | 清仓 | -132 |

### 规律

```
数字买入: 1次 → 赚
感觉买入: 4次 → 全亏

→ 区别不在于事后涨不涨
→ 区别在于买入那一刻有没有算清楚
→ 算不清楚的仓位，迟早会被市场的波动甩出去
```

## 二、卖飞的根本原因：买入时就注定了

### 亏损→割肉→涨停 的循环

```
买入: 「这次不一样·它能涨」（感觉驱动）
持有: 跌了→等反弹→跌更多→信心崩溃
卖出: 「永远不会涨了」
涨停: 「我就知道！」

问题不在卖出时机——在买入那一秒
买入理由不清→持有中不确定→一有波动就卖→卖了就飞
```

### 煤炭为什么没卖飞

```
买入: PE低·高股息（数字）
持有: PE还是低·股息还是高（可验证）
卖出: +4.22%·偏离度太高（同一框架）
→ 买和卖基于同一个标准→不后悔
```

## 四、失败分析原则

### 遇到以下情况，必须刨根问底

```
□ 单笔亏损超过总资产1%
□ 卖出后标的涨停/暴涨
□ 同一个错误出现2次以上
□ 偏离计划的操作（如买不在候选池的品种）
□ 条件单未挂导致亏损
```

### 分析方法

```
1. 买入理由是什么？→ 数字还是感觉？
2. 买入时过了避雷三问吗？→ 缺哪个？
3. 持有过程中什么变了？→ 理由还在吗？
4. 卖出决策基于什么？→ 止损/时间/理由消失？
5. 系统哪个环节允许了这个错误？
```

### 不是归因于运气

```
❌ 「运气不好」 → 每次都一样 → 系统没升级
✅ 「系统缺了什么」 → 加入纪律 → 下次不犯
```
