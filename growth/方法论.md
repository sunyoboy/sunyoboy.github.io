## 核心框架：知行合一 × PDCA

```
          知（认知）                行（实践）
       ┌─────────────┐       ┌─────────────┐
       │  问题清单    │       │   每日复盘   │
       │  知识库      │       │   交易执行   │
       │  阅读/学习   │       │   实验验证   │
       └──────┬──────┘       └──────┬──────┘
              │                    │
              │    ┌──────────┐    │
              └────│ 个人成长  │────┘
                   │ (合一)   │
                   └──────────┘
                        │
                  内化为能力/直觉
```

| PDCA | 知行 | 动作 | 产出 |
|------|------|------|------|
| **P** (Plan) | 知 | 年度 OKR → 月目标 → 周计划 | 目标清晰、方向明确 |
| **D** (Do) | 行 | 每日执行、交易操作、学习实践 | 行动记录、交易日志 |
| **C** (Check) | 行→知 | 复盘对比：实际 vs 目标，找偏差原因 | 周报、月回顾、偏差分析 |
| **A** (Act) | 知→行 | 改进方案写入问题清单，沉淀方法论到知识库 | 问题驱动迭代、知识库更新 |

### 完整体系架构

| 层级 | 工具 | 驱动 |
|------|------|------|
| 🔥 信条 | 不抛弃不放弃 · 三个不相信 | 精神 |
| 🔄 方法论 | PDCA × 知行合一 | 操作系统 |
| 🎯 OKR | 4 O × 14 KR | 方向 |
| 📋 周报 | P→D→C→A 结构 | 节奏 |
| 📖 复盘 | 矛盾论分析主要矛盾 | 深度 |
| 💡 知识库 | 矛盾论/实践论 + 交易方法 | 沉淀 |

```
        信条（精神内核）
            ↓
        方法论（操作系统）
       PDCA × 知行合一
      /    |    |    \
     P     D     C     A
    规划   执行   检查   改进
    OKR   复盘   周报   问题清单
```

## 五步闭环

```
问题清单（发现问题）
    ↓
知识库（寻找答案/方法论）
    ↓
复盘执行（实践检验）     ← 矛盾论：抓住主要矛盾
    ↓
周/月回顾（Check vs OKR）
    ↓
个人成长（内化 + 调整目标）
    ↓ 循环
```

> 「实践论」提醒：每一步都要经过 **实践→认识→再实践→再认识** 的螺旋上升。不是走一遍就完了，是反复走、每遍更深。

## 每日节奏

| 时间 | 动作 | 对应 |
|------|------|------|
| 盘前 | 看日历（FOMC/数据）、定当日策略 | P |
| 盘中 | 执行交易、做T、记录 | D |
| 盘后 | 写日复盘：做了什么、学到了什么、明日计划 | C |
| 睡前 | 一个问题：今天最大的收获/失误是什么？ | A |

---

## 每周节奏

| 时间 | 动作 | 对应 |
|------|------|------|
| 周末 | 填写[个人周报](个人周报模板) | C |
| 周末 | 回顾 OKR 进度，是否需要调整 | A |
| 周末 | 制定下周计划（≤3 件最重要的事） | P |
