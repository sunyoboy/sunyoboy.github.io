
## 达克效应 · 认知四阶

> 不知道自己不知道 → 知道自己不知道 → 知道自己知道 → 不知道自己知道

```
第一阶  不知道自己不知道              第二阶  知道自己不知道
       ┌──────────────────┐         ┌──────────────────┐
       │ 盲目自信，啥都敢说  │  ──→   │ 开始学，越学越觉得浅 │
       │ "这还不简单？"     │  撞墙   │ "原来这么深……"    │
       └──────────────────┘         └──────────────────┘
                                            │ 持续学
                                            ↓
第四阶  不知道自己知道              第三阶  知道自己知道
       ┌──────────────────┐         ┌──────────────────┐
       │ 肌肉记忆，自动调用  │  ←──   │ 能说清、能教人     │
       │ "我没想就做对了"   │  内化   │ "这个问题我懂"     │
       └──────────────────┘         └──────────────────┘
```

**个人定位：**

| 领域 | 当前阶段 | 证据 |
|------|:--:|------|
| A股交易 | 二阶→三阶 | 框架已建立，能说清为什么买/为什么不买，但盘中还会被系统1劫持 |
| Python/量化 | 二阶 | 知道自己不会的太多，pip install 还没装 |
| 写作输出 | 二阶 | 能写复盘但精品文章还没产出 |
| AI工具使用 | 三阶 | Claude Code/脚本/知识库 已形成肌肉记忆 |

> 🎯 最危险的是第一阶——"不知道自己不知道"的人什么都敢追、什么都敢满仓。你能说自己"在二阶"，已经比一阶的人安全得多。

### 四句行动纲领

```
有逻辑  → 结论要能追溯推导过程，不是"我觉得"
重证据  → 用数据说话，不是用故事说话
先专业  → 扎实掌握基本功，再谈创新
再反馈  → 结果出来后对照逻辑修正，PDCA 的 C 和 A
```

**对交易的映射：**

| 纲领 | 交易对应 |
|------|------|
| 有逻辑 | 买之前七层框架跑一遍——不是"我感觉要涨" |
| 重证据 | 成交额、偏离度、量能——不是"别人都在买" |
| 先专业 | 把止损/仓位/入场价三个数字算出来再动手 |
| 再反馈 | 复盘时对照买入理由和实际结果——打脸了就修正框架 |

## 行动准则

1. **遇任务** → PDCA 走一遍：Plan(计划) → Do(动手) → Check(对比) → Act(改进)
2. **遇困难** → 矛盾论：抓主要矛盾，拆解为最小可解决单元，逐一攻克
3. **遇对手/挑战** → 实践论：实践→认识→再实践→再认识，直到战胜
4. **遇恐慌/混乱** → 岿然不动：市场暴跌时不盲从抛售，暴涨时不冲动追高，守住自己的策略和纪律
