## 九、📖 FAQ：国债ETF如何获得收益？

> 很多人对债券ETF的收益机制感到困惑——「价格不怎么动，怎么赚钱？」下面是完整的解释。

### 一、债券ETF收益 = 两部分

```
债券ETF总收益 = 利息收入 + 价格变动（资本利得）
                 ↑            ↑
              稳定持续      利率下行时涨
```

### 二、利息收入（票息）

30年国债ETF持有一篮子30年期国债，这些国债每年支付利息。ETF会把利息累积到净值里。

```
例如：30年国债票息约2.5-3.0%/年
持有一年 ≈ 净值自然增长2.5-3.0%
这和你存银行拿利息是一个道理
```

### 三、价格变动（资本利得）— 这才是大头

**核心规律：利率↓ → 债券价格↑**

```
央行放水1万亿 → 市场利率从2.8%降到2.5%
                ↓
旧债券（锁定2.8%利息）比新债券（只有2.5%利息）更值钱
                ↓
旧债券价格涨 ≈ 利率降幅 × 久期
30年国债久期≈18年 → 利率降0.3% × 18 = 价格上涨约5.4%
```

这就是为什么你看到511090今天涨了+0.17%——对债券来说，0.17%日涨幅相当于股票涨了2-3%。

### 四、具体算一笔账

```
场景：买入100,000元30年国债ETF，持有一年

收入来源1：票息收入    100,000 × 2.8%  = +2,800
收入来源2：利率下行0.3%  100,000 × 5.4%  = +5,400
────────────────────────────────────────────
合计年化收益：            ~8,200（约8.2%）

对比：
  逆回购（当前1.36%）：  100,000 × 1.36% = +1,360
  30年国债ETF：                           +8,200

差额：+6,840 元 ← 这就是"现金升级"的意义
```

### 五、风险是什么？

| 风险 | 概率 | 影响 |
|------|:--:|------|
| 央行加息/收紧 | 极低 | 利率↑→债券价格↓，30年国债可能跌5-10% |
| 通胀超预期 | 低 | 通胀高→央行被迫加息→同上 |
| 信用风险 | 无 | 国债由国家信用担保，不存在违约 |
| 流动性风险 | 低 | 511090日均成交可观，能卖得掉 |

> **当前中国在降息周期（非农弱+央行放水），债券最大的风险（加息）触发概率极低。** 这是配置30年国债的最佳窗口期。

### 六、一句话

> **30年国债不是用来交易的——是用来让你的24万闲置资金从「1.36%逆回购利息」升级到「5-8%年化」的现金管理工具。** 股票负责进攻，债券负责让闲钱不生锈。
