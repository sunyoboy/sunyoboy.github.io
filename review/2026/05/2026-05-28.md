### 盘面特征

探底回升，双创领涨。早盘三大指数集体低开，创业板一度跌近2%，沪指回踩60日均线后企稳。午后科技股爆发，CPO、PCB、半导体集体拉升，指数V型翻转集体收红。成交量2.97万亿，较前日缩量约2700亿。上涨超3000家，涨超9%个股逾160家——连续两日普跌后迎来结构性修复。

### 热门板块

- **CPO/光通信**：掀涨停潮，联特科技涨超15%，中兴通讯涨停，中际旭创新高
- **半导体**：硅片方向领涨，华虹公司涨近16%创历史新高，燕东微20cm涨停
- **培育钻石/超硬材料**：惠丰钻石30cm涨停，四方达、黄河旋风涨停，金刚石散热材料规模化应用突破
- **PCB/覆铜板**：AI服务器PCB价值量暴涨233%，诺德股份、沃格光电涨停
- **MLCC**：日韩厂商集体涨价，高端涨幅达35%，鸿远电子、风华高科涨停
- **超级电容**：AI数据中心供电架构升级，风华高科、火炬电子、江海股份涨停
- **电力/煤炭**：夏季用电高峰+电改加速，宁波能源、华电能源涨停
- **领跌**：白酒（金种子酒-5%）、贵金属（赤峰黄金等）、证券、机器人

### 涨跌原因

**上涨催化：**
1. 科技主线内部高低切换——资金从高位存储芯片/先进封装向CPO、PCB、覆铜板、超级电容等细分扩散
2. MLCC日韩厂商集体涨价，高端涨幅达35%，被动元件板块集体爆发
3. 碳化硅AI领域结构性紧缺，交期延长至52周
4. 金刚石散热材料规模化应用突破，超硬材料全线爆发
5. 电力改革落地加速+夏季用电高峰，防御性配置需求

**量缩价稳说明：**
1. 经历26-27日连续调整后抛压衰竭，午后抄底资金涌入科技主线
2. 缩量反弹力度温和，资金仍偏谨慎

### 机构观点

- 中信证券：市场处于震荡筑底阶段，关注电力、有色等低估值板块
- 华泰证券：指数窄幅震荡，恐慌释放后修复需求强烈

- 星图金融：大科技才是最能凝聚共识的方向，"科技牛"远未结束

## APP 条件单可执行的量化策略

> 来源：Python 量化交易策略全景文档分析，筛选可通过同花顺/东方财富/华泰涨乐等 APP 条件单落地的策略

### 直接可执行

| 策略 | 条件单设置 | 适用 APP |
|------|-----------|---------|
| **双均线交叉** | 触发条件：MA5 上穿 MA10 → 买入；MA5 下穿 MA10 → 卖出 | 同花顺 / 东方财富 / 华泰 |
| **价格突破** | 触发条件：价格 ≥ N日新高 → 买入；价格 ≤ N日新低 → 卖出 | 几乎所有券商 APP |
| **布林带突破** | 触发条件：价格 ≥ 上轨 或 价格 ≤ 下轨 | 同花顺 / 华泰 |
| **RSI 反转** | 触发条件：RSI ≤ 30 → 买入；RSI ≥ 70 → 卖出 | 同花顺 / 东方财富 |
| **止损止盈** | 按百分比或固定价格：跌 N% 止损 / 涨 N% 止盈 | 所有 APP |

### 部分可执行（需 APP 支持多条件组合）

| 策略 | 条件单设置 | 限制 |
|------|-----------|------|
| 三重均线过滤 | 买：MA5上穿MA10 + 价格在MA20上方（两个条件同时满足） | 华泰支持多条件，多数 APP 仅单条件 |
| 唐奇安通道 | 价格≥20日最高价买入 | 仅有入场，无仓位管理/加仓 |
| MACD 信号 | MACD 金叉/死叉触发 | 同花顺高级条件单支持 |

### 无法通过条件单执行

| 策略 | 原因 |
|------|------|
| 海龟交易（完整版） | 需要 ATR 动态仓位 + 金字塔加仓，条件单无法实现 |
| 动量策略（跨月排序） | 需要回溯 N 月涨幅并排序选股 |
| 多因子选股 | 需要数据处理流水线 + 横截面排序 + 组合构建 |
| 轮动策略 | 需定期重新计算排名并批量调仓 |
| 统计套利 | 需实时计算价差 + 多腿同时下单 |
| 事件驱动 | 触发源是公告/新闻，非价格指标 |

### 风控类（条件单最佳场景）

| 手段 | 设置 | 建议参数 |
|------|------|---------|
| 固定止损 | 价格 ≤ 成本价 × (1 - N%) | A 股 5-8% |
| 移动止损 | 从最高点回落 N% | 回落 5% |
| 固定止盈 | 价格 ≥ 成本价 × (1 + N%) | 视策略 |
| 时间止损 | N 日内未达预期则平仓 | 5-10 个交易日 |

### 进阶验证路径

```
简单策略 → 聚宽回测（免费）→ 条件单实盘小仓位验证
中等策略 → 聚宽回测 → Backtrader → QMT 仿真 → 实盘
复杂策略 → 聚宽因子研究 → vnpy 仿真 → 实盘
```
