## 关键收获

- **Gitee Pages**：2024-05-01 起全面下线，国内替代可选 EdgeOne Pages、AtomGit Pages
- **KnowledgeDoing**：远程改为 `git@gitee.com:ThinkingDoing/ThinkingDoing.git`，已提交推送
- **uni-app 代码审查发现 8 个问题**：
  - DeepSeek API Key 硬编码在 `utils/config.ts`，客户端可反编译提取
  - userId='1' 硬编码在 3 个社区页面，点赞/评论/发帖均以固定用户提交
  - `store/index.js` reject(res) 变量错误（res 应为 err）
  - `consult.vue` 调用了不存在的 `chatStore.sendMessage()`
  - `wx.*` API 多处使用，非微信平台会崩溃，需改为 `uni.*`
  - 多处空值保护和 return 缺失
- **Figma vs 实现**：7 个页面逐项对比，首页双版本并存、私信功能缺失、AI 页面碎片化（4 个页面重叠）
- **多机型适配**：CustomTabBar 和 5 个页面缺少 `env(safe-area-inset-bottom)`，iPhone X+ 底部被遮挡；chongniaobang.vue 固定 750rpx 大屏留白
- **UGC 合规 P0 红线**：无内容审核（msgSecCheck/imgSecCheck）、无举报机制、发帖无登录校验、用户协议缺 UGC 条款、类目需企业主体+ICP 许可证
- 所有审查结果已保存至 `CR/` 目录、`UGC.md`、`TODO.md`
- Gitee HTTPS token 失效，已切换为 SSH 推送
- GitHub 远程仓库尚未配置（待添加）
