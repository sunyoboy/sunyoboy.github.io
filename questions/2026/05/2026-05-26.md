# 如何通过手机或其他工具模拟量化交易

> 2026-05-26

---

## 路径一：手机 APP 直接操作（零代码）

### 1. 券商条件单

同花顺 / 东方财富 / 华泰涨乐的「条件单」功能，手机上几步点选即可运行。

覆盖策略：均线交叉、价格突破、RSI、MACD 等约 30% 的策略。

**适合**：验证单一指标的买卖信号是否有效。

### 2. 同花顺「模拟炒股」

> 同花顺 APP → 交易 → 模拟炒股 → 100 万虚拟资金

- 按真实行情成交，T+1 规则与实盘一致
- 可同时跑多个策略，分别建不同模拟账户
- **关键用法**：每周手动按策略信号调仓，记录净值曲线

**适合**：轮动策略、动量策略等需要批量调仓的场景。

### 3. 雪球「组合」

> 雪球 APP → 我的 → 组合 → 创建组合 → 调入调出

- 创建后自动跟踪净值走势，对比沪深 300
- 支持 ETF 组合和行业轮动验证
- 公开组合可以让别人跟投（附带验证策略吸引力）

**适合**：ETF 轮动、股债配置、多因子选股组合。

## 路径二：手机浏览器直开的在线回测平台

### 1. 聚宽 (JoinQuant) — 移动端适配

> 手机浏览器打开 joinquant.com → 登录 → 研究环境

核心代码模板（双均线为例，30 行即可跑通）：

```python
def initialize(context):
    g.stock = '000300.XSHG'
    g.ma_short = 5
    g.ma_long = 20

def handle_data(context, data):
    prices = attribute_history(g.stock, g.ma_long, '1d', 'close')
    ma5 = prices['close'][-g.ma_short:].mean()
    ma20 = prices['close'].mean()
    if ma5 > ma20 and context.portfolio.positions[g.stock].amount == 0:
        order_target_percent(g.stock, 1.0)
    elif ma5 < ma20:
        order_target_percent(g.stock, 0)
```

**适合**：有基本 Python 概念、想跑历史回测的用户。

### 2. TradingView — Pine Script 策略回测

> 手机浏览器打开 tradingview.com → 图表 → Pine 编辑器

TradingView 的「策略测试器」可在手机浏览器上使用，写 Pine Script（比 Python 更简单）后直接在 K 线图上叠加买卖信号和收益曲线。

**优势**：
- 手机端体验优于聚宽
- 覆盖 A 股/美股/期货/外汇/加密货币
- Pine Script 社区大量现成策略可直接复制使用

**适合**：技术指标类策略（均线/MACD/布林带/RSI 组合），不写 Python。

## 路径三：券商量化交易终端（需要 PC）

### 1. QMT / MiniQMT（国金证券）

> 免费开通国金证券账户 → 下载 QMT 客户端 → 策略研究模式

| 能力 | 说明 |
|------|------|
| 回测 | 2010 年至今全量 A 股数据 |
| 仿真 | 虚拟资金按真实行情跑策略，验证 1-3 个月 |
| 实盘 | 通过后一键切换，Python 策略直接下单 |
| 数据 | 内置行情、财务、龙虎榜等，不用自己爬 |

**最大优势**：回测→仿真→实盘在同一个平台无缝切换，策略验证周期从月缩短到天。

### 2. 华泰 MATIC

华泰证券的量化平台，手机 APP（涨乐财富通）内置的「智能条件单」是其简化版，PC 端 MATIC 支持完整 Python 策略。

### 3. 国信 iQuant

浏览器在线平台，无需下载软件，支持 Python 策略回测和仿真交易。

## 路径四：最低成本的「人工量化」

不需要写一行代码，但严格按量化纪律执行：

```
1. 制定策略规则（写在纸上或备忘录）
   例：MA5上穿MA20买入，下穿卖出，单只仓位不超过20%

2. 每天收盘后（或盘中设定时提醒）
   → 打开同花顺APP → 条件选股 → 按规则筛选
   → 在模拟炒股中手动调仓

3. 用 APP 记录每笔交易
   → 备忘录记下：日期、标的、方向、价格、理由

4. 每月汇总
   → 算胜率、盈亏比、最大回撤、跑赢指数多少
```

**工具组合**：

| 功能 | 工具 |
|------|------|
| 信号筛选 | 同花顺「条件选股」 |
| 虚拟交易 | 同花顺模拟炒股 / 雪球组合 |
| 记录跟踪 | 手机备忘录 / Notion / Excel |
| 绩效计算 | 每月手动汇总（10 分钟） |

## 推荐组合

```
第一周：同花顺条件单跑一个双均线策略（感受「自动化触发」）
        + 同花顺模拟炒股手动执行一个轮动策略（理解「调仓节奏」）

第二周：TradingView 手机版复制一个 MACD+RSI 策略，回测看历史表现

一个月后：聚宽写第一个 Python 策略，跑 10 年回测结果

三个月后：策略验证通过 → QMT 仿真 → 小资金实盘
```

**一句话总结**：手机条件单是「让机器帮你盯盘」，TradingView 是「在手机上验证想法」，聚宽是「用数据证明策略有效」，QMT 是「从模拟走到实盘的最后一步」。
