## 一、孕期基础知识

### 40周划分

| 孕期 | 周数 | 关键特征 |
|------|:--:|------|
| 第一孕期 | 0~13周 | 早孕反应·器官形成·最关键 |
| 第二孕期 | 14~27周 | 精力回升·胎动·黄金期 |
| 第三孕期 | 28~40周 | 待产准备·身体笨重 |

### 足月分类

| 分类 | 周数 |
|------|:--:|
| 早期足月 | 37~38周 |
| 完全足月 | **39~40周**（最理想） |
| 晚期足月 | 41周 |
| 过期妊娠 | ≥42周 |

## 二、异地关键节点

### 必须到场的4次

| 周数 | 检查 | 重要性 |
|:--:|------|:--:|
| ~7周 | B超看胎心 | ✈ 第一次确认·意义重大 |
| 11~13周 | NT + 早唐 | ✈ 排畸第一步 |
| 20~24周 | 大排畸 | ✈ 能一起看到宝宝的脸 |
| 36周起 | 待产 | 陪到产后2周+ |

> 底线：4次 ✈ + 最后1个月在身边

## 三、日常异地陪伴

### 每天

```
□ 早安消息（让她一睁眼就看到）
□ 睡前视频（15~30分钟·只听不评判）
□ 用药确认（黄体酮/肝素/羟氯喹/阿司匹林/优甲乐）
```

### 每周

```
□ 同步孕期APP（了解这周胎儿变化）
□ 寄物资（京东次日达）
□ 预约挂号·查报告
```

## 四、陪伴三原则

```
1. "不能天天在"不等于"不在" — 每一个重要节点都出现
2. 她的情绪比解决方案重要 — 多说"辛苦了"·少说"你应该"
3. 你在北京努力工作的意义 — 是另一种"在场"
```
